# 周赛 01 - D 题：成绩排序

**难度**：★★★☆☆  
**考点**：结构体、多关键字排序、自定义比较

---

## 题目描述

某班有 n 名学生，每人有学号、姓名、语文成绩、数学成绩。请按**总分从高到低**排序；若总分相同，则按**语文成绩从高到低**；若仍相同，则按**学号从小到大**。输出排序后的学生名单（学号、姓名、总分），每行一条。

**本题要求使用结构体存储每位学生的信息。**

## 输入格式

第一行一个整数 n（1 ≤ n ≤ 100）。  
接下来 n 行，每行格式为：学号 姓名 语文 数学。学号为不超过 10 位的数字字符串，姓名为不含空格的字符串，语文、数学为 0～100 的整数。

## 输出格式

输出 n 行，每行格式为：学号 姓名 总分。按题目要求的顺序输出。

## 样例

**输入**
```
4
001 Alice 90 85
002 Bob 90 80
003 Carol 85 90
004 David 90 85
```

**输出**
```
004 David 175
001 Alice 175
002 Bob 170
003 Carol 175
```

**说明**：Alice、David 总分均为 175，语文均为 90，学号 001 < 004，故 Alice 在前、David 在后；Bob 与 Carol 总分均为 170，按题意再按语文，Bob 90 > Carol 85，故 Bob 在前。

## 提示

- 定义结构体，包含学号、姓名、语文、数学、总分等成员。
- 使用 `sort` 时传入自定义比较函数或 lambda，按总分降序、语文降序、学号升序。
