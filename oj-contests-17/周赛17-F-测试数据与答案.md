## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <cstring>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    vector<int> adj[105];
    for (int i = 0; i < m; i++) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    int dist[105];
    memset(dist, -1, sizeof(dist));
    queue<int> q;
    dist[1] = 0;
    q.push(1);
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : adj[u]) {
            if (dist[v] == -1) {
                dist[v] = dist[u] + 1;
                q.push(v);
            }
        }
    }
    cout << dist[n] << "\n";
    return 0;
}
```

**关键点**：
1. BFS 保证第一次到达节点时的距离是最短距离；
2. `dist[v] == -1` 判断未访问，避免重复入队；
3. 无向图每条边双向存储；
4. `dist[n]` 即为答案，若不可达则为 `-1`（初始值）。
