# F - 地下密道

**难度**：★★★★★  
**考点**：图存储（邻接表）+ BFS 最短路径  
**GESP 级别**：6级

---

## 题目背景

古堡地下有若干密室，密室之间由隧道相连。探险者从密室 1 出发，想以最少的步数抵达密室 $n$。

## 题目描述

给定一个 $n$ 个节点、$m$ 条无向边的图（节点编号 $1$ 到 $n$），求从节点 $1$ 到节点 $n$ 的最短路径长度（经过的边数）。若不可达，输出 $-1$。

## 输入格式

第一行两个正整数 $n$、$m$（$2 \le n \le 100$，$0 \le m \le 1000$）。

接下来 $m$ 行，每行两个整数 $u$、$v$（$1 \le u, v \le n$），表示节点 $u$ 和 $v$ 之间有一条无向边。

## 输出格式

输出一个整数：从节点 1 到节点 $n$ 的最短路径长度，若不可达输出 $-1$。

## 样例

**输入 1**

```
5 5
1 2
2 3
3 5
1 4
4 5
```

**输出 1**

```
2
```

**说明**：最短路径为 $1 \to 4 \to 5$，长度 2。

---

**输入 2**

```
4 2
1 2
3 4
```

**输出 2**

```
-1
```

**说明**：节点 1 和节点 4 不连通。

## 提示

BFS 模板：

```cpp
#include <queue>
vector<int> adj[105];
int dist[105];
memset(dist, -1, sizeof(dist));

queue<int> q;
dist[1] = 0;
q.push(1);
while (!q.empty()) {
    int u = q.front(); q.pop();
    for (int v : adj[u]) {
        if (dist[v] == -1) {
            dist[v] = dist[u] + 1;
            q.push(v);
        }
    }
}
cout << dist[n];
```

- `dist[v] == -1` 表示未访问；
- BFS 保证第一次到达某节点时的距离就是最短距离；
- 无向图每条边要存两次（双向邻接表）。
