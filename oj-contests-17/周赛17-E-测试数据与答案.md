## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string sa, sb;
    cin >> sa >> sb;
    // 翻转为低位在前
    reverse(sa.begin(), sa.end());
    reverse(sb.begin(), sb.end());
    int lenA = sa.size(), lenB = sb.size();
    int result[105] = {0};
    int borrow = 0;
    for (int i = 0; i < lenA; i++) {
        int a = sa[i] - '0';
        int b = (i < lenB ? sb[i] - '0' : 0);
        int diff = a - b - borrow;
        if (diff < 0) { diff += 10; borrow = 1; }
        else borrow = 0;
        result[i] = diff;
    }
    // 去掉高位前导零
    int len = lenA;
    while (len > 1 && result[len-1] == 0) len--;
    // 从高位到低位输出
    for (int i = len - 1; i >= 0; i--) cout << result[i];
    cout << "\n";
    return 0;
}
```

**关键点**：
1. 翻转字符串使低位在前（方便对齐相减）；
2. 逐位相减，注意借位 `borrow`；
3. 去掉前导零，但至少保留一位（结果为 0 时输出 `0`）；
4. 保证 $A \ge B$，所以最终 `borrow` 一定为 0。
