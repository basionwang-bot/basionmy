# B - 宝藏众数

**难度**：★★☆☆☆  
**考点**：数组统计 + 众数查找  
**GESP 级别**：2级

---

## 题目背景

古堡内散落着多种宝藏，每种宝藏用一个编号标识。探险者想找出出现次数最多的宝藏编号——这就是"众数"，它对应着最丰富的宝藏所在地。

## 题目描述

给定 $n$ 个正整数，找出出现次数最多的那个数（众数）。若有多个数出现次数相同且最多，输出其中**最小的**那个。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 1000$）。

第二行 $n$ 个整数（$1 \le a_i \le 1000$）。

## 输出格式

输出一个整数，表示众数。

## 样例

**输入 1**

```
7
3 1 2 1 3 1 2
```

**输出 1**

```
1
```

**说明**：1 出现 3 次，2 出现 2 次，3 出现 2 次，出现最多的是 1。

---

**输入 2**

```
6
4 2 4 2 3 3
```

**输出 2**

```
2
```

**说明**：2、3、4 各出现 2 次，取最小值 2。

## 提示

用计数数组统计每个数的出现次数，然后找最大值：

```cpp
int cnt[1005] = {0};
for (int i = 0; i < n; i++) cnt[a[i]]++;
int maxCnt = 0;
for (int i = 1; i <= 1000; i++) maxCnt = max(maxCnt, cnt[i]);
for (int i = 1; i <= 1000; i++) {
    if (cnt[i] == maxCnt) { cout << i; break; }
}
```

从小到大遍历，第一个满足最大次数的即为答案（保证最小）。
