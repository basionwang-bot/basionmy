# D - 山峰统计

**难度**：★★★★☆  
**考点**：二维数组 + 四方向遍历  
**GESP 级别**：4级

---

## 题目背景

古堡地图是一个 $n \times m$ 的高度网格，每个格子记录了该区域的海拔高度。探险者想统计地图上"山峰"的数量——一个格子是山峰，当且仅当它的高度**严格大于**它上下左右四个相邻格子（若相邻格子存在的话）。

## 题目描述

给定 $n \times m$ 的整数矩阵，统计其中"山峰"格子的数量。

一个格子 $(i,j)$ 是山峰，当且仅当：对于它所有在矩阵范围内的上下左右四邻居，其高度均**严格小于** $a[i][j]$。

## 输入格式

第一行两个正整数 $n$、$m$（$1 \le n, m \le 100$）。

接下来 $n$ 行，每行 $m$ 个整数（$1 \le a_{ij} \le 10000$）。

## 输出格式

输出一个整数，表示山峰的数量。

## 样例

**输入**

```
3 3
1 3 1
3 5 3
1 3 1
```

**输出**

```
1
```

**说明**：只有中心格子 $(2,2)$ 的值 5 大于其四邻居（均为 3），是唯一的山峰。

---

**输入**

```
2 2
4 3
2 1
```

**输出**

```
1
```

**说明**：格子 $(1,1)$ 的值 4，邻居为 3（右）和 2（下），均小于 4，所以是山峰（边角格子只需比已存在的邻居大）。

## 提示

```cpp
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
        bool isPeak = true;
        for (int d = 0; d < 4; d++) {
            int ni = i + dx[d], nj = j + dy[d];
            if (ni >= 0 && ni < n && nj >= 0 && nj < m) {
                if (a[ni][nj] >= a[i][j]) { isPeak = false; break; }
            }
        }
        if (isPeak) cnt++;
    }
}
```
