## 测试点 2（样例2：角点为山峰）

**输入**
```
2 2
4 3
2 1
```
**预期输出**
```
1
```

**说明**：(0,0)=4，邻居为 3(右) 和 2(下)，均 < 4，是山峰。其他格子均有邻居比自己大或相等，不是山峰。

## 测试点 4（1×1 矩阵，唯一格子是山峰）

**输入**
```
1 1
42
```
**预期输出**
```
1
```

**说明**：1×1 矩阵没有任何邻居，唯一格子满足条件（对所有存在的邻居严格大，但没有邻居，条件空真）。

---

## 测试点 5（多个山峰）

**输入**
```
3 4
9 1 8 1
1 1 1 1
7 1 6 1
```
**预期输出**
```
3
```

**说明**：(0,0)=9 邻居为1(右)和1(下)，是山峰；(0,2)=8 邻居为1,1,1,1，是山峰；(2,0)=7 邻居为1(右)和1(上)，是山峰。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int a[105][105];
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

int main() {
    int n, m;
    cin >> n >> m;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            cin >> a[i][j];
    int cnt = 0;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            bool isPeak = true;
            for (int d = 0; d < 4; d++) {
                int ni = i + dx[d], nj = j + dy[d];
                if (ni >= 0 && ni < n && nj >= 0 && nj < m) {
                    if (a[ni][nj] >= a[i][j]) { isPeak = false; break; }
                }
            }
            if (isPeak) cnt++;
        }
    }
    cout << cnt << "\n";
    return 0;
}
```

**关键点**：
1. 方向数组 `dx/dy` 枚举四个方向；
2. 越界检查：`ni >= 0 && ni < n && nj >= 0 && nj < m`；
3. 只检查存在的邻居，若没有邻居大于等于自身，则是山峰（边角格子更容易成为山峰）。
