# E - 大数相减

**难度**：★★★★★  
**考点**：高精度减法  
**GESP 级别**：5级

---

## 题目背景

古堡宝库的存量用极大的数字记录，普通 `int` 或 `long long` 无法表示。探险者需要计算两笔宝藏数量之差。

## 题目描述

给定两个非负整数 $A$ 和 $B$（以字符串形式输入），保证 $A \ge B$，输出 $A - B$ 的结果。

结果不应有前导零（若结果为 $0$，输出 `0`）。

$A$、$B$ 的位数均不超过 100 位。

## 输入格式

第一行字符串 $A$。

第二行字符串 $B$。

## 输出格式

输出一个整数字符串，表示 $A - B$。

## 样例

**输入 1**

```
1000
999
```

**输出 1**

```
1
```

---

**输入 2**

```
12345678901234567890
9876543210987654321
```

**输出 2**

```
2469135690246913569
```

---

**输入 3**

```
100
100
```

**输出 3**

```
0
```

## 提示

从低位到高位逐位相减，注意借位：

```cpp
// 将字符串存为数字数组（低位在前）
// a[0] 是 A 的个位，a[1] 是十位，...

int borrow = 0;
for (int i = 0; i < lenA; i++) {
    int diff = a[i] - (i < lenB ? b[i] : 0) - borrow;
    if (diff < 0) { diff += 10; borrow = 1; }
    else borrow = 0;
    result[i] = diff;
}
// 去掉前导零（高位方向）
int len = lenA;
while (len > 1 && result[len-1] == 0) len--;
// 从高位到低位输出
for (int i = len-1; i >= 0; i--) cout << result[i];
```
