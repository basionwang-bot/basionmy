# A - 密码校验

**难度**：★☆☆☆☆  
**考点**：条件分支（四位回文数判断）  
**GESP 级别**：1级

---

## 题目背景

古堡的第一道门用四位数字密码锁住。传说只有"回文密码"——从左读和从右读完全相同的四位数——才是有效密码。

## 题目描述

给定一个四位正整数（保证输入范围为 $1000 \le n \le 9999$），判断它是否是回文数。

四位回文数满足：个位 = 千位，且十位 = 百位。

给定 $T$ 组查询，每组输入一个四位数，若是回文数输出 `YES`，否则输出 `NO`。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 100$）。

接下来 $T$ 行，每行一个四位整数 $n$（$1000 \le n \le 9999$）。

## 输出格式

共 $T$ 行，每行输出 `YES` 或 `NO`。

## 样例

**输入**

```
4
1221
1234
9999
1001
```

**输出**

```
YES
NO
YES
YES
```

## 提示

提取各位数字：

```cpp
int d1 = n / 1000;       // 千位
int d2 = n / 100 % 10;   // 百位
int d3 = n / 10 % 10;    // 十位
int d4 = n % 10;         // 个位
if (d1 == d4 && d2 == d3) cout << "YES\n";
else cout << "NO\n";
```
