
## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int cnt[1005] = {0};
    for (int i = 0; i < n; i++) {
        int x; cin >> x;
        cnt[x]++;
    }
    int maxCnt = 0;
    for (int i = 1; i <= 1000; i++) maxCnt = max(maxCnt, cnt[i]);
    for (int i = 1; i <= 1000; i++) {
        if (cnt[i] == maxCnt) {
            cout << i << "\n";
            break;
        }
    }
    return 0;
}
```

**关键点**：从 1 到 1000 正序遍历，第一个达到最大计数的即为最小众数。
