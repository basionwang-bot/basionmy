## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int n;
        cin >> n;
        int d1 = n / 1000;
        int d2 = n / 100 % 10;
        int d3 = n / 10 % 10;
        int d4 = n % 10;
        if (d1 == d4 && d2 == d3) cout << "YES\n";
        else cout << "NO\n";
    }
    return 0;
}
```

**关键点**：四位回文等价于 千位==个位 且 百位==十位。
