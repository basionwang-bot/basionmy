# C - 叫号系统

**难度**：★★☆☆☆
**考点**：模拟、时间累加、顺序处理

---

## 题目背景

科技节的咨询台只有一个服务窗口，负责解答参观者的问题。参观者依次来到咨询台排队，每人需要不同的服务时长。工作人员想知道每位参观者需要等待多久，以及何时能完成咨询。

## 题目描述

共有 $n$ 位参观者前来咨询，**按到达时间从早到晚的顺序给出**。每位参观者有一个到达时间和所需的服务时长。

窗口服务规则：
- 若参观者到达时窗口**空闲**，则立即开始服务；
- 若参观者到达时窗口**正忙**，则等待，直到上一位服务结束后立即开始。

请计算每位参观者的等待时间和完成服务的时刻。

## 输入格式

第一行一个整数 $n$（$1 \leq n \leq 100$），表示参观者人数。

接下来 $n$ 行，每行两个整数 $a_i$ 和 $d_i$（$1 \leq a_i \leq 1000$，$1 \leq d_i \leq 100$），分别表示第 $i$ 位参观者的到达时刻和所需服务时长（单位：分钟）。

输入保证 $a_1 \leq a_2 \leq \cdots \leq a_n$（到达时间不严格递增）。

## 输出格式

共 $n$ 行，第 $i$ 行输出两个整数，用空格分隔：
- 第一个数：第 $i$ 位参观者的等待时间（分钟）；
- 第二个数：第 $i$ 位参观者完成服务的时刻。

## 样例

**输入**

```
3
0 5
3 4
8 2
```

**输出**

```
0 5
2 9
1 11
```

**说明**

| 参观者 | 到达 | 等待 | 开始服务 | 服务时长 | 完成时刻 |
|--------|------|------|----------|----------|----------|
| 1      | 0    | 0    | 0        | 5        | 5        |
| 2      | 3    | 2    | 5        | 4        | 9        |
| 3      | 8    | 1    | 9        | 2        | 11       |

- 参观者 2 在时刻 3 到达，但窗口到时刻 5 才空闲，等待 5-3=2 分钟。
- 参观者 3 在时刻 8 到达，窗口到时刻 9 才空闲，等待 9-8=1 分钟。

## 提示

- 用一个变量 `free` 记录窗口**下次空闲**的时刻，初始为 0。
- 每位参观者的开始服务时刻 = `max(到达时刻, free)`。
- 等待时间 = 开始服务时刻 - 到达时刻。
- 完成时刻 = 开始服务时刻 + 服务时长，同时更新 `free`。
