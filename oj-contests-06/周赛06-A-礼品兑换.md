# A - 礼品兑换

**难度**：★☆☆☆☆
**考点**：整除、取余、多组输入

---

## 题目背景

校园科技节上，同学们参与各类闯关活动赚取积分。活动结束后，可以用积分在礼品兑换区换取礼品。每种礼品都有固定的兑换所需积分，同学们想知道自己能兑换几件礼品，以及还剩多少积分。

## 题目描述

给定若干组测试数据，每组包含一位同学当前持有的积分 $s$ 和目标礼品每件所需积分 $p$，请计算该同学最多能兑换几件礼品，以及兑换后剩余的积分。

## 输入格式

第一行一个整数 $T$（$1 \leq T \leq 100$），表示测试组数。

接下来 $T$ 行，每行两个整数 $s$ 和 $p$（$1 \leq p \leq s \leq 10000$），分别表示当前积分和每件礼品所需积分。

## 输出格式

共 $T$ 行，每行输出两个整数，用一个空格分隔：
- 第一个数：最多能兑换的礼品件数
- 第二个数：兑换后剩余的积分

## 样例

**输入**

```
3
100 30
200 7
50 50
```

**输出**

```
3 10
28 4
1 0
```

**说明**

- 第 1 组：100 积分，每件 30 积分，兑换 3 件（共消耗 90 积分），剩余 10 积分。
- 第 2 组：200 积分，每件 7 积分，兑换 28 件（共消耗 196 积分），剩余 4 积分。
- 第 3 组：50 积分，每件 50 积分，恰好兑换 1 件，剩余 0 积分。

## 提示

- 直接使用整除（`/`）和取余（`%`）运算符即可。
- 注意 $p \leq s$，保证至少能兑换 1 件。
