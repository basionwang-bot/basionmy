## 测试点 2（n=2，m=3，最小规模）

**输入**
```
2 3
60 80 70
90 100 80
```
**预期输出**
```
2 90.00
1 70.00
```

**说明**：
- 队伍 1：去掉 80（最高）和 60（最低），剩余 70，平均 70.00。
- 队伍 2：去掉 100（最高）和 80（最低），剩余 90，平均 90.00。

## 测试点 3（得分并列，按编号升序）

**输入**
```
3 3

```
**预期输出**
```

```

**说明**：每队去掉最高和最低后，剩余 1 个数均为 80，三队并列，按编号升序输出。

## 测试点 4（评委较多，m=7）

**输入**
```
2 7
100 90 85 80 75 70 60
95 90 88 85 80 75 70
```
**预期输出**
```
2 83.60
1 80.00
```

**说明**：
- 队伍 1：去掉 100 和 60，剩余 90+85+80+75+70=400，平均 400/5=80.00。
- 队伍 2：去掉 95 和 70，剩余 90+88+85+80+75=418，平均 418/5=83.60。

## 测试点 5（含满分和 0 分极端数据）

**输入**
```
2 4
0 100 50 50
0 100 60 40
```
**预期输出**
```
1 50.00
2 50.00
```

**说明**：
- 队伍 1：去掉 100 和 0，剩余 50+50=100，平均 100/2=50.00。
- 队伍 2：去掉 100 和 0，剩余 60+40=100，平均 100/2=50.00。
- 两队并列，按编号升序输出。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <iomanip>
#include <algorithm>
using namespace std;

struct Team {
    int id;
    double avg;
};

bool cmp(Team a, Team b) {
    if (a.avg != b.avg) return a.avg > b.avg;
    return a.id < b.id;
}

int main() {
    int n, m;
    cin >> n >> m;

    int scores[20][10];
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            cin >> scores[i][j];

    Team teams[20];
    for (int i = 0; i < n; i++) {
        teams[i].id = i + 1;
        int maxv = scores[i][0], minv = scores[i][0];
        double sum = 0;
        for (int j = 0; j < m; j++) {
            sum += scores[i][j];
            if (scores[i][j] > maxv) maxv = scores[i][j];
            if (scores[i][j] < minv) minv = scores[i][j];
        }
        teams[i].avg = (sum - maxv - minv) / (m - 2);
    }

    sort(teams, teams + n, cmp);

    for (int i = 0; i < n; i++) {
        cout << teams[i].id << " "
             << fixed << setprecision(2) << teams[i].avg << endl;
    }

    return 0;
}
```
