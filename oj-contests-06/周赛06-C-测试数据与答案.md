## 测试点 3（所有人连续到达，全部需要等待）

**输入**
```
4
1 10
2 10
3 10
4 10
```
**预期输出**
```
0 11
9 21
18 31
27 41
```

**说明**：
- 参观者 1：到达 1，立即服务，完成于 11。
- 参观者 2：到达 2，等到 11 才开始，等待 9 分钟，完成于 21。
- 参观者 3：到达 3，等到 21 才开始，等待 18 分钟，完成于 31。
- 参观者 4：到达 4，等到 31 才开始，等待 27 分钟，完成于 41。

## 测试点 5（同时到达多人，按输入顺序服务）

**说明**：3 人同时在时刻 5 到达，按顺序依次服务。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[100], d[100];
    for (int i = 0; i < n; i++) cin >> a[i] >> d[i];

    int windowFree = 0; // 窗口下次空闲的时刻
    for (int i = 0; i < n; i++) {
        int startTime = (a[i] > windowFree) ? a[i] : windowFree;
        int waitTime = startTime - a[i];
        int finishTime = startTime + d[i];
        windowFree = finishTime;
        cout << waitTime << " " << finishTime << endl;
    }

    return 0;
}
```
