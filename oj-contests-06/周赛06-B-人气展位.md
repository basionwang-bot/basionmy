# B - 人气展位

**难度**：★★☆☆☆
**考点**：数组遍历、最值记录、浮点平均值、条件统计

---

## 题目背景

科技节共设有 $n$ 个参展展位，工作人员统计了每个展位的参观人数。请你根据数据，找出最受欢迎的展位，以及哪些展位的人气低于平均水平。

## 题目描述

给定 $n$ 个展位的参观人数，请计算：
1. 参观人数最多的展位编号（若多个展位并列第一，输出编号最小的）；
2. 参观人数严格低于平均参观人数的展位编号（按编号从小到大输出，若不存在则输出 `None`）；
3. 所有展位的平均参观人数（保留一位小数）。

## 输入格式

第一行一个整数 $n$（$1 \leq n \leq 100$），表示展位数量。

第二行 $n$ 个整数，第 $i$ 个整数表示第 $i$ 号展位的参观人数（$0 \leq$ 人数 $\leq 1000$）。

## 输出格式

共三行：
- 第一行：参观人数最多的展位编号；
- 第二行：所有参观人数严格低于平均值的展位编号，用空格分隔；若不存在则输出 `None`；
- 第三行：平均参观人数，保留一位小数。

## 样例

**输入**

```
5
120 85 200 85 160
```

**输出**

```
3
1 2 4
130.0
```

**说明**

- 参观人数最多的是 3 号展位（200 人）。
- 平均参观人数 = (120+85+200+85+160) / 5 = 130.0。
- 低于 130.0 的展位：1 号（120）、2 号（85）、4 号（85），依次输出编号。

## 提示

- 注意平均值是浮点数，判断"严格低于平均值"时，用浮点比较。
- 输出平均值时使用 `fixed << setprecision(1)`。
- 若所有展位人数相同，则没有展位低于平均值，第二行应输出 `None`。
