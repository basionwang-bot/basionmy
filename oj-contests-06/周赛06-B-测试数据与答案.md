## 测试点 4（最高值有并列，取编号最小）

**输入**
```
4
300 150 300 100
```
**预期输出**
```
1
4
212.5
```

**说明**：平均 = (300+150+300+100)/4 = 212.5；低于 212.5 的有 2 号（150）和 4 号（100）。

---

## 测试点 5（含 0 人展位）

**输入**
```
3
0 600 300
```
**预期输出**
```
2
1
300.0
```

**说明**：平均 = 900/3 = 300.0；低于 300.0 的有 1 号（0）。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int n;
    cin >> n;
    int a[100];
    for (int i = 0; i < n; i++) cin >> a[i];

    // 找最大值编号（并列取最小编号）
    int maxIdx = 0;
    for (int i = 1; i < n; i++) {
        if (a[i] > a[maxIdx]) maxIdx = i;
    }
    cout << maxIdx + 1 << endl;

    // 计算平均值
    double sum = 0;
    for (int i = 0; i < n; i++) sum += a[i];
    double avg = sum / n;

    // 输出低于平均值的展位编号
    bool found = false;
    for (int i = 0; i < n; i++) {
        if (a[i] < avg) {
            if (found) cout << " ";
            cout << i + 1;
            found = true;
        }
    }
    if (!found) cout << "None";
    cout << endl;

    // 输出平均值
    cout << fixed << setprecision(1) << avg << endl;

    return 0;
}
```
