# 周赛 05 - D 题：学生档案排序

**难度**：★★★☆☆  
**考点**：结构体、多关键字排序

---

## 题目背景

期末考试结束，老师需要对全班同学的成绩进行综合排名。每位同学参加了语文、数学、英语三科考试。  
**本题要求使用结构体存储每位学生的信息。**

---

## 题目描述

共有 n 名学生，每人一行：`学号 姓名 语文 数学 英语`。学号与姓名为不含空格的字符串；三科成绩为非负整数。

**总分** = 语文 + 数学 + 英语。

排序规则：

1. 按总分从高到低；  
2. 总分相同则按语文成绩从高到低；  
3. 若仍相同则按学号字典序从小到大。

输出 n 行，每行格式：`学号 姓名 总分`。

---

## 输入格式

第一行一个整数 n（1 ≤ n ≤ 100）。  
接下来 n 行，每行：`学号 姓名 语文 数学 英语`。

## 输出格式

按排序规则输出 n 行：`学号 姓名 总分`。

---

## 样例

**输入**
```
3
S001 Zhang 90 85 88
S002 Li 92 80 91
S003 Wang 90 88 85
```

**输出**
```
S002 Li 263
S001 Zhang 263
S003 Wang 263
```

**说明**：三人总分均为 263；按语文降序：S002(92) > S003(90) = S001(90)；S001 与 S003 语文相同，按学号字典序：S001 < S003，故 S001 在前。
