# 周赛 05 - A 题：糖果分配

**难度**：★☆☆☆☆  
**考点**：基础语法、整除、取余

---

## 题目背景

班里要给同学们分糖果，老师想知道每位同学能分到几颗，以及还剩几颗没法均分。

---

## 题目描述

给定 T 组数据，每组两个整数 n（人数）和 m（糖果总数），求每人分到的糖果数和剩余糖果数。

---

## 输入格式

第一行一个整数 T（1 ≤ T ≤ 100），表示查询次数。  
接下来 T 行，每行两个整数 n 和 m（1 ≤ n ≤ 1000，0 ≤ m ≤ 10000）。

## 输出格式

输出 T 行，每行两个整数，用空格隔开：每人分到的糖果数，剩余糖果数。

---

## 样例

**输入**
```
3
5 23
3 9
4 7
```

**输出**
```
4 3
3 0
1 3
```

**说明**：23 ÷ 5 = 4 余 3；9 ÷ 3 = 3 余 0；7 ÷ 4 = 1 余 3。
