# 周赛 05 - B 题：数组翻转

**难度**：★★☆☆☆  
**考点**：一维数组、循环

---

## 题目背景

小明有一排玩具，他想把前 k 个玩具移到最后面，其余的依次往前移，但不改变每组内部的顺序。

---

## 题目描述

给定长度为 n 的整数数组，将数组**向左循环移动 k 位**后输出。  
即原数组 [a₁, a₂, …, aₖ, aₖ₊₁, …, aₙ] 变成 [aₖ₊₁, …, aₙ, a₁, …, aₖ]。

---

## 输入格式

第一行两个整数 n 和 k（1 ≤ k < n ≤ 100）。  
第二行 n 个整数（每个整数绝对值不超过 1000）。

## 输出格式

一行输出移位后的 n 个整数，相邻整数间用一个空格隔开，行末无多余空格。

---

## 样例

**输入**
```
5 2
1 2 3 4 5
```

**输出**
```
3 4 5 1 2
```

**说明**：将前 2 个元素移到末尾。
