## 测试点 4（多个最长/最短，取最先出现）

**输入**
```
apple mango grape kiwi pear
```

**预期输出**
```
apple
kiwi
```

**说明**：apple、mango、grape 均长度为 5，取最先出现的 apple；kiwi、pear 均长度为 4，取最先出现的 kiwi。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <sstream>
#include <string>
using namespace std;

int main() {
    string line, word;
    getline(cin, line);
    istringstream ss(line);
    string longest, shortest;
    bool first = true;
    while (ss >> word) {
        if (first) {
            longest = shortest = word;
            first = false;
        } else {
            if ((int)word.size() > (int)longest.size()) longest = word;
            if ((int)word.size() < (int)shortest.size()) shortest = word;
        }
    }
    cout << longest << endl;
    cout << shortest << endl;
    return 0;
}
```
