## 测试点 3（含负数）

**输入**
```
6 3
-1 -2 -3 4 5 6
```

**预期输出**
```
4 5 6 -1 -2 -3
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    int a[105];
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) {
        if (i > 0) cout << " ";
        cout << a[(i + k) % n];
    }
    cout << endl;
    return 0;
}
```
