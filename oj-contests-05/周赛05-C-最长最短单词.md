# 周赛 05 - C 题：最长最短单词

**难度**：★★☆☆☆  
**考点**：字符串处理

---

## 题目背景

语文老师让同学们找出一段英文句子中最长和最短的单词，以了解词汇难度分布。

---

## 题目描述

给定一行英文句子，单词之间恰好用一个空格分隔，句子不以空格开头或结尾。  
找出**最长**的单词和**最短**的单词。若有多个长度相同的，输出**最先出现**的那个。

---

## 输入格式

一行字符串，仅包含小写英文字母和空格（长度不超过 200，至少含一个单词）。

## 输出格式

两行：  
第一行输出最长单词；  
第二行输出最短单词。  
（若最长与最短为同一个单词，两行输出相同）

---

## 样例

**输入**
```
the quick brown fox jumps over the lazy dog
```

**输出**
```
quick
the
```

**说明**：quick、brown、jumps 均长度为 5，取最先出现的 quick；the、fox、the、dog 均长度为 3，取最先出现的 the。
