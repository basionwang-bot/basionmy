## 测试点 3（总分不同）

**输入**
```
4
B01 Alice 70 80 90
B02 Bob 85 85 85
B03 Carol 60 90 70
B04 Dave 95 50 95
```

**预期输出**
```
B04 Dave 240
B02 Bob 255
B01 Alice 240
B03 Carol 220
```

**说明**：总分：B02=255，B04=240，B01=240，B03=220；B04 与 B01 总分相同(240)，语文：B04(95) > B01(70)，故 B04 在前。

---

## 测试点 4（语文与总分均相同，比学号）

**输入**
```
3
C03 Ann 80 60 60
C01 Ben 80 50 70
C02 Cen 80 70 50
```

**预期输出**
```
C01 Ben 200
C02 Cen 200
C03 Ann 200
```

**说明**：三人总分均为 200，语文均为 80，按学号字典序：C01 < C02 < C03。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

struct Student {
    string id, name;
    int chinese, math, english, total;
} s[105];

bool cmp(const Student& x, const Student& y) {
    if (x.total != y.total) return x.total > y.total;
    if (x.chinese != y.chinese) return x.chinese > y.chinese;
    return x.id < y.id;
}

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> s[i].id >> s[i].name >> s[i].chinese >> s[i].math >> s[i].english;
        s[i].total = s[i].chinese + s[i].math + s[i].english;
    }
    sort(s, s + n, cmp);
    for (int i = 0; i < n; i++)
        cout << s[i].id << " " << s[i].name << " " << s[i].total << endl;
    return 0;
}
```
