# A - 会员折扣

**难度**：★☆☆☆☆
**考点**：多分支条件、浮点输出
**GESP 级别**：1级

---

## 题目背景

某餐厅推出会员折扣活动：
- 消费金额 **小于 100 元**：原价（无折扣）；
- 消费金额 **100 元（含）至 300 元（不含）**：享 9 折优惠；
- 消费金额 **300 元（含）及以上**：享 8 折优惠。

请根据顾客的消费金额，计算实付金额。

## 题目描述

给定 $T$ 组数据，每组输入一笔消费金额 $x$（整数），输出实付金额（保留一位小数）。

## 输入格式

第一行一个整数 $T$（$1 \leq T \leq 100$）。

接下来 $T$ 行，每行一个整数 $x$（$0 \leq x \leq 10000$）。

## 输出格式

共 $T$ 行，每行一个数，表示该组的实付金额（保留一位小数）。

## 样例

**输入**

```
3
50
150
500
```

**输出**

```
50.0
135.0
400.0
```

**说明**

| 消费 | 折扣 | 实付 |
|------|------|------|
| 50 | 无 | 50.0 |
| 150 | 9折 | 150×0.9=135.0 |
| 500 | 8折 | 500×0.8=400.0 |

## 提示

- 用 `if / else if / else` 分段判断折扣率；
- 输出时使用 `fixed << setprecision(1)`。

```cpp
double price;
if (x < 100) price = x;
else if (x < 300) price = x * 0.9;
else price = x * 0.8;
```
