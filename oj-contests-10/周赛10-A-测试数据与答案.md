# A - 会员折扣 · 测试数据与答案

---

## 测试点 1（样例）

**输入**
```
3
50
150
500
```
**预期输出**
```
50.0
135.0
400.0
```

---

## 测试点 2（边界值：99/100/300）

**输入**
```
3
99
100
300
```
**预期输出**
```
99.0
90.0
240.0
```

**说明**：99 < 100 无折扣；100 ≥ 100 享9折，100×0.9=90.0；300 ≥ 300 享8折，300×0.8=240.0。

---

## 测试点 4（最大金额）

**输入**
```
2
10000
299
```

```
8000.0
269.1
```

**说明**：10000×0.8=8000.0；299×0.9=269.1。

---

## 测试点 5（混合多组）

**输入**
```
5
80
200
299
300
1000
```
**预期输出**
```
80.0
180.0
269.1
240.0
800.0
```

**说明**：80无折扣；200×0.9=180.0；299×0.9=269.1；300×0.8=240.0；1000×0.8=800.0。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int x;
        cin >> x;
        double price;
        if (x < 100) price = x;
        else if (x < 300) price = x * 0.9;
        else price = x * 0.8;
        cout << fixed << setprecision(1) << price << endl;
    }
    return 0;
}
```
