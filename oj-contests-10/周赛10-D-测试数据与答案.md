## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

void hanoi(int n, char from, char via, char to) {
    if (n == 0) return;
    hanoi(n - 1, from, to, via);
    cout << from << "->" << to << endl;
    hanoi(n - 1, via, from, to);
}

int main() {
    int n;
    cin >> n;
    hanoi(n, 'A', 'B', 'C');
    return 0;
}
```

**关键点**：
1. 递归出口：`n == 0` 时不做任何操作；
2. 三步法：先搬上面 $n-1$ 个到中转柱，再搬最底下的盘子到目标柱，最后把 $n-1$ 个从中转柱搬到目标柱；
3. 时间复杂度 $O(2^n)$，所以题目限制 $n \leq 6$ 防止输出过长。
