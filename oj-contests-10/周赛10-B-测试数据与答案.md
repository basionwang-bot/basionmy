# B - 连续好评 · 测试数据与答案

---

## 测试点 1（样例）

**输入**
```
8
3 4 5 4 2 5 5 3
```
**预期输出**
```
3
```

---

## 测试点 2（全部好评）

**输入**
```
5
4 5 5 4 5
```
**预期输出**
```
5
```

---

## 测试点 3（无人好评）

**输入**
```
4
0 1 2 3
```
**预期输出**
```
0
```

**说明**：所有评分均 < 4，无好评。

---

## 测试点 4（首尾好评在两端）

**输入**
```
6
5 5 0 0 5 5
```
**预期输出**
```
2
```

**说明**：第 1-2 天好评 2 天；第 5-6 天好评 2 天。最长 = 2。

---

## 测试点 5（n=1，临界值）

**输入**
```
1
4
```
**预期输出**
```
1
```

**说明**：1 天且评分 4 = 1 天好评。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int cur = 0, ans = 0;
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        if (x >= 4) cur++;
        else cur = 0;
        if (cur > ans) ans = cur;
    }

    cout << ans << endl;
    return 0;
}
```
