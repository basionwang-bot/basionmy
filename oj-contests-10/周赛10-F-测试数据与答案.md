# F - 黄金档期 · 测试数据与答案

## 测试点 1（样例）

**输入**
```
6
-2 11 -4 13 -5 -2
```
**预期输出**
```
20
```

**说明**：选 [11, -4, 13]，和 = 20。

## 测试点 2（全部为正，整段最优）

**输入**
```
5
1 2 3 4 5
```
**预期输出**
```
15
```

**说明**：全部为正数，整段相加最大；1+2+3+4+5=15。

## 测试点 3（全部为负，必须选一个）

## 测试点 5（经典负正混合）

**输入**
```
9
-2 1 -3 4 -1 2 1 -5 4
```
**预期输出**
```
6
```

**说明**：选子段 [4, -1, 2, 1]，和 = 4-1+2+1 = 6。

DP 过程：
| i | a[i] | dp | ans |
|---|------|-----|-----|
| 1 | -2 | -2 | -2 |
| 2 | 1 | 1 | 1 |
| 3 | -3 | -2 | 1 |
| 4 | 4 | 4 | 4 |
| 5 | -1 | 3 | 4 |
| 6 | 2 | 5 | 5 |
| 7 | 1 | 6 | 6 |
| 8 | -5 | 1 | 6 |
| 9 | 4 | 5 | 6 |

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[1000];
    for (int i = 0; i < n; i++) cin >> a[i];

    int dp = a[0], ans = a[0];
    for (int i = 1; i < n; i++) {
        dp = max(a[i], dp + a[i]);
        ans = max(ans, dp);
    }

    cout << ans << endl;
    return 0;
}
```

**关键点**：
1. `dp` 只需一个变量（滚动）即可，因为只依赖前一项；
2. 初值 `dp = a[0]`、`ans = a[0]`，从第 2 个数开始遍历；
3. 注意 `ans` 不能初始化为 0，必须用 $a_0$ 或 `INT_MIN`，否则全负数据会错。
