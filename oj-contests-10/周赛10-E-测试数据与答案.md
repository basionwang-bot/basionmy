# E - 账本累计 · 测试数据与答案

---

## 测试点 1（样例）

**输入**
```
3
123456789
987654321
1
```
**预期输出**
```
1111111111
```

**说明**：123456789 + 987654321 = 1111111110；再 +1 = 1111111111。

## 测试点 3（进位边界）

**输入**
```
2
99
1
```
**预期输出**
```
100
```

**说明**：99 + 1 = 100，最高位产生新进位。

## 测试点 4（长大数相加）

**输入**
```
2
99999999999999999999999999999999999999999999999999
1
```
**预期输出**
```
100000000000000000000000000000000000000000000000000
```

**说明**：50 个 9 加 1，得到 1 后跟 50 个 0。

## 测试点 5（多个大数累加）

**输入**
```
5
12345678901234567890
98765432109876543210
11111111111111111111
22222222222222222222
33333333333333333333
```
**预期输出**
```
177777777677777777766
```

**说明**：逐次累加结果如下表。
- 12345678901234567890 + 98765432109876543210 = 111111111011111111100
- 111111111011111111100 + 11111111111111111111 = 122222222122222222211
- 122222222122222222211 + 22222222222222222222 = 144444444344444444433
- 144444444344444444433 + 33333333333333333333 = 177777777677777777766

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

string add(string a, string b) {
    if (a.size() < b.size()) swap(a, b);
    int n = a.size(), m = b.size();
    string result(n + 1, '0');
    int carry = 0;
    for (int i = 0; i < n; i++) {
        int x = a[n - 1 - i] - '0';
        int y = (i < m) ? b[m - 1 - i] - '0' : 0;
        int s = x + y + carry;
        result[n - i] = (s % 10) + '0';
        carry = s / 10;
    }
    result[0] = carry + '0';
    if (result[0] == '0') result = result.substr(1);
    return result;
}

int main() {
    int n;
    cin >> n;
    string sum = "0";
    for (int i = 0; i < n; i++) {
        string x;
        cin >> x;
        sum = add(sum, x);
    }
    cout << sum << endl;
    return 0;
}
```
