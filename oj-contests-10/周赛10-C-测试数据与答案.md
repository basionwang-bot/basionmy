## 测试点 5（达标线为0，全部输出，含价格并列）

**输入**
```
4 0

2 50 30
3 50 50
4 80 40
```
**预期输出**
```
3 50 50
4 80 40
2 50 30
1 100 30
```

**说明**：销量降序：50 > 40 > 30；销量同为 30 时，2(50) 排在 1(100) 前面。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

struct Dish {
    int id, price, sales;
};

bool cmp(Dish a, Dish b) {
    if (a.sales != b.sales) return a.sales > b.sales;
    return a.price < b.price;
}

int main() {
    int n, X;
    cin >> n >> X;

    Dish d[100];
    for (int i = 0; i < n; i++)
        cin >> d[i].id >> d[i].price >> d[i].sales;

    Dish ok[100];
    int cnt = 0;
    for (int i = 0; i < n; i++)
        if (d[i].sales >= X)
            ok[cnt++] = d[i];

    if (cnt == 0) {
        cout << "None" << endl;
    } else {
        sort(ok, ok + cnt, cmp);
        for (int i = 0; i < cnt; i++)
            cout << ok[i].id << " " << ok[i].price << " " << ok[i].sales << endl;
    }

    return 0;
}
```
