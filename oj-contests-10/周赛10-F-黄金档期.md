# F - 黄金档期

**难度**：★★★☆☆
**考点**：动态规划入门（最大子段和）
**GESP 级别**：6级

---

## 题目背景

餐厅记录了**连续 $n$ 天**的盈利情况（每天的盈利可能为负，表示当天亏损）。老板想找出一段连续天数（**至少一天**），使得这段时间的总盈利**最大**。

## 题目描述

给定长度为 $n$ 的整数序列 $a_1, a_2, \ldots, a_n$（可正可负可零），找出一个连续子段 $a_l, a_{l+1}, \ldots, a_r$（$1 \leq l \leq r \leq n$），使得 $\sum_{i=l}^{r} a_i$ 最大。输出这个最大和。

## 输入格式

第一行一个整数 $n$（$1 \leq n \leq 1000$）。

第二行 $n$ 个整数 $a_i$（$-10000 \leq a_i \leq 10000$）。

## 输出格式

一个整数，最大连续子段和。

## 样例

**输入**

```
6
-2 11 -4 13 -5 -2
```

**输出**

```
20
```

**说明**

| 子段选法 | 和 |
|----------|-----|
| 仅 [11] | 11 |
| [11, -4, 13] | 20 ✓ |
| [-2, 11, -4, 13] | 18 |
| [11, -4, 13, -5] | 15 |

最大的是子段 $a_2, a_3, a_4 = 11 + (-4) + 13 = 20$。

## 提示

**动态规划经典模型**：

设 `dp[i]` 表示**以第 $i$ 天为结尾**的最大子段和。状态转移：

$$dp[i] = \max(a_i,\ dp[i-1] + a_i)$$

含义：以 $a_i$ 结尾的最大子段，要么只取 $a_i$ 自己，要么把 $a_i$ 接在以 $a_{i-1}$ 结尾的最大子段之后。

最终答案 = $\max\{dp[1], dp[2], \ldots, dp[n]\}$。

```cpp
int dp = a[0], ans = a[0];
for (int i = 1; i < n; i++) {
    dp = max(a[i], dp + a[i]);
    ans = max(ans, dp);
}
```

**注意**：题目要求至少选一个数，所以即使所有数都是负数，也要选其中最大的那个负数（不能视为空段输出 0）。
