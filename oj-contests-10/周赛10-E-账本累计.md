# E - 账本累计

**难度**：★★★☆☆
**考点**：高精度加法（用字符串/数组模拟）
**GESP 级别**：5级

---

## 题目背景

餐厅老板每天结账后，会把当日营业额记入账本。由于经营多年、规模庞大，账本里出现的数字可能远远超出 `long long` 范围（最多达 100 位）。请你帮老板算出 $n$ 天累计的总营业额。

## 题目描述

给定 $n$ 个非负整数（每个数最多 100 位），计算它们的总和并输出。

## 输入格式

第一行一个整数 $n$（$1 \leq n \leq 100$）。

接下来 $n$ 行，每行一个非负整数（无前导 0；当该数为 0 时，恰好为字符 `0`），每个数的位数不超过 100。

## 输出格式

一个整数，所有 $n$ 个数的累计和（无前导 0）。

## 样例

**输入**

```
3
123456789
987654321
1
```

**输出**

```
1111111111
```

**说明**：
- 123456789 + 987654321 = 1111111110
- 1111111110 + 1 = 1111111111

## 提示

由于数字位数远超 `long long`，必须用**高精度**实现。常用做法：

**方法一（字符串实现）**：
1. 把当前累计和与新读入的数都看作字符串；
2. 从两个字符串末尾开始逐位对齐相加，处理进位；
3. 长度不一致时，短的高位补 0。

```cpp
string add(string a, string b) {
    if (a.size() < b.size()) swap(a, b);
    int n = a.size(), m = b.size();
    string result(n + 1, '0');
    int carry = 0;
    for (int i = 0; i < n; i++) {
        int x = a[n - 1 - i] - '0';
        int y = (i < m) ? b[m - 1 - i] - '0' : 0;
        int s = x + y + carry;
        result[n - i] = (s % 10) + '0';
        carry = s / 10;
    }
    result[0] = carry + '0';
    if (result[0] == '0') result = result.substr(1);
    return result;
}
```

**方法二（整型数组倒序）**：用 `int a[110]` 存每一位，下标 `0` 表示个位，逐位加，最后从高位到低位输出。

注意点：
- 最后可能产生**新的最高位进位**（如 `99 + 1 = 100`，长度从 2 变 3）；
- 输出时去掉前导 0（除非整个结果就是 0）。
