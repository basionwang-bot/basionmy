# B - 连续好评

**难度**：★★☆☆☆
**考点**：一维数组、最长连续段统计
**GESP 级别**：2级

---

## 题目背景

某餐厅连续 $n$ 天接受顾客评分（每天给出 0-5 的整数评分），其中 **评分 $\geq 4$** 视为好评。老板想知道这 $n$ 天里**最长**的一段连续好评天数有多长。

## 题目描述

给定 $n$ 天的评分序列，求最长连续好评（评分 $\geq 4$）的天数。

## 输入格式

第一行一个整数 $n$（$1 \leq n \leq 1000$）。

第二行 $n$ 个整数，每个整数 $a_i$（$0 \leq a_i \leq 5$），表示第 $i$ 天的评分。

## 输出格式

一个整数，最长连续好评天数。若所有天均无好评，输出 `0`。

## 样例

**输入**

```
8
3 4 5 4 2 5 5 3
```

**输出**

```
3
```

**说明**

| 天 | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 |
|----|---|---|---|---|---|---|---|---|
| 评分 | 3 | 4 | 5 | 4 | 2 | 5 | 5 | 3 |
| 是否好评 | ✗ | ✓ | ✓ | ✓ | ✗ | ✓ | ✓ | ✗ |

第 2-4 天连续好评 3 天；第 6-7 天连续好评 2 天。最长 = 3。

## 提示

- 用一个变量 `cur` 记录当前连续好评长度，遍历到好评时 `cur++`，否则 `cur=0`；
- 用 `ans` 记录历史最大值，`ans = max(ans, cur)`。

```cpp
int cur = 0, ans = 0;
for (int i = 0; i < n; i++) {
    if (a[i] >= 4) cur++;
    else cur = 0;
    if (cur > ans) ans = cur;
}
```
