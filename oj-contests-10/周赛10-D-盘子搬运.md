# D - 盘子搬运

**难度**：★★★☆☆
**考点**：递归（汉诺塔模型）
**GESP 级别**：4级

---

## 题目背景

餐厅打烊后，厨师需要把厨房 A 操作台上一摞 $n$ 个盘子（从下到大、从上到小）整理到 C 操作台。中间有一个 B 操作台可以临时周转。规则：
1. 一次只能搬 1 个盘子；
2. 任何时候，**大盘不能放在小盘上面**。

请输出按规则搬运的全部步骤。

## 题目描述

给定整数 $n$，输出把 $n$ 个盘子从 A 搬到 C 的完整搬运序列，每步形如 `X->Y`，表示从 $X$ 操作台搬一个盘子到 $Y$ 操作台。要求步数最少（恰为 $2^n - 1$ 步）。

## 输入格式

一个整数 $n$（$1 \leq n \leq 6$）。

## 输出格式

共 $2^n - 1$ 行，每行一步操作 `X->Y`，按搬运顺序输出。

## 样例 1

**输入**

```
2
```

**输出**

```
A->B
A->C
B->C
```

**说明**：先把上面的小盘搬到 B；再把大盘搬到 C；最后把 B 上的小盘放到 C。共 $2^2-1=3$ 步。

## 样例 2

**输入**

```
3
```

**输出**

```
A->C
A->B
C->B
A->C
B->A
B->C
A->C
```

共 $2^3-1=7$ 步。

## 提示

**经典递归思路**：要把 $n$ 个盘子从 `from` 通过 `via` 搬到 `to`，
1. 先把上面 $n-1$ 个借助 `to` 搬到 `via`；
2. 把最底下那个最大盘从 `from` 搬到 `to`（输出 `from->to`）；
3. 再把 `via` 上的 $n-1$ 个借助 `from` 搬到 `to`。

```cpp
void hanoi(int n, char from, char via, char to) {
    if (n == 0) return;
    hanoi(n - 1, from, to, via);
    cout << from << "->" << to << endl;
    hanoi(n - 1, via, from, to);
}
```

调用：`hanoi(n, 'A', 'B', 'C');`
