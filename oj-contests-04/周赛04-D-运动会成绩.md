# 周赛 04 - D 题：运动会成绩

**难度**：★★★☆☆  
**考点**：结构体、多关键字排序

---

## 题目背景

学校运动会结束后，需要将各班级运动员的成绩汇总公示。每名运动员有：编号、姓名、短跑得分、跳远得分。  
**总分** = 短跑得分 + 跳远得分。请按规则排序后输出。

**本题要求使用结构体存储每名运动员的信息。**

---

## 题目描述

共有 n 名运动员，每人一行：`编号 姓名 短跑得分 跳远得分`。编号与姓名为不含空格的字符串；两个得分为非负整数。

排序规则：

1. 按总分从高到低；  
2. 总分相同则按短跑得分从高到低；  
3. 若仍相同则按编号字典序从小到大。

输出 n 行，每行格式：`编号 姓名 总分`。

---

## 输入格式

第一行一个整数 n（1 ≤ n ≤ 100）。  
接下来 n 行，每行：`编号 姓名 短跑 跳远`。

## 输出格式

按排序规则输出 n 行：`编号 姓名 总分`。

---

## 样例

**输入**
```
3
A01 Zhang 30 40
A02 Li 35 35
A03 Wang 40 30
```

**输出**
```
A03 Wang 70
A02 Li 70
A01 Zhang 70
```

**说明**：三人总分均为 70；按短跑分从高到低：A03(40) > A02(35) > A01(30)。
