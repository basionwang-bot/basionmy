# 周赛 04 - B 题：累计积分

**难度**：★☆☆☆☆  
**考点**：一维数组、前缀和

---

## 题目背景

小强在玩一款闯关游戏，每一关结束后会得到一个整数积分（可为负，表示扣分）。  
他想知道：打完第 1 关、第 2 关、……、第 n 关后，**累计积分**分别是多少。

---

## 题目描述

给定 n 个整数，表示第 1 关到第 n 关各关获得的积分。  
请输出 n 行：第 i 行表示前 i 关的累计积分之和（即前缀和）。

---

## 输入格式

第一行一个整数 n（1 ≤ n ≤ 1000）。  
第二行 n 个整数，用空格分隔，每个整数绝对值不超过 10000。

## 输出格式

输出 n 行，第 i 行一个整数，表示前 i 个数的和。

---

## 样例

**输入**
```
4
3 -1 5 2
```

**输出**
```
3
2
7
9
```

**说明**：前 1 个数和为 3；前 2 个为 3+(-1)=2；前 3 个为 2+5=7；前 4 个为 7+2=9。
