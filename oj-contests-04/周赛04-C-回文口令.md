# 周赛 04 - C 题：回文口令

**难度**：★★☆☆☆  
**考点**：字符串、双指针

---

## 题目背景

小强注册了一个学习网站，系统要求设置「回文口令」：从左往右读和从右往左读完全一样（忽略大小写在本题中不适用——本题只含小写字母）。  
给定一串只含小写字母的字符串，判断它是否为回文。

---

## 题目描述

给定一行字符串 s，仅由小写字母组成。若 s 是回文串，输出 `YES`；否则输出 `NO`。

**回文串**：正读和反读相同的字符串，如 `aba`、`aa`、`a`。

---

## 输入格式

一行字符串 s，长度 1 ≤ |s| ≤ 1000，只包含小写字母。

## 输出格式

一行，`YES` 或 `NO`（大写）。

---

## 样例

**输入**
```
aba
```

**输出**
```
YES
```

**输入**
```
abc
```

**输出**
```
NO
```
