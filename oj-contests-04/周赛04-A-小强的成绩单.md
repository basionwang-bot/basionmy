# 周赛 04 - A 题：小强的成绩单

**难度**：★☆☆☆☆  
**考点**：基础语法、循环、分支

---

## 题目背景

期末考试后，小强想快速知道自己每科分数对应的等级。学校规定：  
90 分及以上为 A，80～89 为 B，70～79 为 C，60～69 为 D，60 分以下为 F。  
请你帮他写一个小程序：输入多科分数，每行输出对应的等级字母。

---

## 题目描述

给定 T 组数据，每组一个整数分数 score（0 ≤ score ≤ 100），按下面规则输出等级：

- A：score ≥ 90  
- B：80 ≤ score ≤ 89  
- C：70 ≤ score ≤ 79  
- D：60 ≤ score ≤ 69  
- F：score < 60  

---

## 输入格式

第一行一个整数 T（1 ≤ T ≤ 100），表示查询次数。  
接下来 T 行，每行一个整数 score。

## 输出格式

输出 T 行，每行一个大写字母 A、B、C、D 或 F。

---

## 样例

**输入**
```
5
95
85
72
60
55
```

**输出**
```
A
B
C
D
F
```
