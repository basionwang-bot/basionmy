# D - 路径计数·测试数据与答案

## 测试点 1（样例 1）

**输入**
```
3 3
```

**输出**
```
6
```

**说明**：$\binom{4}{2}=6$，共需向右 2 步、向下 2 步，共 6 种排列。

---

## 测试点 2（样例 2，单行）

**输入**
```
1 5
```

**输出**
```
1
```

**说明**：只有一行，只能一直向右，仅 1 条路径。

---

## 测试点 3（样例 3）

**输入**
```
3 7
```

**输出**
```
28
```

**说明**：$\binom{8}{2}=28$。

---

## 测试点 4（单列）

**输入**
```
5 1
```

**输出**
```
1
```

**说明**：只有一列，只能一直向下，仅 1 条路径。

---

## 测试点 5（$2 \times 2$ 小网格）

**输入**
```
2 2
```

**输出**
```
2
```

**说明**：右下或下右，共 2 条。

---

## 测试点 6（$5 \times 5$）

**输入**
```
5 5
```

**输出**
```
70
```

**说明**：$\binom{8}{4}=70$。

---

## 测试点 7（最大规模 $18 \times 18$）

**输入**
```
18 18
```

**输出**
```
2333606220
```

**说明**：$\binom{34}{17}=2333606220$，超过 int 范围（约 $2.3 \times 10^9$），需用 `long long`。

---

## 参考代码（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n, m; cin >> n >> m;
    long long dp[20][20] = {};
    for (int i = 1; i <= n; i++) dp[i][1] = 1;
    for (int j = 1; j <= m; j++) dp[1][j] = 1;
    for (int i = 2; i <= n; i++)
        for (int j = 2; j <= m; j++)
            dp[i][j] = dp[i-1][j] + dp[i][j-1];
    cout << dp[n][m] << "\n";
}
```
