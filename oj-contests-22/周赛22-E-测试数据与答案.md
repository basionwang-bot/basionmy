# E - 滑窗最值·测试数据与答案

## 测试点 1（样例 1）

**输入**
```
8 3
1 3 -1 -3 5 3 6 7
```

**输出**
```
3
3
5
5
6
7
```

**说明**：6 个窗口的最大值依次为 3, 3, 5, 5, 6, 7。

---

## 测试点 2（样例 2，k=1）

**输入**
```
4 1
3 1 4 1
```

**输出**
```
3
1
4
1
```

**说明**：k=1 时每个元素自身即为窗口最大值。

---

## 测试点 3（k=n，整个数组为一个窗口）

**输入**
```
5 5
1 3 2 5 4
```

**输出**
```
5
```

**说明**：仅一个窗口，最大值为 5。

---

## 测试点 4（严格递减序列）

**输入**
```
5 3
9 8 7 6 5
```

**输出**
```
9
8
7
```

**说明**：
- [9,8,7]→9；[8,7,6]→8；[7,6,5]→7。
- 单调队列队头始终是窗口最左的最大值。

---

## 测试点 5（含负数，严格递增序列）

**输入**
```
6 2
-5 -3 -1 0 2 4
```

**输出**
```
-3
-1
0
2
4
```

**说明**：每次滑入的新元素均更大，队列始终只保留最新元素为最大值。

---

## 测试点 6（大范围值，k=3）

**输入**
```
7 3
1000000000 -1000000000 999999999 -999999999 0 500000000 -500000000
```

**输出**
```
1000000000
999999999
999999999
500000000
500000000
```

**说明**：验证 $a_i$ 取极值 $\pm 10^9$ 时程序仍正确。

---

## 参考代码（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n, k; cin >> n >> k;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    deque<int> dq;  // 存下标，队头为当前窗口最大值的下标
    for (int i = 0; i < n; i++) {
        // 移除队尾所有不大于 a[i] 的元素
        while (!dq.empty() && a[dq.back()] <= a[i]) dq.pop_back();
        dq.push_back(i);
        // 移除超出窗口左边界的队头
        if (dq.front() < i - k + 1) dq.pop_front();
        // 窗口满时输出
        if (i >= k - 1) cout << a[dq.front()] << "\n";
    }
}
```
