# B - 关键证据

**难度**：★★☆☆☆  
**考点**：数组遍历（局部极大值统计）  
**GESP 级别**：2级

---

## 题目背景

侦探在案件现场收集到 $n$ 条证据，每条证据有一个强度值。"关键证据"是指强度值**严格大于**其相邻证据的那条——它在局部是最突出的。

## 题目描述

给定 $n$ 个整数，统计**关键证据**（局部极大值）的数量。

判断规则：
- 位置 $i$（$0 < i < n-1$）是关键证据，当且仅当 $a[i] > a[i-1]$ 且 $a[i] > a[i+1]$
- 首位置 $i=0$ 是关键证据，当且仅当 $a[0] > a[1]$（只比右邻居）
- 末位置 $i=n-1$ 是关键证据，当且仅当 $a[n-1] > a[n-2]$（只比左邻居）

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 1000$）。

第二行 $n$ 个整数（$1 \le a_i \le 10^6$）。

## 输出格式

输出一个整数，表示关键证据的数量。

## 样例

**输入 1**

```
8
1 3 2 5 4 2 7 8
```

**输出 1**

```
3
```

**说明**：
- $a[1]=3$：大于左邻 1 和右邻 2 → 关键证据 ✓
- $a[3]=5$：大于左邻 2 和右邻 4 → 关键证据 ✓
- $a[7]=8$：末位，大于左邻 7 → 关键证据 ✓

---

**输入 2**

```
3
5 5 5
```

**输出 2**

```
0
```

**说明**：全部相等，无严格大于的。

## 提示

```cpp
int cnt = 0;
for (int i = 0; i < n; i++) {
    bool left_ok  = (i == 0)   || a[i] > a[i-1];
    bool right_ok = (i == n-1) || a[i] > a[i+1];
    if (left_ok && right_ok) cnt++;
}
```
