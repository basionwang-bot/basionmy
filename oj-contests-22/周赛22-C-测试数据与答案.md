# C - 案卷整理·测试数据与答案

## 测试点 1（样例）

**输入**
```
5
302 Alice CRITICAL
105 Bob NORMAL
201 Carol CRITICAL
408 Dave LOW
103 Eve CRITICAL
```

**输出**
```
Eve
Carol
Alice
```

**说明**：筛选 CRITICAL：103(Eve)、201(Carol)、302(Alice)，按编号升序输出姓名。

## 参考代码（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;
struct Case {
    int id;
    string name, level;
};
int main() {
    int n; cin >> n;
    vector<Case> cases(n);
    for (int i = 0; i < n; i++)
        cin >> cases[i].id >> cases[i].name >> cases[i].level;
    vector<Case> res;
    for (auto& c : cases)
        if (c.level == "CRITICAL") res.push_back(c);
    if (res.empty()) { cout << "none\n"; return 0; }
    sort(res.begin(), res.end(), [](const Case& a, const Case& b){
        return a.id < b.id;
    });
    for (auto& c : res) cout << c.name << "\n";
}
```
