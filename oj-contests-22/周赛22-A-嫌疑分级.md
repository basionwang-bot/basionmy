# A - 嫌疑分级

**难度**：★☆☆☆☆  
**考点**：条件分支、多组输入  
**GESP 级别**：1级

---

## 题目背景

侦探事务所对每名嫌疑人进行评分，根据证据强度给出嫌疑等级。

## 题目描述

给定证据分值 $e$（整数），按以下规则输出嫌疑等级：

- $e \ge 80$：输出 `GUILTY`
- $50 \le e < 80$：输出 `SUSPECT`
- $e < 50$：输出 `INNOCENT`

给定 $T$ 组查询，每组输出对应等级。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 100$）。

接下来 $T$ 行，每行一个整数 $e$（$0 \le e \le 100$）。

## 输出格式

共 $T$ 行，每行输出对应等级字符串。

## 样例

**输入**

```
4
90
65
50
30
```

**输出**

```
GUILTY
SUSPECT
SUSPECT
INNOCENT
```

**说明**：$e=50$ 属于 SUSPECT（$\ge 50$）。
