# B - 关键证据·测试数据与答案

## 测试点 1（样例 1）

**输入**
```
8
1 3 2 5 4 2 7 8
```

**输出**
```
3
```

**说明**：
- $a[1]=3$：大于左邻 1 和右邻 2 ✓
- $a[3]=5$：大于左邻 2 和右邻 4 ✓
- $a[7]=8$：末位，大于左邻 7 ✓

---

## 测试点 2（样例 2，全相等）

**输入**
```
3
5 5 5
```

**输出**
```
0
```

---

## 测试点 3（n=1，单元素）

**输入**
```
1
7
```

**输出**
```
1
```

**说明**：单个元素既是首位也是末位，无需与任何邻居比较，必为关键证据。

---

## 测试点 4（严格递减，仅首位为极大）

**输入**
```
5
9 7 5 3 1
```

**输出**
```
1
```

**说明**：只有 $a[0]=9$（首位，大于右邻 7）满足条件，末位 1 是最小值不满足。

---

## 测试点 5（单峰序列）

**输入**
```
5
1 2 3 2 1
```

**输出**
```
1
```

**说明**：只有 $a[2]=3$ 同时大于左邻 2 和右邻 2。

---

## 测试点 6（多极值，含首尾）

**输入**
```
6
3 1 4 1 5 2
```

**输出**
```
3
```

**说明**：
- $a[0]=3$：首位，3>1 ✓
- $a[2]=4$：4>1 且 4>1 ✓
- $a[4]=5$：5>1 且 5>2 ✓

---

## 参考代码（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n; cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    int cnt = 0;
    for (int i = 0; i < n; i++) {
        bool left_ok  = (i == 0)   || a[i] > a[i-1];
        bool right_ok = (i == n-1) || a[i] > a[i+1];
        if (left_ok && right_ok) cnt++;
    }
    cout << cnt << "\n";
}
```
