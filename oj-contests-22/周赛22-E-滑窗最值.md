# E - 滑窗最值

**难度**：★★★★★  
**考点**：单调队列（deque）实现滑动窗口最大值  
**GESP 级别**：5级/6级

---

## 题目背景

侦探事务所的监控系统每秒记录一个信号强度，需要对每个长度为 $k$ 的时间窗口计算信号最大值，以便快速定位异常时刻。

## 题目描述

给定 $n$ 个整数和窗口大小 $k$，对每个长度为 $k$ 的连续子数组，输出其**最大值**。共输出 $n-k+1$ 个最大值。

## 输入格式

第一行两个正整数 $n$、$k$（$1 \le k \le n \le 10^5$）。

第二行 $n$ 个整数（$-10^9 \le a_i \le 10^9$）。

## 输出格式

输出 $n-k+1$ 个整数，每行一个，表示每个窗口的最大值。

## 样例

**输入 1**

```
8 3
1 3 -1 -3 5 3 6 7
```

**输出 1**

```
3
3
5
5
6
7
```

**说明**：
| 窗口 | 内容 | 最大值 |
|------|------|--------|
| [0,2] | 1,3,-1 | 3 |
| [1,3] | 3,-1,-3 | 3 |
| [2,4] | -1,-3,5 | 5 |
| [3,5] | -3,5,3 | 5 |
| [4,6] | 5,3,6 | 6 |
| [5,7] | 3,6,7 | 7 |

---

**输入 2**

```
4 1
3 1 4 1
```

**输出 2**

```
3
1
4
1
```

## 提示

朴素 $O(nk)$ 双层循环在 $n=10^5, k=10^4$ 时会超时。用**单调递减队列（deque 存下标）**实现 $O(n)$：

```cpp
deque<int> dq;  // 存下标，队头为当前窗口最大值的下标
for (int i = 0; i < n; i++) {
    // 移除队尾所有小于 a[i] 的元素（保持队列单调递减）
    while (!dq.empty() && a[dq.back()] <= a[i]) dq.pop_back();
    dq.push_back(i);
    // 移除超出窗口左边界的队头
    if (dq.front() < i - k + 1) dq.pop_front();
    // 窗口满时输出
    if (i >= k - 1) cout << a[dq.front()] << "\n";
}
```

**单调队列的核心思想**：队列中存的下标对应的值单调递减，队头始终是当前窗口的最大值。
