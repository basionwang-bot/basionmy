# A - 嫌疑分级·测试数据与答案

## 测试点 1（样例）

**输入**
```
5
85 50 30 79 65
```

**输出**
```
GUILTY
SUSPECT
INNOCENT
SUSPECT
SUSPECT
```

**说明**：85≥80→GUILTY；50≥50→SUSPECT；30<50→INNOCENT；79≥50→SUSPECT；65≥50→SUSPECT。

---

## 测试点 2（边界值）

**输入**
```
4
80 50 49 79
```

**输出**
```
GUILTY
SUSPECT
INNOCENT
SUSPECT
```

**说明**：80 恰好是 GUILTY 下界；50 恰好是 SUSPECT 下界；49 是 INNOCENT 上界；79 是 SUSPECT 上界。

---

## 测试点 3（全部 GUILTY）

**输入**
```
3
100 80 95
```

**输出**
```
GUILTY
GUILTY
GUILTY
```

---

## 测试点 4（全部 INNOCENT）

**输入**
```
3
0 25 49
```

**输出**
```
INNOCENT
INNOCENT
INNOCENT
```

---

## 测试点 5（混合极值）

**输入**
```
6
0 49 50 79 80 100
```

**输出**
```
INNOCENT
INNOCENT
SUSPECT
SUSPECT
GUILTY
GUILTY
```

**说明**：依次验证 0、49 为 INNOCENT；50、79 为 SUSPECT；80、100 为 GUILTY。

---

## 参考代码（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n; cin >> n;
    while (n--) {
        int e; cin >> e;
        if (e >= 80) cout << "GUILTY\n";
        else if (e >= 50) cout << "SUSPECT\n";
        else cout << "INNOCENT\n";
    }
}
```
