# C - 案卷整理

**难度**：★★★☆☆  
**考点**：结构体 + 条件筛选 + 排序  
**GESP 级别**：3级

---

## 题目背景

侦探事务所存有大量案卷，每份案卷都有编号、嫌疑人姓名和密级。探员只能查阅密级为"CRITICAL"的案卷，并按案件编号整理目录。

## 题目描述

给定 $n$ 份案卷，每份包含案件编号（整数）、嫌疑人姓名和密级（字符串）。

筛选密级为 `CRITICAL` 的案卷，按**案件编号从小到大**排序，输出嫌疑人姓名（每行一个）。

若没有 `CRITICAL` 级别的案卷，输出 `none`。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，每行格式为：

```
案件编号 嫌疑人姓名 密级
```

案件编号为正整数（$1 \le$ 编号 $\le 10000$），姓名和密级均为不含空格的字符串。

## 输出格式

按案件编号升序，每行输出一个嫌疑人姓名。

## 样例

**输入**

```
5
302 Alice CRITICAL
105 Bob NORMAL
201 Carol CRITICAL
408 Dave LOW
103 Eve CRITICAL
```

**输出**

```
Eve
Carol
Alice
```

**说明**：筛选密级 CRITICAL：103(Eve)、201(Carol)、302(Alice)，按编号升序输出。
