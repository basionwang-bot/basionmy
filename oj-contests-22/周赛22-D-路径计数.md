# D - 路径计数

**难度**：★★★★☆  
**考点**：二维数组递推（网格路径计数DP）  
**GESP 级别**：4级

---

## 题目背景

侦探需要从证据室（左上角）出发到达审讯室（右下角）。走廊是 $n \times m$ 的网格，每步只能向**右**或向**下**移动。侦探想知道共有多少条不同的行走路径。

## 题目描述

给定 $n$ 行 $m$ 列的网格，从左上角 $(1,1)$ 出发，每步只能向右或向下移动，到达右下角 $(n,m)$。计算不同路径的数量。

## 输入格式

一行两个正整数 $n$、$m$（$1 \le n, m \le 18$）。

## 输出格式

输出一个整数，表示不同路径的数量。

## 样例

**输入 1**

```
3 3
```

**输出 1**

```
6
```

**说明**：$3 \times 3$ 网格从左上到右下，需要向右 2 步、向下 2 步，共 $\binom{4}{2}=6$ 种。

---

**输入 2**

```
1 5
```

**输出 2**

```
1
```

**说明**：只有一行，只能一直向右，只有 1 条路径。

---

**输入 3**

```
3 7
```

**输出 3**

```
28
```

## 提示

经典递推（等价于杨辉三角）：

```cpp
long long dp[20][20];
dp[1][j] = 1 for all j;   // 第一行只能一路向右
dp[i][1] = 1 for all i;   // 第一列只能一路向下
dp[i][j] = dp[i-1][j] + dp[i][j-1];  // 从上方或左方来
```

数学上等价于 $\binom{n+m-2}{n-1}$。使用 `long long`，$n=m=18$ 时结果约 $1.17 \times 10^9$，在 int 范围内但接近上限，建议用 `long long`。
