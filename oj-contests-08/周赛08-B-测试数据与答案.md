## 测试点 2（全部轻货）

**输入**
```
3
100 250 500
```
**预期输出**
```
3
0
0
283.3
```

**说明**：平均 = (100+250+500)/3 = 850/3 ≈ 283.3。

## 测试点 3（边界值：恰好 500g 和 2000g）

**输入**
```
4
500 501 2000 2001
```
**预期输出**
```
1
2
1
1250.5
```

**说明**：500→轻，501→中，2000→中，2001→重；平均=(500+501+2000+2001)/4=5002/4=1250.5。

## 测试点 5（混合数据）

**输入**
```
6
300 600 900 2500 400 1800
```


**说明**：平均=(300+600+900+2500+400+1800)/6=6500/6≈1083.3。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int n;
    cin >> n;

    int light = 0, medium = 0, heavy = 0;
    double total = 0;

    for (int i = 0; i < n; i++) {
        int w;
        cin >> w;
        total += w;
        if (w <= 500) light++;
        else if (w <= 2000) medium++;
        else heavy++;
    }

    cout << light << endl;
    cout << medium << endl;
    cout << heavy << endl;
    cout << fixed << setprecision(1) << total / n << endl;

    return 0;
}
```
