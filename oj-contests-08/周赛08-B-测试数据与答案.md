## 测试点 5（较大数，验证不溢出）

**输入**
```
2
9999 6666
10000 7500
```
**预期输出**
```
3333 19998
2500 30000
```

**验证**：
- gcd(9999, 6666)：9999=1×6666+3333，6666=2×3333+0 → gcd=3333；lcm=9999/3333×6666=3×6666=19998 ✓
- gcd(10000, 7500)：10000=1×7500+2500，7500=3×2500+0 → gcd=2500；lcm=10000/2500×7500=4×7500=30000 ✓

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    return b == 0 ? a : gcd(a % b, b);
}

int main() {
    int T, a, b;
    cin >> T;
    while (T--) {
        cin >> a >> b;
        int g = gcd(a, b);
        cout << g << " " << a / g * b << endl;
    }
    return 0;
}
```

### 关键代码解析

```cpp
int gcd(int a, int b) {
    return b == 0 ? a : gcd(a % b, b);  // 辗转相除：余数为0时返回被除数
}

// LCM：先除后乘，防止 a*b 溢出
int lcm = a / gcd(a, b) * b;
```

**辗转相除法手推示例**（gcd(12, 8)）：
```
gcd(12, 8) → gcd(8, 4) → gcd(4, 0) → 返回 4
```
