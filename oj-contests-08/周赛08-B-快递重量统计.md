# B - 快递重量统计

**难度**：★★☆☆☆
**考点**：一维数组、分类统计、浮点平均值
**GESP 级别**：2级

---

## 题目背景

快递分拣中心每天处理大量包裹，按重量分为三类：
- **轻货**：重量 $\leq 500$ 克
- **中货**：$501$ 克 $\leq$ 重量 $\leq 2000$ 克
- **重货**：重量 $> 2000$ 克

请根据输入的包裹重量，统计各类数量和所有包裹的平均重量。

## 题目描述

给定 $n$ 件包裹的重量（克），请输出：
1. 轻货件数；
2. 中货件数；
3. 重货件数；
4. 所有包裹的平均重量（保留一位小数）。

## 输入格式

第一行一个整数 $n$（$1 \leq n \leq 100$）。

第二行 $n$ 个正整数，表示每件包裹的重量（$1 \leq$ 重量 $\leq 10000$，单位：克）。

## 输出格式

共四行，依次输出轻货件数、中货件数、重货件数、平均重量（保留一位小数）。

## 样例

**输入**

```
5
200 800 1500 3000 450
```

**输出**

```
2
2
1
1190.0
```

**说明**

| 包裹 | 重量 | 分类 |
|------|------|------|
| 1 | 200g | 轻货（≤500） |
| 2 | 800g | 中货（501~2000） |
| 3 | 1500g | 中货 |
| 4 | 3000g | 重货（>2000） |
| 5 | 450g | 轻货 |

平均重量 = (200+800+1500+3000+450) / 5 = 5950 / 5 = 1190.0 克。

## 提示

- 用三个计数器分别统计轻、中、重货件数；
- 用一个累加器求总重量，最后除以 $n$ 得平均值；
- 输出平均值时使用 `fixed << setprecision(1)`。
