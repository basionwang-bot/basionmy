# A - 打车计费

**难度**：★☆☆☆☆
**考点**：条件分支、浮点运算、多组输入
**GESP 级别**：1级

---

## 题目背景

城市出租车采用分段计费方式：行驶距离在 3 公里以内（含 3 公里）收起步价 12 元；超出 3 公里后，每超出 1 公里加收 2.5 元。请根据行驶里程计算每次的打车费用。

## 题目描述

给定 $T$ 次出行，每次给出行驶里程 $d$（正整数，单位：公里），请计算每次的打车费用，保留一位小数。

## 输入格式

第一行一个整数 $T$（$1 \leq T \leq 100$）。

接下来 $T$ 行，每行一个正整数 $d$（$1 \leq d \leq 100$）。

## 输出格式

共 $T$ 行，每行输出一个保留一位小数的实数，表示打车费用（元）。

## 样例

**输入**

```
4
2
3
10
1
```

**输出**

```
12.0
12.0
29.5
12.0
```

**说明**

- $d=2$：未超出 3 公里，收起步价 12.0 元。
- $d=3$：恰好 3 公里，仍收起步价 12.0 元。
- $d=10$：超出 7 公里，12 + 7 × 2.5 = 29.5 元。
- $d=1$：未超出，12.0 元。

## 提示

- 用 `if-else` 分两种情况计算：
  - $d \leq 3$：费用 = 12.0
  - $d > 3$：费用 = 12.0 + $(d - 3)$ × 2.5
- 输出时使用 `fixed << setprecision(1)` 保留一位小数。
