# A - 幸运门牌号

**难度**：★☆☆☆☆
**考点**：素数判断（试除法）
**GESP 级别**：5级

---

## 题目背景

城市新建了一批住宅，居委会认为门牌号是**质数**的房间特别吉利，称为"幸运房间"。给定若干个门牌号，请判断哪些是幸运房间。

## 题目描述

给定 $T$ 个门牌号，判断每个门牌号是否是质数。若是质数，输出 `Lucky`；否则输出 `Not Lucky`。

> 质数（素数）：大于 1 的自然数中，只能被 1 和自身整除的数。

## 输入格式

第一行一个整数 $T$（$1 \leq T \leq 100$）。

接下来 $T$ 行，每行一个整数 $n$（$1 \leq n \leq 10^6$）。

## 输出格式

共 $T$ 行，每行输出 `Lucky` 或 `Not Lucky`。

## 样例

**输入**

```
5
7
10
1
2
13
```

**输出**

```
Lucky
Not Lucky
Not Lucky
Lucky
Lucky
```

**说明**

- 7：只有 1 和 7 两个因数 → 质数 → Lucky
- 10：2×5 → 非质数 → Not Lucky
- 1：质数定义要求大于 1 → Not Lucky
- 2：最小的质数 → Lucky
- 13：只有 1 和 13 → 质数 → Lucky

## 提示

- **试除法**：对于 $n$，只需检查 $2$ 到 $\lfloor\sqrt{n}\rfloor$ 之间是否有因数，若有则不是质数。
- 注意 $n=1$ 的特殊情况：1 不是质数。
- `sqrt(n)` 需引入 `<cmath>`，或用 `i*i <= n` 避免浮点误差。
