# E - 行程合法性检查

**难度**：★★★☆☆
**考点**：栈（括号匹配）
**GESP 级别**：6级

---

## 题目背景

城市出行 App 支持用括号标注行程嵌套结构，例如 `{地铁(换乘[公交])}` 表示乘地铁后换乘公交。App 需要检查用户输入的行程字符串中，括号是否**合法配对**，以便正确解析出行结构。

## 题目描述

给定 $T$ 个字符串，每个字符串只包含字母、数字和括号（`(`、`)`、`[`、`]`、`{`、`}`）。

请判断每个字符串中的括号是否合法配对：
- 每个左括号必须有对应的同类右括号与之匹配；
- 匹配必须按正确的嵌套顺序进行（不能交叉）。

合法输出 `Yes`，不合法输出 `No`。

## 输入格式

第一行一个整数 $T$（$1 \leq T \leq 100$）。

接下来 $T$ 行，每行一个字符串（长度 $\leq 1000$，只含字母、数字及括号字符）。

## 输出格式

共 $T$ 行，每行输出 `Yes` 或 `No`。

## 样例

**输入**

```
5
({[]})
([)]
{hello(world)}
(((
abc
```

**输出**

```
Yes
No
Yes
No
Yes
```

**说明**

- `({[]})` ：括号嵌套合法 → `Yes`
- `([)]` ：`(` 对应的应是 `)`，但遇到了 `]` → `No`
- `{hello(world)}` ：忽略字母，括号合法 → `Yes`
- `(((` ：三个左括号无对应右括号 → `No`
- `abc` ：没有括号，视为合法 → `Yes`

## 提示

- 遍历字符串中的每个字符：
  - 若是左括号（`(`、`[`、`{`），压入栈；
  - 若是右括号（`)`、`]`、`}`），检查栈顶是否为对应的左括号：
    - 栈为空 → 不合法；
    - 栈顶不匹配 → 不合法；
    - 栈顶匹配 → 弹出栈顶，继续；
  - 其他字符忽略。
- 遍历结束后，若栈为空则合法，否则不合法。
- C++ 中使用 `#include <stack>` 和 `stack<char>`。
