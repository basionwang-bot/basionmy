## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <stack>
#include <string>
using namespace std;

bool isValid(const string& s) {
    stack<char> st;
    for (char c : s) {
        if (c == '(' || c == '[' || c == '{') {
            st.push(c);
        } else if (c == ')' || c == ']' || c == '}') {
            if (st.empty()) return false;
            char top = st.top();
            st.pop();
            if (c == ')' && top != '(') return false;
            if (c == ']' && top != '[') return false;
            if (c == '}' && top != '{') return false;
        }
        // 字母、数字等其他字符忽略
    }
    return st.empty(); // 栈空说明所有左括号都被匹配
}

int main() {
    int T;
    cin >> T;
    while (T--) {
        string s;
        cin >> s;
        cout << (isValid(s) ? "Yes" : "No") << endl;
    }
    return 0;
}
```

### 栈的执行过程示意（`({[]})` ）

```
遇到 ( → 栈: [(]
遇到 { → 栈: [(, {]
遇到 [ → 栈: [(, {, []
遇到 ] → 栈顶为 [，匹配 → 弹出，栈: [(, {]
遇到 } → 栈顶为 {，匹配 → 弹出，栈: [(]
遇到 ) → 栈顶为 (，匹配 → 弹出，栈: []
遍历结束，栈为空 → Yes
```
