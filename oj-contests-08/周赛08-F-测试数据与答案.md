## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int a[10], perm[10], n;
bool used[10];

void dfs(int step) {
    if (step == n) {
        for (int i = 0; i < n; i++) {
            if (i > 0) cout << " ";
            cout << perm[i];
        }
        cout << endl;
        return;
    }
    for (int i = 0; i < n; i++) {
        if (!used[i]) {
            used[i] = true;
            perm[step] = a[i];
            dfs(step + 1);
            used[i] = false; // 回溯
        }
    }
}

int main() {
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    sort(a, a + n); // 排序，保证 DFS 按字典序枚举
    dfs(0);
    return 0;
}
```

### DFS 执行过程示意（n=3，编号 [1,2,3]）

```
dfs(0)
├── 选 1，dfs(1)
│   ├── 选 2，dfs(2)
│   │   └── 选 3，dfs(3) → 输出 1 2 3
│   └── 选 3，dfs(2)
│       └── 选 2，dfs(3) → 输出 1 3 2
├── 选 2，dfs(1)
│   ├── 选 1，dfs(2) → 输出 2 1 3
│   └── 选 3，dfs(2) → 输出 2 3 1
└── 选 3，dfs(1)
    ├── 选 1，dfs(2) → 输出 3 1 2
    └── 选 2，dfs(2) → 输出 3 2 1
```

**关键点**：`used[i]=false`（回溯）是 DFS 枚举的核心——撤销当前选择，让后续选择可以重新使用该编号。
