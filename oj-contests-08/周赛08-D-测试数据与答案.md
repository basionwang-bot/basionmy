## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;

    int a[100005];
    for (int i = 0; i < n; i++) cin >> a[i];

    while (m--) {
        int x;
        cin >> x;

        int lo = 0, hi = n - 1, ans = -1;
        while (lo <= hi) {
            int mid = (lo + hi) / 2;
            if (a[mid] == x) {
                ans = mid + 1; // 1-indexed
                break;
            } else if (a[mid] < x) {
                lo = mid + 1;  // 目标在右半
            } else {
                hi = mid - 1;  // 目标在左半
            }
        }
        cout << ans << endl;
    }
    return 0;
}
```

### 二分查找过程示意（在 [1,3,5,7,9,11] 中查找 7）

```
lo=0, hi=5 → mid=2, a[2]=5 < 7 → lo=3
lo=3, hi=5 → mid=4, a[4]=9 > 7 → hi=3
lo=3, hi=3 → mid=3, a[3]=7 == 7 → 找到！返回 3+1=4
```
