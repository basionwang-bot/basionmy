## 测试点 4（T=1，最短里程）

**输入**
```
1
1
```
**预期输出**
```
12.0
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int T, d;
    cin >> T;
    while (T--) {
        cin >> d;
        double cost;
        if (d <= 3) {
            cost = 12.0;
        } else {
            cost = 12.0 + (d - 3) * 2.5;
        }
        cout << fixed << setprecision(1) << cost << endl;
    }
    return 0;
}
```
