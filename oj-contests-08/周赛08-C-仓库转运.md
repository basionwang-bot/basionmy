# C - 仓库转运

**难度**：★★☆☆☆
**考点**：递归（汉诺塔）
**GESP 级别**：4级

---

## 题目背景

城市物流中心有三个仓库 A、B、C。A 仓库中堆放着 $n$ 箱货物，按大小从下到上排列（最大的在最底层）。现在需要将这 $n$ 箱货物全部转运到 C 仓库，B 仓库作为中转，规则如下：

1. 每次只能移动一箱货物；
2. 大箱子不能放在小箱子上面；
3. 货物只能放在三个仓库中的某一个。

请输出具体的移动步骤，以及总共需要移动多少次。

## 题目描述

给定 $n$（$1 \leq n \leq 10$），输出将 $n$ 箱货物从 A 转移到 C（经由 B 中转）的所有移动步骤，最后一行输出总移动次数。

## 输入格式

一行一个整数 $n$（$1 \leq n \leq 10$）。

## 输出格式

每行格式为 `X->Y`，表示将最上面的货物从仓库 X 移到仓库 Y。最后一行输出总移动次数。

## 样例

**输入**

```
2
```

**输出**

```
A->B
A->C
B->C
3
```

**说明**

2 箱货物的移动过程：
- 将小箱（上层）从 A 移到 B
- 将大箱（下层）从 A 移到 C
- 将小箱从 B 移到 C

共 3 次，等于 $2^2 - 1 = 3$。

## 提示

- 总移动次数公式：$2^n - 1$（可以用公式验证输出是否正确）。
- **递归思路**：把 $n$ 个箱子从 A 移到 C，可以拆分为：
  1. 把上面 $n-1$ 个箱子从 A 移到 B（借助 C）；
  2. 把最大的箱子从 A 移到 C；
  3. 把 $n-1$ 个箱子从 B 移到 C（借助 A）。
- 递归终止条件：$n=1$ 时直接输出 `A->C`（根据具体参数）。
