# F - 星际迷航

**难度**：★★★☆☆
**考点**：BFS（队列广度优先搜索）、网格最短路径
**GESP 级别**：6级

---

## 题目背景

飞船穿越小行星带时，所处空间被划分成 $n \times m$ 的网格地图：
- `'S'` 表示飞船起点；
- `'T'` 表示终点（任务目的地）；
- `'.'` 表示安全区，可以通行；
- `'#'` 表示陨石带，不能通行。

飞船每步可以向**上、下、左、右**移动一格（不能斜向移动），不能进入陨石带，也不能越过地图边界。请计算从 S 到 T 的**最少步数**。

## 题目描述

给定 $n \times m$ 的字符地图，输出从 S 到 T 的最少步数。若无法到达，输出 `-1`。

## 输入格式

第一行两个整数 $n$、$m$（$1 \leq n, m \leq 100$）。

接下来 $n$ 行，每行 $m$ 个字符，仅由 `S`、`T`、`.`、`#` 组成。地图中**恰好**有一个 `S` 和一个 `T`。

## 输出格式

一个整数，从 S 到 T 的最少步数；若无法到达，输出 `-1`。

## 样例

**输入**

```
4 5
S....
.###.
....#
##..T
```

**输出**

```
7
```

**说明**

地图：
```
S . . . .
. # # # .
. . . . #
# # . . T
```

一条最短路径（共 7 步）：

(0,0) → (1,0) → (2,0) → (2,1) → (2,2) → (2,3) → (3,3) → (3,4)

因为最上一行右端 `(0,4)` 虽可达，但其下一格 `(1,4)` 是死胡同（被陨石环绕），所以飞船必须绕下行。

## 提示

**广度优先搜索（BFS）**：求**无权图**最短路径的标准算法，通过队列按层扩展。

核心步骤：
1. 准备距离数组 `dist[][]`，初始化为 `-1`（表示未访问）；
2. 把起点 S 加入队列，`dist[sx][sy] = 0`；
3. 循环：从队列头取出当前格 $(x, y)$，尝试向 4 个方向扩展；
4. 若新格在边界内、不是障碍、且 `dist[][] == -1`，则更新 `dist[nx][ny] = dist[x][y] + 1` 并入队；
5. 直至队列空或访问到 T；最后输出 `dist[tx][ty]`。

```cpp
queue<pair<int,int>> q;
q.push({sx, sy});
dist[sx][sy] = 0;

int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

while (!q.empty()) {
    auto [x, y] = q.front(); q.pop();
    for (int d = 0; d < 4; d++) {
        int nx = x + dx[d], ny = y + dy[d];
        if (nx < 0 || nx >= n || ny < 0 || ny >= m) continue;
        if (grid[nx][ny] == '#') continue;
        if (dist[nx][ny] != -1) continue;
        dist[nx][ny] = dist[x][y] + 1;
        q.push({nx, ny});
    }
}
```

复杂度 $O(n \times m)$，每个格子最多入队/出队各一次。
