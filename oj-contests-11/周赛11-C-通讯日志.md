# C - 通讯日志

**难度**：★★☆☆☆
**考点**：字符串 `find` / 子串查找
**GESP 级别**：3级

---

## 题目背景

太空基地接收到 $n$ 条加密通讯日志。情报员怀疑某些日志中包含一个特定关键词，希望快速筛出含有该关键词的所有日志编号。

## 题目描述

给定 $n$ 条日志（仅由小写英文字母和下划线 `_` 组成）和一个关键词 $\text{key}$，依次输出**包含**关键词 $\text{key}$ 的日志编号（从 1 开始计数）。若没有任何日志包含该关键词，输出 `None`。

注意：必须是**子串完整出现**（连续匹配），不要求是独立单词。例如日志 `unknown_alien_signal` 包含关键词 `alien`。

## 输入格式

第一行两个内容：整数 $n$（$1 \leq n \leq 100$）和关键词 $\text{key}$（长度 $1 \leq |\text{key}| \leq 20$），用空格分隔。

接下来 $n$ 行，每行一条日志（长度 $1 \leq |\text{log}| \leq 100$）。

## 输出格式

每行输出一个编号（按日志原顺序），表示该编号的日志包含关键词；若没有匹配的日志，输出 `None`。

## 样例 1

**输入**

```
4 alien
ufo_detected_today
no_alien_signal_now
unknown_object_alien
weather_report
```

**输出**

```
2
3
```

**说明**：第 2 条日志在 `no_` 后含有 `alien`；第 3 条日志末尾含有 `alien`。

## 样例 2

**输入**

```
3 fire
water_flow
ice_cold
warm_breeze
```

**输出**

```
None
```

## 提示

C++ `string` 类提供 `find` 函数：

```cpp
string s = "hello_world";
if (s.find("world") != string::npos) {
    // 包含子串 "world"
}
```

`find` 返回子串首次出现的位置，找不到时返回 `string::npos`（一个特殊常量）。

```cpp
for (int i = 1; i <= n; i++) {
    string log;
    cin >> log;
    if (log.find(key) != string::npos) {
        cout << i << endl;
    }
}
```

注意：要用 `bool found` 标记是否输出过，循环结束后没找到则打印 `None`。
