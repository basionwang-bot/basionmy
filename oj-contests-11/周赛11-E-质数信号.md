# E - 质数信号

**难度**：★★★☆☆
**考点**：埃氏筛法（批量求质数）
**GESP 级别**：5级

---

## 题目背景

太空望远镜在某频段连续接收信号，频率范围在 1 ~ 10000 之间。研究表明，**质数频率**的信号是稳定信号。请帮研究员找出指定频率区间 $[l, r]$ 内的全部稳定信号。

## 题目描述

给定整数区间 $[l, r]$，输出该区间内全部质数（升序，空格分隔）。若区间内没有质数，输出 `None`。

回顾：**质数**是只能被 1 和它本身整除的大于 1 的整数（如 2、3、5、7、11 …）。1 不是质数。

## 输入格式

一行两个整数 $l$ 和 $r$（$1 \leq l \leq r \leq 10000$），用空格分隔。

## 输出格式

一行：区间 $[l, r]$ 内全部质数，升序，空格分隔；若无质数则输出 `None`。

## 样例 1

**输入**

```
2 20
```

**输出**

```
2 3 5 7 11 13 17 19
```

## 样例 2

**输入**

```
8 10
```

**输出**

```
None
```

## 提示

**埃拉托斯特尼筛法（埃氏筛）**：批量求出 $[2, N]$ 内所有质数的高效方法。

核心思路：
1. 准备布尔数组 `composite[]`，初始全为 `false`；
2. 从 $i = 2$ 开始遍历，若 `composite[i] == false`（说明 $i$ 是质数），把它的所有倍数 $i^2, i^2 + i, i^2 + 2i, \ldots$ 标记为合数；
3. 直到 $i^2 > N$ 即可停止筛（更小的合数已被更小的质因子筛掉）。

```cpp
const int MAXN = 10005;
bool composite[MAXN];

void sieve() {
    composite[0] = composite[1] = true;  // 0、1 都不是质数
    for (int i = 2; (long long)i * i < MAXN; i++) {
        if (!composite[i]) {
            for (int j = i * i; j < MAXN; j += i)
                composite[j] = true;
        }
    }
}
```

复杂度：$O(N \log \log N)$，对 $N = 10^4$ 几乎瞬间完成。

完成筛法后，`composite[x] == false` 表示 $x$ 是质数（$x \geq 2$）。
