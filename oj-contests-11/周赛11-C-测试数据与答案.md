## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int n;
    string key;
    cin >> n >> key;

    bool found = false;
    for (int i = 1; i <= n; i++) {
        string log;
        cin >> log;
        if (log.find(key) != string::npos) {
            cout << i << endl;
            found = true;
        }
    }
    if (!found) cout << "None" << endl;
    return 0;
}
```

**关键点**：
1. `string::find` 返回首次出现的下标；找不到返回 `string::npos`（约等于 unsigned 最大值）；
2. 同一日志中关键词重复出现不影响判断，匹配一次即输出；
3. 输入用 `cin >> str` 即可（日志保证不含空格）。
