# A - 航行燃料费

**难度**：★☆☆☆☆
**考点**：多分支条件、分段计费、浮点输出
**GESP 级别**：1级

---

## 题目背景

某星际飞船按航行距离 $d$（光年）阶梯计费燃料：
- $0 < d \leq 100$：每光年 **10 元**；
- $100 < d \leq 500$：前 100 光年仍按 10 元/光年，超出部分每光年 **8 元**；
- $d > 500$：前 100 光年按 10 元，101–500 光年按 8 元，超出 500 部分每光年 **5 元**。

## 题目描述

给定 $T$ 组数据，每组一个航行距离 $d$，输出该次航行的总燃料费用（保留一位小数）。

## 输入格式

第一行一个整数 $T$（$1 \leq T \leq 100$）。

接下来 $T$ 行，每行一个正整数 $d$（$1 \leq d \leq 10000$，单位光年）。

## 输出格式

共 $T$ 行，每行一个数，表示该组的燃料费（保留一位小数）。

## 样例

**输入**

```
3
50
200
600
```

**输出**

```
500.0
1800.0
4700.0
```

**说明**

| d | 计算 | 费用 |
|---|------|------|
| 50 | 50×10 | 500.0 |
| 200 | 100×10 + 100×8 | 1000+800=1800.0 |
| 600 | 100×10 + 400×8 + 100×5 | 1000+3200+500=4700.0 |

## 提示

- 用 `if / else if / else` 三段判断；
- 输出时使用 `fixed << setprecision(1)`。
