## 测试点 2（边界值：100 与 500）

**输入**
```
2
100
500
```
**预期输出**
```
1000.0
4200.0
```

**说明**：100 光年恰好为第 1 段上限，1000；500 光年正好为第 2 段上限，1000+400×8=4200。

## 测试点 4（超长航行）

**输入**
```
2
10000
1000
```
**预期输出**
```
51700.0
6700.0
```

**说明**：
- 10000：1000 + 3200 + 9500×5 = 1000+3200+47500 = 51700.0
- 1000：1000 + 3200 + 500×5 = 1000+3200+2500 = 6700.0

---

## 测试点 5（混合多组）

**输入**
```
5
99
101
499
501
5000
```
**预期输出**
```
990.0
1008.0
4192.0
4205.0
26700.0
```

**说明**：
- 99：99×10 = 990.0
- 101：1000 + 1×8 = 1008.0
- 499：1000 + 399×8 = 4192.0
- 501：1000 + 400×8 + 1×5 = 4205.0
- 5000：1000 + 3200 + 4500×5 = 26700.0

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int d;
        cin >> d;
        double cost;
        if (d <= 100) cost = d * 10;
        else if (d <= 500) cost = 100 * 10 + (d - 100) * 8;
        else cost = 100 * 10 + 400 * 8 + (d - 500) * 5;
        cout << fixed << setprecision(1) << cost << endl;
    }
    return 0;
}
```
