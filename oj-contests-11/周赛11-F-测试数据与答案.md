# F - 星际迷航 · 测试数据与答案

---

## 测试点 1（样例）

**输入**
```
4 5
S....
.###.
....#
##..T
```
**预期输出**
```
7
```

## 测试点 2（S 与 T 相邻）

**输入**
```
1 2
ST
```
**预期输出**
```
1
```

---

## 测试点 3（一行直线，无障碍）

**输入**
```
1 5
S...T
```
**预期输出**
```
4
```

**说明**：直线走 4 步即可，BFS 退化成线性扩展。

---

## 测试点 4（被障碍完全隔开，无法到达）

**输入**
```
3 3
S.#
.##
##T
```
**预期输出**
```
-1
```

**说明**：S 可到达的区域为 `{(0,0), (0,1), (1,0)}`，T 周围全为障碍，无法连通。

---

## 测试点 5（5×5 迷宫，需要绕路）

**输入**
```
5 5
S....
.###.
.#.#.
.###.
....T
```
**预期输出**
```
8
```

**说明**：中心被障碍包围，飞船必须沿外圈绕行。一条最短路径：
(0,0)→(1,0)→(2,0)→(3,0)→(4,0)→(4,1)→(4,2)→(4,3)→(4,4)，共 8 步。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <queue>
#include <cstring>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;

    char grid[105][105];
    int sx = 0, sy = 0, tx = 0, ty = 0;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            cin >> grid[i][j];
            if (grid[i][j] == 'S') { sx = i; sy = j; }
            if (grid[i][j] == 'T') { tx = i; ty = j; }
        }
    }

    int dist[105][105];
    memset(dist, -1, sizeof(dist));

    queue<pair<int,int>> q;
    q.push({sx, sy});
    dist[sx][sy] = 0;

    int dx[] = {-1, 1, 0, 0};
    int dy[] = {0, 0, -1, 1};

    while (!q.empty()) {
        pair<int,int> cur = q.front(); q.pop();
        int x = cur.first, y = cur.second;

        for (int d = 0; d < 4; d++) {
            int nx = x + dx[d];
            int ny = y + dy[d];
            if (nx < 0 || nx >= n || ny < 0 || ny >= m) continue;
            if (grid[nx][ny] == '#') continue;
            if (dist[nx][ny] != -1) continue;
            dist[nx][ny] = dist[x][y] + 1;
            q.push({nx, ny});
        }
    }

    cout << dist[tx][ty] << endl;
    return 0;
}
```

**关键点**：
1. `dist` 数组兼具"是否访问"与"距离记录"两个用途，初始 `-1` 表示未访问；
2. BFS 第一次到达终点就是最短距离，无需继续扩展（也可以等队列空再读 `dist[tx][ty]`）；
3. 4 方向偏移用 `dx[]/dy[]` 数组写起来最简洁；
4. 二维下标用 `pair<int,int>`，C++14 后可以用 `auto [x,y] = q.front();` 解构；
5. 不能到达时 `dist[tx][ty]` 仍为初始的 `-1`，正好是题目要求的输出。
