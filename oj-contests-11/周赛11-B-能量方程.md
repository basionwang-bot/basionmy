# B - 能量方程

**难度**：★★☆☆☆
**考点**：双层循环枚举、数字逻辑（平方和数对）
**GESP 级别**：2级

---

## 题目背景

飞船同时启动两个推进器，速度分量分别为**正整数** $i$ 和 $j$（$i \leq j$）。根据能量守恒方程，当 $i^2 + j^2 = m$ 时，飞船能量恰好达到目标值 $m$。

工程师需要枚举所有可行的速度分量组合。

## 题目描述

给定一个正整数 $m$，输出所有满足 $i^2 + j^2 = m$ 且 $1 \leq i \leq j$ 的正整数对 $(i, j)$，按 $i$ 从小到大输出（同一 $i$ 至多对应一个 $j$）。若不存在任何解，输出 `None`。

## 输入格式

一个正整数 $m$（$1 \leq m \leq 10000$）。

## 输出格式

若存在解，每行一对 `i j`，按 $i$ 升序排列；否则输出 `None`。

## 样例 1

**输入**

```
25
```

**输出**

```
3 4
```

**说明**：$3^2 + 4^2 = 9 + 16 = 25$。

## 样例 2

**输入**

```
50
```

**输出**

```
1 7
5 5
```

**说明**：$1^2 + 7^2 = 1 + 49 = 50$；$5^2 + 5^2 = 25 + 25 = 50$。

## 样例 3

**输入**

```
7
```

**输出**

```
None
```

**说明**：无法把 7 写成两个正整数的平方和。

## 提示

**双层循环枚举**：

- 外层 $i$ 从 1 开始；由于 $i \leq j$，所以 $2i^2 \leq m$，即 $i \leq \sqrt{m/2}$；
- 内层 $j$ 从 $i$ 开始；由于 $j^2 \leq m$，所以 $j \leq \sqrt{m}$；
- 内层验证 $i^2 + j^2 = m$ 时输出。

```cpp
for (int i = 1; i * i * 2 <= m; i++) {
    for (int j = i; j * j <= m; j++) {
        if (i * i + j * j == m) {
            cout << i << " " << j << endl;
        }
    }
}
```

数据范围 $m \leq 10000$，最多枚举 $70 \times 100 = 7000$ 次，完全可行。
