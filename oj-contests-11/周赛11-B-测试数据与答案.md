## 测试点 3（无解，输出 None）

**输入**
```
7
```
**预期输出**
```
None
```

**说明**：1²+1²=2，1²+2²=5，2²+2²=8，1²+3²=10……无法凑出 7。

## 测试点 4（中等数据，多解）

**输入**
```
125
```
**预期输出**
```
2 11
5 10
```

**说明**：2²+11² = 4+121 = 125；5²+10² = 25+100 = 125。

## 测试点 5（最大数据 m=10000）

**输入**
```
10000
```
**预期输出**
```
28 96
60 80
```

**说明**：
- 28² + 96² = 784 + 9216 = 10000
- 60² + 80² = 3600 + 6400 = 10000

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int m;
    cin >> m;

    bool found = false;
    for (int i = 1; i * i * 2 <= m; i++) {
        for (int j = i; j * j <= m; j++) {
            if (i * i + j * j == m) {
                cout << i << " " << j << endl;
                found = true;
            }
        }
    }
    if (!found) cout << "None" << endl;
    return 0;
}
```

**关键点**：
1. 用 $i \leq j$ 约束避免输出重复对（如 (3,4) 与 (4,3)）；
2. 外层 $i$ 上界 $\sqrt{m/2}$，内层 $j$ 上界 $\sqrt{m}$，避免无效遍历；
3. 整数比较 `i*i + j*j == m` 不涉及浮点误差。
