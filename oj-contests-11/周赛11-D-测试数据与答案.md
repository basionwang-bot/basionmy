## 答案（C++ 参考代码）

**版本 1：递推打表（推荐）**

```cpp
#include <iostream>
using namespace std;

int main() {
    long long f[45];
    f[1] = 1; f[2] = 1;
    for (int i = 3; i <= 40; i++) f[i] = f[i-1] + f[i-2];

    int T;
    cin >> T;
    while (T--) {
        int n;
        cin >> n;
        cout << f[n] << endl;
    }
    return 0;
}
```

**版本 2：记忆化递归**

```cpp
#include <iostream>
using namespace std;

long long memo[45];

long long fib(int n) {
    if (n <= 2) return 1;
    if (memo[n] != 0) return memo[n];
    return memo[n] = fib(n - 1) + fib(n - 2);
}

int main() {
    int T;
    cin >> T;
    while (T--) {
        int n;
        cin >> n;
        cout << fib(n) << endl;
    }
    return 0;
}
```

**关键点**：
1. 朴素递归 `fib(40)` 重复计算约 $10^8$ 次，必须用记忆化或改递推；
2. 多组查询打表后 $O(1)$ 响应，最高效；
3. F(40) = 102334155，仍在 int 范围。
