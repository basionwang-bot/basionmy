# 周赛 03 - C 题：聊天过滤器

**难度**：★★☆☆☆  
**考点**：字符串处理、查找与替换

---

## 题目背景

小强和朋友们在一个线上学习群里聊天。为了避免聊天时被奇怪的广告打扰，  
群管理员写了一个简单的「词语过滤器」：只要消息中出现了敏感词，就用星号 `*` 替换掉。

管理员给出了一个「敏感词列表」，每个敏感词是一个只包含小写字母的字符串。  
如果消息中出现了某个敏感词（作为子串），就需要将这个子串的每个字符都替换为 `*`。

例如，敏感词是 `bad`，消息是 `a bad boy`，过滤后应为 `a *** boy`。

---

## 题目描述

给定一个聊天消息字符串 s，以及 m 个敏感词。  
请你实现过滤器，将 s 中出现的每一个敏感词全部替换为 `*`，其他字符保持不变。

当不同敏感词出现的位置有重叠时，重叠部分也都用 `*` 覆盖即可。

---

## 输入格式

第一行一个整数 m（1 ≤ m ≤ 20），表示敏感词数量。  
接下来 m 行，每行一个由小写字母组成的字符串，表示一个敏感词。  
最后一行是聊天消息字符串 s，长度不超过 1000，由可见字符组成（空格、字母、数字、标点等）。

---

## 输出格式

输出一行，为过滤后的字符串。

---

## 样例

**输入**
```
2
bad
xx
I think this is bad, xx?
```

**输出**
```
I think this is ***, **?
```

**样例解释**

- `bad` 被替换为 `***`；  
- `xx` 被替换为 `**`；  
- 其他字符保持不变。
