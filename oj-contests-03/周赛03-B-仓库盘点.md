# 周赛 03 - B 题：仓库盘点

**难度**：★☆☆☆☆  
**考点**：一维数组、遍历、简单统计

---

## 题目背景

小强在一家网店做兼职，每天晚上需要对仓库里的一批商品做盘点。  
老板给了他一张清单，记录了每种商品相对于前一天库存的变化量：

- 正数表示当天到货（库存增加）；  
- 负数表示当天发货（库存减少）；  
- 零表示当天该商品库存没变化。

清单按商品编号顺序给出，小强想知道：  
哪些商品**库存为正**（仍有货），哪些商品已经**卖空或为负**（需要补货）。

---

## 题目描述

给定 n 个整数，表示编号 1 到 n 的商品当天的库存变化量。  
假设前一天所有商品库存都为 0，则当天结束时，第 i 种商品的库存就是该整数本身。

请你统计：

1. 仍有库存的商品种数；  
2. 库存为正的所有商品编号（从小到大输出）；  
3. 所有商品库存变化量的总和。

---

## 输入格式

第一行一个整数 n（1 ≤ n ≤ 1000）。  
第二行 n 个整数 a1, a2, ..., an（-1000 ≤ ai ≤ 1000），表示每种商品的库存变化量。

---

## 输出格式

输出三行：

1. 第一行一个整数，表示库存为正的商品种数 cnt；  
2. 第二行输出 cnt 个整数，为库存大于 0 的商品编号（下标从 1 开始，按从小到大顺序，编号之间用一个空格分隔）。  
   - 若 cnt 为 0，则输出一行 `None`；  
3. 第三行输出一个整数，表示所有 ai 的总和。

---

## 样例

**输入**
```
5
3 -1 0 2 -4
```

**输出**
```
2
1 4
0
```

**样例解释**

- 第 1 种商品库存为 3（有货）；  
- 第 4 种商品库存为 2（有货）；  
- 其余商品库存为 0 或负数；  
- 总和 3 + (-1) + 0 + 2 + (-4) = 0。
