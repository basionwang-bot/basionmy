## 测试点 1（样例）

**输入**
```
2
bad
xx
I think this is bad, xx?
```

**预期输出**
```
I think this is ***, **?
```

## 测试点 2（多个相同敏感词）

**输入**
```
1
test
this is a test, another test.
```

**预期输出**
```
this is a ****, another ****.
```

---

## 测试点 3（重叠敏感词）

**输入**
```
2
aba
bab
ababa
```

**预期输出**
```
*****
```

解释：位置 0-2 的 "aba" 与 1-3 的 "bab" 重叠，最终 0-4 全部被 `*` 覆盖。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
using namespace std;

const int MAXN = 1010;
string words[25];

int main() {
    int m;
    if (!(cin >> m)) return 0;
    for (int i = 0; i < m; i++) cin >> words[i];
    string s;
    cin.ignore();               // 忽略上一行末尾的换行
    getline(cin, s);            // 读取整行消息

    int n = s.size();
    string res = s;

    for (int i = 0; i < n; i++) {
        for (int k = 0; k < m; k++) {
            const string &w = words[k];
            int len = w.size();
            if (i + len <= n && s.substr(i, len) == w) {
                for (int j = 0; j < len; j++) {
                    res[i + j] = '*';
                }
            }
        }
    }

    cout << res << endl;
    return 0;
}
```
