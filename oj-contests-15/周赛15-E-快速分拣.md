# E - 快速分拣

**难度**：★★★★★  
**考点**：快速排序实现  
**GESP 级别**：5级

---

## 题目背景

机器人工厂的分拣线需要将产品按重量从小到大排列。工程师希望学生用**快速排序**算法实现这一功能，理解分治排序的核心思想。

## 题目描述

给定 $n$ 个整数，用**快速排序**从小到大排序后输出。

快速排序核心思路：选取一个基准值（pivot），将数组分成"小于等于基准"和"大于基准"两部分，递归排序两部分。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 10^5$）。

第二行 $n$ 个整数（$1 \le a_i \le 10^9$）。

## 输出格式

输出排序后的数组（数字间用一个空格隔开）。

## 样例

**输入**

```
6
5 3 8 1 9 2
```

**输出**

```
1 2 3 5 8 9
```

## 提示

经典快速排序模板（选取最右元素为 pivot）：

```cpp
void quickSort(int a[], int l, int r) {
    if (l >= r) return;
    int pivot = a[r];
    int i = l - 1;
    for (int j = l; j < r; j++) {
        if (a[j] <= pivot) {
            i++;
            swap(a[i], a[j]);
        }
    }
    swap(a[i + 1], a[r]);
    int p = i + 1;
    quickSort(a, l, p - 1);
    quickSort(a, p + 1, r);
}
```

- `pivot`：基准值，选最右边的元素；
- 循环结束后 `a[i+1]` 就是 pivot 的最终位置；
- 递归排序 pivot 左侧和右侧两段；
- 平均时间复杂度 $O(n \log n)$，最坏 $O(n^2)$（输入已有序时）。
