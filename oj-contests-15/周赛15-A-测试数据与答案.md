4
90
60
30
80
```

HIGH

HIGH
```

## 测试点 2（边界值）

**输入**
```
4
80
79
40
39
```
**预期输出**
```
HIGH
NORMAL
NORMAL
LOW
```

**说明**：$n=80$ 属于 HIGH，$n=79$ 属于 NORMAL，$n=40$ 属于 NORMAL，$n=39$ 属于 LOW。

---

## 测试点 3（极值）

**输入**
```
2
0
100
```
**预期输出**
```
LOW
HIGH
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int n;
        cin >> n;
        if (n >= 80) cout << "HIGH\n";
        else if (n >= 40) cout << "NORMAL\n";
        else cout << "LOW\n";
    }
    return 0;
}
```

**关键点**：先判断 $\ge80$，再判断 $\ge40$，顺序不能颠倒。
