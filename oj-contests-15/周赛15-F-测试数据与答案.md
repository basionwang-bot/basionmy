## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <vector>
using namespace std;

bool visited[105];
vector<int> adj[105];

void dfs(int u) {
    visited[u] = true;
    for (int v : adj[u]) {
        if (!visited[v]) dfs(v);
    }
}

int main() {
    int n, m;
    cin >> n >> m;
    for (int i = 0; i < m; i++) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    int cnt = 0;
    for (int i = 1; i <= n; i++) {
        if (!visited[i]) {
            dfs(i);
            cnt++;
        }
    }
    cout << cnt << "\n";
    return 0;
}
```

**关键点**：
1. 无向图，每条边要在邻接表中存两次（双向）；
2. 遍历所有节点，每次遇到未访问节点发起 DFS，计数+1；
3. DFS 会把整个连通分量的节点都标记为已访问；
4. 时间复杂度 $O(n+m)$。
