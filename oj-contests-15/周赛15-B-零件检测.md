# B - 零件检测

**难度**：★★☆☆☆  
**考点**：最长连续满足条件的段（cur/ans 模式）  
**GESP 级别**：2级

---

## 题目背景

机器人工厂的质检线对每个零件进行评分（0-100 分），合格零件的评分不低于 60 分。工厂想找出评分最高的"连续合格段"有多长，以评估生产线的稳定性。

## 题目描述

给定 $n$ 个零件的质检评分，找出最长的**连续合格**段的长度（合格：评分 $\ge 60$）。若没有合格零件，输出 $0$。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 1000$）。

第二行 $n$ 个整数，第 $i$ 个为第 $i$ 个零件的评分（$0 \le$ 评分 $\le 100$）。

## 输出格式

输出一个整数，表示最长连续合格段的长度。

## 样例

**输入**

```
8
55 70 85 90 45 60 75 80
```

**输出**

```
3
```

**说明**：合格段有 `[70,85,90]`（长度3）和 `[60,75,80]`（长度3），最长为 3。

## 提示

```cpp
int cur = 0, ans = 0;
for (int i = 0; i < n; i++) {
    if (score[i] >= 60) {
        cur++;
        ans = max(ans, cur);
    } else {
        cur = 0;
    }
}
```
