# D - 精准传参

**难度**：★★★★☆  
**考点**：指针基础（传地址实现交换）  
**GESP 级别**：4级

---

## 题目背景

机器人工厂的调试系统需要对零件编号数组执行一系列"位置互换"操作，每次操作通过指针传参方式交换两个位置上的零件编号。

## 题目描述

给定 $n$ 个零件编号组成的数组（下标从 1 开始），执行 $m$ 次交换操作。每次操作给出两个位置 $x$ 和 $y$，将 $a[x]$ 与 $a[y]$ 互换。

要求用**指针传参**的 `swapByPtr` 函数实现交换（函数签名：`void swapByPtr(int* p, int* q)`）。

输出所有操作完成后的数组。

## 输入格式

第一行两个正整数 $n$、$m$（$1 \le n \le 100$，$1 \le m \le 100$）。

第二行 $n$ 个整数，为初始数组（$1 \le a_i \le 1000$）。

接下来 $m$ 行，每行两个整数 $x$、$y$（$1 \le x, y \le n$），表示交换位置 $x$ 和位置 $y$ 的元素。

## 输出格式

输出一行，为操作后的数组（数字间用一个空格隔开）。

## 样例

**输入**

```
5 3
10 20 30 40 50
1 3
2 5
3 4
```

**输出**

```
30 50 40 10 20
```

**说明**：初始 `[10,20,30,40,50]`
- 交换位置 1 和 3：`[30,20,10,40,50]`
- 交换位置 2 和 5：`[30,50,10,40,20]`
- 交换位置 3 和 4：`[30,50,40,10,20]`

## 提示

```cpp
void swapByPtr(int* p, int* q) {
    int tmp = *p;
    *p = *q;
    *q = tmp;
}

// 调用：
swapByPtr(&a[x], &a[y]);
```

- `*p` 是对指针 `p` 解引用，可以读取/修改它指向的变量；
- `&a[x]` 取得 `a[x]` 的地址传入函数，函数内修改会直接影响原数组。
