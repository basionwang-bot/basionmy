## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

void swapByPtr(int* p, int* q) {
    int tmp = *p;
    *p = *q;
    *q = tmp;
}

int main() {
    int n, m;
    cin >> n >> m;
    int a[105];
    for (int i = 1; i <= n; i++) cin >> a[i];
    for (int i = 0; i < m; i++) {
        int x, y;
        cin >> x >> y;
        swapByPtr(&a[x], &a[y]);
    }
    for (int i = 1; i <= n; i++) {
        if (i > 1) cout << " ";
        cout << a[i];
    }
    cout << "\n";
    return 0;
}
```

**关键点**：
1. `swapByPtr` 接收两个 `int*`（指针），通过 `*p`、`*q` 直接修改原变量；
2. 传入时用 `&a[x]` 取地址；
3. 若 `x==y`，交换自身，值不变，程序正常运行不需要特判。
