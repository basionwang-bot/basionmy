# F - 电路连通

**难度**：★★★★★  
**考点**：DFS 求无向图连通分量数  
**GESP 级别**：6级

---

## 题目背景

机器人工厂的电路板上有 $n$ 个节点，各节点通过导线相连。工程师需要统计电路板上共有多少个**独立的连通区域**（即连通分量数），以判断电路是否需要重新布线。

## 题目描述

给定一个 $n$ 个节点、$m$ 条无向边的图，统计**连通分量**的数量。

如果两个节点之间有路径（通过若干条边可以互达），则它们在同一连通分量中。

## 输入格式

第一行两个正整数 $n$、$m$（$1 \le n \le 100$，$0 \le m \le 1000$）。

接下来 $m$ 行，每行两个整数 $u$、$v$（$1 \le u, v \le n$），表示节点 $u$ 和节点 $v$ 之间有一条无向边。

## 输出格式

输出一个整数，表示连通分量的数量。

## 样例

**输入 1**

```
5 4
1 2
2 3
4 5
3 1
```

**输出 1**

```
2
```

**说明**：节点 {1,2,3} 相互连通，节点 {4,5} 相互连通，共 2 个连通分量。

---

**输入 2**

```
4 0
```

**输出 2**

```
4
```

**说明**：没有边，每个节点自成一个连通分量，共 4 个。

## 提示

```cpp
bool visited[105] = {false};
vector<int> adj[105];

void dfs(int u) {
    visited[u] = true;
    for (int v : adj[u]) {
        if (!visited[v]) dfs(v);
    }
}

int main() {
    // 建图：读入每条边加入邻接表
    // adj[u].push_back(v); adj[v].push_back(u);
    
    int cnt = 0;
    for (int i = 1; i <= n; i++) {
        if (!visited[i]) {
            dfs(i);
            cnt++;
        }
    }
    cout << cnt;
}
```

- 对每个未访问的节点发起一次 DFS，整棵可达子图都会被标记；
- DFS 次数即连通分量数。
