## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

void quickSort(int a[], int l, int r) {
    if (l >= r) return;
    int pivot = a[r];
    int i = l - 1;
    for (int j = l; j < r; j++) {
        if (a[j] <= pivot) {
            i++;
            swap(a[i], a[j]);
        }
    }
    swap(a[i + 1], a[r]);
    int p = i + 1;
    quickSort(a, l, p - 1);
    quickSort(a, p + 1, r);
}

int main() {
    int n;
    cin >> n;
    int a[100005];
    for (int i = 0; i < n; i++) cin >> a[i];
    quickSort(a, 0, n - 1);
    for (int i = 0; i < n; i++) {
        if (i > 0) cout << " ";
        cout << a[i];
    }
    cout << "\n";
    return 0;
}
```

**关键点**：
1. `pivot = a[r]`（选最右元素为基准）；
2. 分区后 `a[i+1]` 是 pivot 的最终位置，左侧全 ≤ pivot，右侧全 > pivot；
3. 递归对两段继续排序；
4. 平均 $O(n\log n)$，此题 $n\le10^5$ 足够。
