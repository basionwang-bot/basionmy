# C - 产品分级

**难度**：★★★☆☆  
**考点**：结构体 + 条件筛选 + 多关键字排序  
**GESP 级别**：3级

---

## 题目背景

机器人工厂生产的每款产品都有耐久度和性能评分两项指标。工厂需要从所有产品中筛选出达标的型号，按性能排序后展示给客户。

## 题目描述

给定 $n$ 款产品的信息（型号名、耐久度、性能分），筛选满足以下条件的产品：

- 耐久度 $\ge 60$
- 性能分 $\ge 60$

对筛选结果按以下规则排序：

1. 性能分**从大到小**
2. 性能分相同时，耐久度**从大到小**
3. 仍相同时，型号名**字典序从小到大**

输出所有符合条件的产品型号（每行一个）。若没有，输出 `none`。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，每行格式为：

```
型号名 耐久度 性能分
```

型号名为不含空格的字符串（长度 $\le 20$），耐久度和性能分均为整数（$0 \le$ 值 $\le 100$）。

## 输出格式

按排序后顺序，每行输出一个型号名。

## 样例

**输入**

```
5
RX200 75 80
TX100 50 90
MK300 65 80
LX400 70 60
ZX500 40 70
```

**输出**

```
RX200
MK300
LX400
```

**说明**：TX100 耐久度 50 < 60，ZX500 耐久度 40 < 60，淘汰。  
剩余三款排序：性能分降序 → RX200(80)、MK300(80)、LX400(60)。  
RX200 与 MK300 性能相同，耐久度降序：75 > 65，故 RX200 排前。

## 提示

```cpp
struct Product {
    string name;
    int dur, perf;
};
bool cmp(const Product& a, const Product& b) {
    if (a.perf != b.perf) return a.perf > b.perf;
    if (a.dur  != b.dur)  return a.dur  > b.dur;
    return a.name < b.name;
}
```
