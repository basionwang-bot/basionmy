# A - 能量等级

**难度**：★☆☆☆☆  
**考点**：条件分支、多组输入  
**GESP 级别**：1级

---

## 题目背景

机器人工厂为每台机器人分配能量值，并根据能量等级进行分类管理。

## 题目描述

给定机器人的能量值 $n$，按以下规则输出等级：

- $n \ge 80$：输出 `HIGH`
- $40 \le n < 80$：输出 `NORMAL`
- $n < 40$：输出 `LOW`

给定 $T$ 组查询，每组输入一个能量值，输出对应等级。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 100$）。

接下来 $T$ 行，每行一个整数 $n$（$0 \le n \le 100$）。

## 输出格式

共 $T$ 行，每行输出对应等级字符串。

## 样例

**输入**

```
4
90
60
30
80
```

**输出**

```
HIGH
NORMAL
LOW
HIGH
```

## 提示

注意边界：$n=80$ 属于 `HIGH`，$n=40$ 属于 `NORMAL`，$n=39$ 属于 `LOW`。
