## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
long long pre[505][505];
int main() {
    int n, m, Q; cin >> n >> m >> Q;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++) {
            int x; cin >> x;
            pre[i][j] = x + pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1];
        }
    while (Q--) {
        int r1,c1,r2,c2; cin >> r1 >> c1 >> r2 >> c2;
        cout << pre[r2][c2] - pre[r1-1][c2] - pre[r2][c1-1] + pre[r1-1][c1-1] << "\n";
    }
    return 0;
}
```

**关键点**：二维前缀和公式（容斥）：大矩形 − 上方 − 左方 + 左上角（防止重复减）。
