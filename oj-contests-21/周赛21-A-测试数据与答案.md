## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    int T; cin >> T;
    while (T--) {
        int w; cin >> w;
        if (w <= 10) cout << 10 << "\n";
        else if (w <= 50) cout << 25 << "\n";
        else cout << 60 << "\n";
    }
    return 0;
}
```
