# D - 矩形区域和

**难度**：★★★★☆  
**考点**：二维前缀和  
**GESP 级别**：4级

---

## 题目背景

商队在地图上标记了各区域的货物价值，地图被划分为 $n \times m$ 的网格。每次询问一个矩形区域，求该区域内所有格子的货物价值总和。

## 题目描述

给定 $n \times m$ 的整数矩阵，以及 $Q$ 次查询，每次查询一个矩形子区域 $[r_1, c_1]$ 到 $[r_2, c_2]$（行列下标从 1 开始），输出该区域内所有元素之和。

## 输入格式

第一行三个正整数 $n$、$m$、$Q$（$1 \le n, m \le 500$，$1 \le Q \le 10^5$）。

接下来 $n$ 行，每行 $m$ 个整数（$0 \le a_{ij} \le 1000$）。

接下来 $Q$ 行，每行四个整数 $r_1$、$c_1$、$r_2$、$c_2$（$1 \le r_1 \le r_2 \le n$，$1 \le c_1 \le c_2 \le m$）。

## 输出格式

共 $Q$ 行，每行输出对应矩形区域的元素之和。

## 样例

**输入**

```
3 3 3
1 2 3
4 5 6
7 8 9
1 1 2 2
2 2 3 3
1 1 3 3
```

**输出**

```
12
28
45
```

**说明**：
- $[1,1]$到$[2,2]$：$1+2+4+5=12$
- $[2,2]$到$[3,3]$：$5+6+8+9=28$
- $[1,1]$到$[3,3]$：所有元素之和 $=45$

## 提示

二维前缀和 $pre[i][j]$ 表示左上角 $(1,1)$ 到 $(i,j)$ 矩形内所有元素之和：

```cpp
// 预处理
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++)
        pre[i][j] = a[i][j] + pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1];

// 查询 [r1,c1] 到 [r2,c2]
int ans = pre[r2][c2] - pre[r1-1][c2] - pre[r2][c1-1] + pre[r1-1][c1-1];
```

容斥原理：大矩形 − 上方 − 左方 + 左上角（避免多减一次）。
