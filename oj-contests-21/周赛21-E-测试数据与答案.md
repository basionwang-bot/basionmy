## 测试点 5（n=1）

**输入**
```
1
3 7
```
**预期输出**
```
3 7
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
int main() {
    int n; cin >> n;
    vector<pair<int,int>> v(n);
    for (auto& p : v) cin >> p.first >> p.second;
    sort(v.begin(), v.end());
    int curL = v[0].first, curR = v[0].second;
    for (int i = 1; i < n; i++) {
        if (v[i].first <= curR) {
            curR = max(curR, v[i].second);
        } else {
            cout << curL << " " << curR << "\n";
            curL = v[i].first;
            curR = v[i].second;
        }
    }
    cout << curL << " " << curR << "\n";
    return 0;
}
```

**关键点**：
1. 先按左端点排序；
2. 若当前区间左端点 ≤ 上一区间右端点，则合并（取更大的右端点）；
3. 最后一个区间在循环结束后单独输出。
