# B - 驿站补给

**难度**：★★☆☆☆  
**考点**：前缀和 + 固定长度子数组最小和  
**GESP 级别**：2级

---

## 题目背景

商队沿路经过 $n$ 个驿站，每个驿站需要消耗一定补给。商队需要连续经过 $k$ 个驿站，希望找到消耗补给最少的连续 $k$ 站路段。

## 题目描述

给定 $n$ 个驿站的补给消耗量，以及整数 $k$，找出所有长度为 $k$ 的连续子数组中，**元素之和最小**的那个，输出该最小和。

## 输入格式

第一行两个正整数 $n$、$k$（$1 \le k \le n \le 10^5$）。

第二行 $n$ 个正整数（$1 \le a_i \le 1000$）。

## 输出格式

输出一个整数，表示长度为 $k$ 的子数组的最小和。

## 样例

**输入 1**

```
6 3
5 2 7 3 1 4
```

**输出 1**

```
8
```

**说明**：长度为 3 的子数组之和：$5+2+7=14$，$2+7+3=12$，$7+3+1=11$，$3+1+4=8$，最小为 **8**。

---

**输入 2**

```
4 4
1 2 3 4
```

**输出 2**

```
10
```

**说明**：只有一个长度为 4 的子数组，和为 10。

## 提示

用前缀和预处理，每个长度 $k$ 窗口的和 $O(1)$ 计算：

```cpp
pre[0] = 0;
for (int i = 1; i <= n; i++) pre[i] = pre[i-1] + a[i];
int ans = INT_MAX;
for (int i = k; i <= n; i++)
    ans = min(ans, pre[i] - pre[i-k]);
```
