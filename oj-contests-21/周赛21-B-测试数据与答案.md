## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <climits>
using namespace std;
int main() {
    int n, k; cin >> n >> k;
    long long pre[100005] = {};
    for (int i = 1; i <= n; i++) {
        int x; cin >> x;
        pre[i] = pre[i-1] + x;
    }
    long long ans = LLONG_MAX;
    for (int i = k; i <= n; i++)
        ans = min(ans, pre[i] - pre[i-k]);
    cout << ans << "\n";
    return 0;
}
```

**关键点**：前缀和预处理后每次窗口查询 $O(1)$，总时间 $O(n)$，适合大数据量。
