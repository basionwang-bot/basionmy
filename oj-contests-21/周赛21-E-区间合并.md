# E - 区间合并

**难度**：★★★★★  
**考点**：贪心（排序 + 合并重叠区间）  
**GESP 级别**：5级

---

## 题目背景

商队在丝绸之路的不同路段设置了巡逻区间。为了统计实际覆盖的路段长度，需要将所有重叠或相邻的区间合并为不重叠的区间。

## 题目描述

给定 $n$ 个区间 $[l_i, r_i]$，将所有重叠（或端点相同）的区间合并，输出合并后的区间列表，按左端点从小到大排序。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 10^5$）。

接下来 $n$ 行，每行两个整数 $l_i$、$r_i$（$0 \le l_i \le r_i \le 10^9$）。

## 输出格式

每行输出一个合并后的区间，格式为 `l r`。

## 样例

**输入 1**

```
4
1 3
2 6
8 10
15 18
```

**输出 1**

```
1 6
8 10
15 18
```

**说明**：$[1,3]$ 和 $[2,6]$ 重叠（$2 \le 3$），合并为 $[1,6]$。

---

**输入 2**

```
3
1 4
4 5
6 8
```

**输出 2**

```
1 5
6 8
```

**说明**：$[1,4]$ 和 $[4,5]$ 端点相同，合并为 $[1,5]$。

## 提示

```cpp
sort(intervals, ...); // 按左端点升序排序
int curL = intervals[0].l, curR = intervals[0].r;
for (int i = 1; i < n; i++) {
    if (intervals[i].l <= curR) {
        curR = max(curR, intervals[i].r); // 扩展右端点
    } else {
        输出 [curL, curR];
        curL = intervals[i].l;
        curR = intervals[i].r;
    }
}
输出最后一个区间 [curL, curR];
```
