## 测试点 4（多条路，验证字典序最小）

**输入**
```
6 4
1 4
2 4
3 5
5 6
```
**预期输出**
```
1 2 3 4 5 6
```

**说明**：初始入度为0的节点有 {1,2,3}，优先选1，再选2，再选3；3选完后5入度变0，选5；5选完6入度变0，选6；1选完后4入度还有1，待2选完后4入度变0，选4。
最终顺序：1 2 3 5 6 4。

等等，让我重新计算：
- 初始入度：1→0，2→0，3→0，4→2（来自1和2），5→1（来自3），6→1（来自5）
- 选1（最小）→ 4的入度变1
- 选2（最小）→ 4的入度变0，4入堆
- 选3（最小）→ 5的入度变0，5入堆
- 堆中：{4,5}，选4
- 堆中：{5}，选5 → 6入度变0，6入堆
- 选6
- 结果：1 2 3 4 5 6

**修正输出**：`1 2 3 4 5 6`

## 测试点 5（n=1，无依赖）

**输入**
```
1 0
```
**预期输出**
```
1
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <vector>
#include <queue>
using namespace std;
int main() {
    int n, m; cin >> n >> m;
    vector<vector<int>> adj(n+1);
    vector<int> indegree(n+1, 0);
    for (int i = 0; i < m; i++) {
        int a, b; cin >> a >> b;
        adj[a].push_back(b);
        indegree[b]++;
    }
    priority_queue<int, vector<int>, greater<int>> pq;
    for (int i = 1; i <= n; i++) if (indegree[i] == 0) pq.push(i);
    vector<int> result;
    while (!pq.empty()) {
        int u = pq.top(); pq.pop();
        result.push_back(u);
        for (int v : adj[u]) {
            if (--indegree[v] == 0) pq.push(v);
        }
    }
    if ((int)result.size() != n) {
        cout << "IMPOSSIBLE\n";
    } else {
        for (int i = 0; i < n; i++) {
            if (i > 0) cout << " ";
            cout << result[i];
        }
        cout << "\n";
    }
    return 0;
}
```

**关键点**：
1. 统计每个节点的入度；
2. 用**小顶堆**（而非普通队列）保证每次取编号最小的无依赖节点；
3. 若最终处理节点数 < n，说明有环，输出 IMPOSSIBLE。
