# C - 商队档案

**难度**：★★★☆☆  
**考点**：结构体 + 条件筛选 + 多关键字排序  
**GESP 级别**：3级

---

## 题目背景

丝绸之路商队联盟需要整理各商队的档案，筛选出盈利达标的商队，并按收益排序公示。

## 题目描述

给定 $n$ 支商队的信息（名称、货物类型、年收益），筛选满足以下条件的商队：

- 年收益 $\ge 500$ 金

对筛选结果按以下规则排序：

1. 年收益**从大到小**
2. 年收益相同时，名称**字典序从小到大**

输出所有符合条件的商队名称（每行一个）。若没有，输出 `none`。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，每行格式为：

```
名称 货物类型 年收益
```

名称和货物类型均为不含空格的字符串（长度 $\le 20$），年收益为正整数（$1 \le$ 收益 $\le 10000$）。

## 输出格式

按排序后顺序，每行输出一个商队名称。

## 样例

**输入**

```
5
SilkRoad Silk 800
SpiceTeam Spice 300
JadeGroup Jade 800
GoldPath Gold 600
PearlCrew Pearl 200
```

**输出**

```
JadeGroup
SilkRoad
GoldPath
```

**说明**：SpiceTeam(300) 和 PearlCrew(200) 收益不足，淘汰。剩余：SilkRoad(800)、JadeGroup(800)、GoldPath(600)。收益降序：前两者并列，字母序 J < S，JadeGroup 排前。
