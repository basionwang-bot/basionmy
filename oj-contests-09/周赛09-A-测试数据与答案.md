## 测试点 3（只有金牌）

**输入**
```
2
5 0 0
10 0 0
```
**预期输出**
```
15
30
```

**说明**：5×3=15，10×3=30。

---

## 测试点 4（最大值）

**输入**
```
1
100 100 100
```
**预期输出**
```
600
```

**说明**：100×3+100×2+100×1=600。

---

## 测试点 5（混合多组）

**输入**
```
4
2 3 5
0 0 10
7 0 3
1 2 0
```
**预期输出**
```
17
10
24
7
```

**说明**：2×3+3×2+5×1=17；0+0+10=10；7×3+0+3×1=24；1×3+2×2+0=7。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int g, s, b;
        cin >> g >> s >> b;
        cout << 3 * g + 2 * s + b << endl;
    }
    return 0;
}
```
