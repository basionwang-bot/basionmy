# B - 体能测试统计

**难度**：★★☆☆☆
**考点**：一维数组、最值、平均值、计数统计
**GESP 级别**：2级

---

## 题目背景

运动会开始前，体育老师对全班同学进行体能测试，满分 100 分，60 分及以上为达标。

## 题目描述

给定 $n$ 名同学的体能测试成绩，请输出：
1. 最高分；
2. 最低分；
3. 平均分（保留一位小数）；
4. 达标人数（成绩 $\geq 60$ 分的人数）。

## 输入格式

第一行一个整数 $n$（$1 \leq n \leq 100$）。

第二行 $n$ 个整数，表示每位同学的成绩（$0 \leq$ 成绩 $\leq 100$）。

## 输出格式

共四行，依次输出最高分、最低分、平均分（保留一位小数）、达标人数。

## 样例

**输入**

```
5
72 55 88 61 45
```

**输出**

```
88
45
64.2
3
```

**说明**

| 成绩 | 是否达标 |
|------|----------|
| 72 | ✓（≥60） |
| 55 | ✗（<60） |
| 88 | ✓ |
| 61 | ✓ |
| 45 | ✗ |

最高 88，最低 45，平均 = (72+55+88+61+45)/5 = 321/5 = 64.2，达标 3 人。

## 提示

- 遍历数组时同步更新最大值、最小值、累加和、达标计数器。
- 输出平均值时使用 `fixed << setprecision(1)`。
