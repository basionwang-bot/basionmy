# C - 运动员达标

**难度**：★★☆☆☆
**考点**：结构体、条件筛选、双关键字排序
**GESP 级别**：3级

---

## 题目背景

运动会选拔赛设定达标线，只有成绩达到指定分数线的运动员才能晋级，晋级运动员按成绩从高到低公布名单。

## 题目描述

给定 $n$ 名运动员的编号和成绩，以及达标线 $m$，筛选出成绩 $\geq m$ 的运动员，按成绩从高到低排序后输出编号和成绩。成绩相同时，按编号从小到大排列。

若没有运动员达标，输出 `None`。

## 输入格式

第一行两个整数 $n$（$1 \leq n \leq 100$）和 $m$（$0 \leq m \leq 100$）。

接下来 $n$ 行，每行两个整数，分别表示运动员编号 $id$（$1 \leq id \leq 9999$）和成绩 $score$（$0 \leq score \leq 100$）。

## 输出格式

若存在达标运动员，每行输出一名，格式为 `编号 成绩`，按排名从高到低；否则输出 `None`。

## 样例

**输入**

```
5 80
1001 100
1002 75
1003 95
1004 80
1005 78
```

**输出**

```
1001 100
1003 95
1004 80
```

**说明**

| 编号 | 成绩 | 是否达标（≥80） |
|------|------|-----------------|
| 1001 | 100 | ✓ |
| 1002 | 75 | ✗（75<80） |
| 1003 | 95 | ✓ |
| 1004 | 80 | ✓（恰好等于80） |
| 1005 | 78 | ✗ |

达标3人，按成绩降序：100 > 95 > 80。

## 提示

- 定义结构体存储 `id` 和 `score`，先筛选再排序。
- 比较函数：成绩不同按成绩降序；成绩相同按编号升序。
- 筛选条件：`score >= m`。
