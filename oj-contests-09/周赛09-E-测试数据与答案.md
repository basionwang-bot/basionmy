## 测试点 2（互质，GCD=1）

**输入**
```
2
7 9
13 5
```
**预期输出**
```
1 63
1 65
```

**说明**：7和9互质，lcm=63；13和5互质，lcm=65。

---

## 测试点 3（相同数）

**输入**
```
2
6 6
100 100
```
**预期输出**
```
6 6
100 100
```

**说明**：相同的两个数，gcd=lcm=自身。

---

## 测试点 4（一个是另一个的倍数）

**输入**
```
2
12 4
100 25
```
**预期输出**
```
4 12
25 100
```

**说明**：12=4×3，gcd=4，lcm=12；100=25×4，gcd=25，lcm=100。

---

## 测试点 5（较大数值）

**输入**
```
3
1000 750
9999 3333
10000 10000
```
**预期输出**
```
250 3000
3333 9999
10000 10000
```

**说明**：gcd(1000,750)=250，lcm=1000/250×750=3000；gcd(9999,3333)=3333，lcm=9999；gcd(10000,10000)=10000。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

int main() {
    int n;
    cin >> n;
    while (n--) {
        int a, b;
        cin >> a >> b;
        int g = gcd(a, b);
        cout << g << " " << a / g * b << endl;
    }
    return 0;
}
```
