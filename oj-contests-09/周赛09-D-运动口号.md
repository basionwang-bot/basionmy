# D - 运动口号

**难度**：★★★☆☆
**考点**：字符串处理（回文判断）
**GESP 级别**：4级

---

## 题目背景

运动会开幕式上，各班要喊出独特的运动口号。主办方规定，口号必须是"回文串"——正着读和倒着读完全相同，寓意"始终如一、坚持到底"。请帮助主办方检查各班的口号是否合法。

## 题目描述

给定 $T$ 个字符串，判断每个字符串是否为回文串。

**回文串**：正序与逆序完全相同的字符串。例如 `racecar`、`level`、`abba` 是回文串，`hello` 不是。

## 输入格式

第一行一个整数 $T$（$1 \leq T \leq 100$）。

接下来 $T$ 行，每行一个非空字符串（只含小写字母，长度 $1 \leq |s| \leq 100$）。

## 输出格式

共 $T$ 行，若该字符串是回文串输出 `Yes`，否则输出 `No`。

## 样例

**输入**

```
4
racecar
hello
level
abba
```

**输出**

```
Yes
No
Yes
Yes
```

**说明**

| 字符串 | 逆序 | 是否相同 |
|--------|------|----------|
| racecar | racecar | ✓ |
| hello | olleh | ✗ |
| level | level | ✓ |
| abba | abba | ✓ |

## 提示

**方法一（双指针）**：用两个下标 `i`（从头）和 `j`（从尾）同时向中间移动，只要有一对字符不同就不是回文串。

```cpp
bool isPalin = true;
int i = 0, j = s.size() - 1;
while (i < j) {
    if (s[i] != s[j]) { isPalin = false; break; }
    i++; j--;
}
```

**方法二（反转比较）**：将字符串复制一份并翻转，判断是否与原串相等。

```cpp
string rev = s;
reverse(rev.begin(), rev.end());
bool isPalin = (s == rev);
```
