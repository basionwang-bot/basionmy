# E - 训练周期

**难度**：★★★☆☆
**考点**：GCD / LCM（欧几里得算法）
**GESP 级别**：5级

---

## 题目背景

运动会备战期间，两位运动员分别按各自的训练周期循环训练。教练想知道：他们最早能在哪一天同时休息（最小公倍数），以及他们训练周期最长的公共因子是多少（最大公因数），以便安排团队活动。

## 题目描述

给定 $n$ 对正整数 $(a, b)$，分别表示两位运动员的训练周期天数，输出每对数的**最大公因数（GCD）**和**最小公倍数（LCM）**。

## 输入格式

第一行一个整数 $n$（$1 \leq n \leq 100$）。

接下来 $n$ 行，每行两个正整数 $a$、$b$（$1 \leq a, b \leq 10000$）。

## 输出格式

共 $n$ 行，每行两个整数，分别为 $\gcd(a, b)$ 和 $\text{lcm}(a, b)$，空格分隔。

## 样例

**输入**

```
3
12 8
15 10
7 3
```

**输出**

```
4 24
5 30
1 21
```

**说明**

| 输入 | GCD | LCM | 验证 |
|------|-----|-----|------|
| 12, 8 | 4 | 24 | 12=4×3，8=4×2；lcm=4×3×2=24 |
| 15, 10 | 5 | 30 | 15=5×3，10=5×2；lcm=5×3×2=30 |
| 7, 3 | 1 | 21 | 互质；lcm=7×3=21 |

## 提示

**欧几里得算法（辗转相除法）**求 GCD：

$$\gcd(a, b) = \begin{cases} a & b = 0 \\ \gcd(b,\ a \bmod b) & b \neq 0 \end{cases}$$

```cpp
int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}
```

**LCM 与 GCD 的关系**：

$$\text{lcm}(a, b) = \frac{a \times b}{\gcd(a, b)}$$

注意：先除后乘避免溢出：`a / gcd(a, b) * b`。
