## 测试点 4（输入乱序，验证排序正确性）

**输入**
```
4
5 8
1 4
3 6
7 9
```
**预期输出**
```
2
```

**说明**：按结束时间排序后顺序为 (1,4),(3,6),(5,8),(7,9)。
- 选 (1,4)，last=4，cnt=1
- (3,6)：3 < 4，跳过
- (5,8)：5 ≥ 4，选，last=8，cnt=2
- (7,9)：7 < 8，跳过

共选 2 个：(1,4) 和 (5,8)。

## 测试点 5（n=1）

**输入**
```
1
5 10
```
**预期输出**
```
1
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

struct Event {
    int start, end;
};

bool cmp(Event a, Event b) {
    return a.end < b.end;
}

int main() {
    int n;
    cin >> n;

    Event events[100];
    for (int i = 0; i < n; i++)
        cin >> events[i].start >> events[i].end;

    sort(events, events + n, cmp);

    int cnt = 0, last = -1;
    for (int i = 0; i < n; i++) {
        if (events[i].start >= last) {
            cnt++;
            last = events[i].end;
        }
    }

    cout << cnt << endl;
    return 0;
}
```
