## 测试点 3（偶数长度回文与非回文）

**输入**
```

```

```
Yes
```

**说明**：abba逆序abba✓；abcd逆序dcba✗；aabbaa逆序aabbaa✓。

## 测试点 5（长字符串边界）

**输入**
```
2
abcdefedcba
abcdefghijk
```
**预期输出**
```
Yes
No
```

**说明**：abcdefedcba长度11，中心为e，正逆序相同✓；abcdefghijk所有字符各异✗。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        string s;
        cin >> s;
        string rev = s;
        reverse(rev.begin(), rev.end());
        cout << (s == rev ? "Yes" : "No") << endl;
    }
    return 0;
}
```
