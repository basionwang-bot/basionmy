## 测试点 2（n=1）

**输入**
```
1
75
```
**预期输出**
```
75
75
75.0
1
```

**说明**：只有一人，最高=最低=平均=75，达标1人。

---

## 测试点 3（全部达标）

**输入**
```
4
60 70 80 90
```
**预期输出**
```
90
60
75.0
4
```

**说明**：平均=(60+70+80+90)/4=300/4=75.0，4人全达标。

---

## 测试点 4（无人达标）

**输入**
```
3
30 45 59
```
**预期输出**
```
59
30
44.7
0
```

**说明**：平均=(30+45+59)/3=134/3≈44.7，无人达标。

---

## 测试点 5（恰好60分边界）

**输入**
```
4
60 59 61 60
```
**预期输出**
```
61
59
60.0
3
```

**说明**：60、61、60 达标，59 不达标；平均=(60+59+61+60)/4=240/4=60.0。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int n;
    cin >> n;

    int maxv = -1, minv = 101, cnt = 0;
    double total = 0;

    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        total += x;
        if (x > maxv) maxv = x;
        if (x < minv) minv = x;
        if (x >= 60) cnt++;
    }

    cout << maxv << endl;
    cout << minv << endl;
    cout << fixed << setprecision(1) << total / n << endl;
    cout << cnt << endl;

    return 0;
}
```
