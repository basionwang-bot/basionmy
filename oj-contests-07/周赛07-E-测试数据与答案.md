# E - 宝箱配对 · 测试数据与答案

## 测试点 1（样例）

**输入**
```
5 15
6 7 9 8 3
```
**预期输出**
```
1 3
2 4
```

**说明**：(1,3)=6+9=15 ✓，(2,4)=7+8=15 ✓。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, T;
    cin >> n >> T;

    int a[100];
    for (int i = 0; i < n; i++) cin >> a[i];

    bool found = false;
    for (int i = 0; i < n - 1; i++) {        // 外层：枚举第一个宝箱
        for (int j = i + 1; j < n; j++) {    // 内层：枚举第二个宝箱（保证 j > i）
            if (a[i] + a[j] == T) {
                cout << i + 1 << " " << j + 1 << endl;
                found = true;
            }
        }
    }

    if (!found) cout << "None" << endl;

    return 0;
}
```

    for (int j = i + 1; j < n; j++) {   // j 从 i+1 到最后一个，保证 j > i 避免重复
        if (a[i] + a[j] == T) { ... }
    }
}
// 总比较次数 = n*(n-1)/2，这是枚举所有无序对的标准双层循环写法
