## 测试点 2（背包满时继续 put）

**输入**
```
2 4
put axe
put helmet
put boots
take axe
```
**预期输出**
```
1
2
Full
1
1
```

**说明**：第 3 次 put 时背包已满（2 格均占用），输出 `Full`；随后取出 axe，格子 1 清空；最终剩余 1 件（helmet）。

## 测试点 4（取出再放入，复用格子）

**输入**
```

```
**预期输出**

**说明**：
- put a → 格子1；put b → 格子2；take a → 清空格子1，输出1
- put c → 格子1（最小空格）；take b → 清空格子2，输出2
- put d → 格子2（最小空格）；最终剩余2件。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;

    string bag[20];
    for (int i = 0; i < n; i++) bag[i] = ""; // 初始化为空

    for (int q = 0; q < m; q++) {
        string op, item;
        cin >> op >> item;

        if (op == "put") {
            int slot = -1;
            for (int i = 0; i < n; i++) {
                if (bag[i] == "") { slot = i; break; }
            }
            if (slot == -1) {
                cout << "Full" << endl;
            } else {
                bag[slot] = item;
                cout << slot + 1 << endl; // 1-indexed
            }
        } else { // take
            int slot = -1;
            for (int i = 0; i < n; i++) {
                if (bag[i] == item) { slot = i; break; }
            }
            if (slot == -1) {
                cout << "Not found" << endl;
            } else {
                bag[slot] = "";
                cout << slot + 1 << endl; // 1-indexed
            }
        }
    }

    // 统计剩余物品数
    int count = 0;
    for (int i = 0; i < n; i++)
        if (bag[i] != "") count++;
    cout << count << endl;

    return 0;
}
```
