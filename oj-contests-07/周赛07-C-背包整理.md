# C - 背包整理

**难度**：★★☆☆☆
**考点**：模拟、字符串操作、数组状态维护

---

## 题目背景

冒险者的背包有 $n$ 个编号为 $1$ 到 $n$ 的格子，每个格子最多放一件物品。出发前需要对背包进行一系列整理操作：放入物品或取出物品。请你模拟整个整理过程。

## 题目描述

背包初始为空，依次执行 $m$ 条操作：

- `put 物品名`：将物品放入编号**最小的空格子**，输出该格子的编号；若背包已满（无空格子），输出 `Full`。
- `take 物品名`：从背包中取出该物品，输出其所在的格子编号，并将该格子清空；若背包中不存在该物品，输出 `Not found`。

所有操作执行完毕后，在最后一行输出背包中剩余的物品数量。

## 输入格式

第一行两个整数 $n$ 和 $m$（$1 \leq n \leq 20$，$1 \leq m \leq 50$）。

接下来 $m$ 行，每行一条操作，格式为 `put 物品名` 或 `take 物品名`（物品名仅含小写英文字母，长度不超过 10）。

## 输出格式

共 $m + 1$ 行：前 $m$ 行每行输出对应操作的结果，最后一行输出剩余物品数量。

## 样例

**输入**

```
3 5
put sword
put shield
put potion
take shield
put bow
```

**输出**

```
1
2
3
2
2
3
```

**说明**

| 操作 | 执行结果 | 背包状态 |
|------|----------|----------|
| put sword | 放入格子 1，输出 `1` | [sword, -, -] |
| put shield | 放入格子 2，输出 `2` | [sword, shield, -] |
| put potion | 放入格子 3，输出 `3` | [sword, shield, potion] |
| take shield | 格子 2 取出，输出 `2` | [sword, -, potion] |
| put bow | 放入格子 2（最小空格），输出 `2` | [sword, bow, potion] |

最终背包有 3 件物品，输出 `3`。

## 提示

- 用字符串数组 `string bag[20]` 表示背包，空格子存储空字符串 `""`。
- `put` 操作：从编号 1 开始顺序找第一个 `bag[i] == ""` 的格子。
- `take` 操作：遍历数组找到物品名匹配的格子，清空后输出编号。
- 注意：背包中可能有多件同名物品——本题保证同一时刻背包中每件物品名唯一，`take` 找到第一个匹配即可。
