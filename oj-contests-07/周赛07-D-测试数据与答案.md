## 测试点 2（含负数，最大值为负）

**输入**
```
3 3
-1 -2 -3
-4 -5 -6
-7 -8 -9
```
**预期输出**
```
1 1
1
-7 -4 -1
-8 -5 -2
-9 -6 -3
```

**说明**：
- 最大值为 -1，在 (1,1)。
- 行和：-6、-15、-24，最大为第 1 行。
- 旋转后（3×3）：第 j 列原从上到下为 col，旋转后新第 j 行从左到右为该列自下而上。

## 测试点 5（1×n 行向量旋转为 n×1 列向量）

**输入**
```
1 4
3 1 4 2
```
**预期输出**
```
1 3
1
3
1
4
2
```

**说明**：
- 最大值 4 在 (1,3)；只有一行，最优行为第 1 行。
- 旋转后（4×1）：原矩阵第 j 列变为新矩阵第 j 行，仅 1 列，从上到下分别为 3、1、4、2。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;

    int a[20][20];
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            cin >> a[i][j];

    // 1. 找最大值坐标（行优先，列次之）
    int maxVal = a[0][0], maxRow = 0, maxCol = 0;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            if (a[i][j] > maxVal) {
                maxVal = a[i][j];
                maxRow = i;
                maxCol = j;
            }
        }
    }
    cout << maxRow + 1 << " " << maxCol + 1 << endl;

    // 2. 找行总和最大的行号
    int maxRowSum = -1000000, maxRowIdx = 0;
    for (int i = 0; i < n; i++) {
        int rowSum = 0;
        for (int j = 0; j < m; j++) rowSum += a[i][j];
        if (rowSum > maxRowSum) {
            maxRowSum = rowSum;
            maxRowIdx = i;
        }
    }
    cout << maxRowIdx + 1 << endl;

    // 3. 顺时针旋转 90°：b[j][n-1-i] = a[i][j]，新矩阵大小 m×n
    int b[20][20];
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            b[j][n - 1 - i] = a[i][j];

    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            if (j > 0) cout << " ";
            cout << b[i][j];
        }
        cout << endl;
    }

    return 0;
}
```

### 旋转公式推导

顺时针旋转 90°，直觉理解：
- 原矩阵第 1 列（从上到下）→ 新矩阵第 1 行（从左到右）
- 原矩阵第 2 列（从上到下）→ 新矩阵第 2 行（从左到右）
- ……

用 0-indexed 表示：原第 $i$ 行第 $j$ 列 → 新第 $j$ 行第 $(n-1-i)$ 列。

```cpp
b[j][n - 1 - i] = a[i][j];
// 新矩阵行数 = m，列数 = n
```
