## 测试点 2（恰好整除）

**输入**
```
2
40 100 60
0 200 50
```
**预期输出**
```
1
4
```

**说明**：
- 需补 60 点，60÷60=1，整除无需进位。
- 需补 200 点，200÷50=4。

---

## 测试点 3（差 1 点需多一瓶）

**输入**
```
1
99 100 10
```
**预期输出**
```
1
```

**说明**：差 1 点，每瓶 10，⌈1÷10⌉=1。

## 测试点 5（多组混合）

**输入**
```
5
1 10000 1
5000 10000 3333
9999 10000 1
0 1 1
500 500 100
```
**预期输出**
```
9999
2
1
1
0
```

**说明**：
- 差 9999 点，每瓶 1，需 9999 瓶。
- 差 5000 点，每瓶 3333，⌈5000÷3333⌉=2（3333+3333=6666≥5000）。
- 差 1 点，每瓶 1，需 1 瓶。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int h, M, v;
        cin >> h >> M >> v;
        cout << (M - h + v - 1) / v << endl;
    }
    return 0;
}
```
