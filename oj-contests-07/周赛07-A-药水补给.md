# A - 药水补给

**难度**：★☆☆☆☆
**考点**：向上取整、基础运算、多组输入

---

## 题目背景

在 RPG 冒险中，角色受伤后需要使用药水恢复血量。每瓶药水可以恢复固定点数的血量，但不会超过血量上限。勇者想知道至少需要喝几瓶药水才能回满血。

## 题目描述

给定若干组数据，每组给出角色当前血量 $h$、血量上限 $M$ 和每瓶药水的回血量 $v$，请计算至少需要几瓶药水才能使血量达到上限 $M$。

若当前已满血，输出 $0$。

## 输入格式

第一行一个整数 $T$（$1 \leq T \leq 100$），表示测试组数。

接下来 $T$ 行，每行三个整数 $h$、$M$、$v$（$1 \leq h \leq M \leq 10000$，$1 \leq v \leq M$）。

## 输出格式

共 $T$ 行，每行一个整数，表示所需药水的最少瓶数。

## 样例

**输入**

```
3
70 100 15
50 100 50
100 100 10
```

**输出**

```
2
1
0
```

**说明**

- 第 1 组：需补充 30 点血量，每瓶 +15，需要 ⌈30÷15⌉ = 2 瓶。
- 第 2 组：需补充 50 点血量，每瓶 +50，需要 ⌈50÷50⌉ = 1 瓶。
- 第 3 组：已满血，输出 0。

## 提示

- 关键公式：$\lceil (M - h) \div v \rceil$，即向上取整。
- C++ 中对正整数做向上取整：`(M - h + v - 1) / v`（当 $M - h = 0$ 时结果自然为 0）。
