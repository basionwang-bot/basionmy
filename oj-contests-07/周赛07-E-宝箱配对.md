# E - 宝箱配对

**难度**：★★☆☆☆
**考点**：双层循环枚举、条件判断

---

## 题目背景

勇者在地牢中发现了 $n$ 个宝箱，每个宝箱里有一定数量的金币。传说中，有一扇神秘的门需要同时打开**恰好两个**宝箱才能开启，且这两个宝箱的金币之和必须恰好等于指定值 $T$。勇者想找出所有满足条件的宝箱对。

## 题目描述

给定 $n$ 个宝箱的金币数量和目标值 $T$，找出所有金币之和恰好等于 $T$ 的宝箱对 $(i, j)$（$i < j$），按 $i$ 从小到大、$i$ 相同则 $j$ 从小到大的顺序输出。

若不存在满足条件的宝箱对，输出 `None`。

## 输入格式

第一行两个整数 $n$ 和 $T$（$2 \leq n \leq 100$，$1 \leq T \leq 10000$）。

第二行 $n$ 个整数，第 $i$ 个表示第 $i$ 号宝箱的金币数量（$1 \leq$ 金币数 $\leq 5000$）。

## 输出格式

若存在满足条件的宝箱对，每行输出一对宝箱编号（1-indexed），用空格分隔，按 $i$ 升序，$i$ 相同则按 $j$ 升序；若不存在，输出 `None`。

## 样例

**输入**

```
5 15
6 7 9 8 3
```

**输出**

```
1 3
2 4
```

**说明**

枚举所有 $i < j$ 的宝箱对，检查金币之和是否等于 15：

| 宝箱对 | 金币 | 之和 | 是否等于 15 |
|--------|------|------|------------|
| (1, 2) | 6, 7 | 13 | ✗ |
| **(1, 3)** | **6, 9** | **15** | **✓** |
| (1, 4) | 6, 8 | 14 | ✗ |
| (1, 5) | 6, 3 | 9 | ✗ |
| (2, 3) | 7, 9 | 16 | ✗ |
| **(2, 4)** | **7, 8** | **15** | **✓** |
| (2, 5) | 7, 3 | 10 | ✗ |
| (3, 4) | 9, 8 | 17 | ✗ |
| (3, 5) | 9, 3 | 12 | ✗ |
| (4, 5) | 8, 3 | 11 | ✗ |

## 提示

- 外层循环枚举第一个宝箱编号 $i$（从 1 到 $n-1$）；
- 内层循环枚举第二个宝箱编号 $j$（从 $i+1$ 到 $n$）；
- 若 `a[i] + a[j] == T`，输出 `i j`；
- 用布尔变量记录是否找到过满足条件的对，循环结束后若没找到则输出 `None`。
