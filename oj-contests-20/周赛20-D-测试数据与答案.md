# D - 快速求幂 · 测试数据与答案

---

## 测试点 1（样例1）

**输入**
```
2 10 1000
```
**预期输出**
```
24
```
**验证**：$2^{10} = 1024$，$1024 \bmod 1000 = 24$。

---

## 测试点 2（样例2：n=0）

**输入**
```
3 0 7
```
**预期输出**
```
1
```
**验证**：任意数的 0 次幂为 1，$1 \bmod 7 = 1$。

---

## 测试点 3（样例3）

**输入**
```
5 3 13
```
**预期输出**
```
8
```
**验证**：$5^3 = 125$，$125 = 13 \times 9 + 8$，余数为 8。

---

## 测试点 4（m=1，结果必为0）

**输入**
```
999999999 1000000000 1
```
**预期输出**
```
0
```
**验证**：任何整数模 1 均为 0。

---

## 测试点 5（a=1，结果必为1）

**输入**
```
1 1000000000 1000000000
```
**预期输出**
```
1
```
**验证**：$1^n = 1$，$1 \bmod 10^9 = 1$。

---

## 测试点 6（大指数）

**输入**
```
2 30 1000000007
```
**预期输出**
```
1073741824
```
**验证**：$2^{30} = 1073741824$，该值小于 $10^9 + 7$，故余数为本身。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
long long power(long long a, long long n, long long m) {
    if (n == 0) return 1;
    if (n % 2 == 0) {
        long long half = power(a, n / 2, m);
        return half * half % m;
    }
    return a % m * power(a, n - 1, m) % m;
}
int main() {
    long long a, n, m;
    cin >> a >> n >> m;
    cout << power(a, n, m) << "\n";
    return 0;
}
```

**关键点**：
1. 递归边界：$n=0$ 时返回 1；
2. 偶数分支：递归算 $a^{n/2}$，结果平方后取模，避免重复计算；
3. 奇数分支：提出一个 $a$，令指数减 1 变偶；
4. 所有乘法结果可能超 `int`，全程使用 `long long`；
5. 递归深度为 $O(\log n)$，约 30 层，效率极高。
