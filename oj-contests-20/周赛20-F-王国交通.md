# F - 王国交通

**难度**：★★★★★★★  
**考点**：图存储（邻接表）+ Dijkstra 单源最短路  
**GESP 级别**：7级

---

## 题目背景

数字王国有 $n$ 座城市，城市间通过有向道路相连，每条道路都有通行费用（正整数权重）。国王想知道从首都（城市 1）出发，到达其余各城市的最低费用是多少。

## 题目描述

给定 $n$ 个城市（编号 $1$ 到 $n$）和 $m$ 条有向带权边，求从城市 $1$ 出发到达其他每个城市的**最短距离**。若某城市不可达，输出 `-1`。

## 输入格式

第一行两个整数 $n$、$m$（$2 \le n \le 500$，$0 \le m \le 5000$）。

接下来 $m$ 行，每行三个整数 $u$、$v$、$w$，表示从城市 $u$ 到城市 $v$ 有一条权重为 $w$ 的有向边（$1 \le u, v \le n$，$1 \le w \le 10^4$）。

## 输出格式

共 $n-1$ 行，第 $i$ 行（$i$ 从 2 到 $n$）输出：

```
i: 距离
```

若城市 $i$ 不可达，输出 `i: -1`。

## 样例

**输入**

```
5 6
1 2 2
1 3 9
2 3 6
2 4 4
3 5 1
4 5 3
```

**输出**

```
2: 2
3: 8
4: 6
5: 9
```

**说明**：

- 城市 2：$1 \to 2$，距离 $2$。
- 城市 3：$1 \to 2 \to 3$，距离 $2+6=8$（直达 $1 \to 3$ 需 9，更远）。
- 城市 4：$1 \to 2 \to 4$，距离 $2+4=6$。
- 城市 5：$1 \to 2 \to 4 \to 5$，距离 $2+4+3=9$（经过城市 3 需 $8+1=9$ 相同，取其一）。

## 提示

使用**邻接表**存图，配合**优先队列（小根堆）**实现 Dijkstra 算法：

```cpp
#include <vector>
#include <queue>
#include <climits>
using namespace std;

typedef pair<int,int> pii;  // (距离, 节点)
vector<pii> graph[505];     // graph[u] = {(v, w), ...}
int dist[505];

void dijkstra(int src, int n) {
    fill(dist + 1, dist + n + 1, INT_MAX);
    dist[src] = 0;
    priority_queue<pii, vector<pii>, greater<pii>> pq;
    pq.push({0, src});
    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (d > dist[u]) continue;  // 跳过过时状态
        for (auto [v, w] : graph[u]) {
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
}
```

初始化 `dist` 为极大值（表示不可达），松弛时更新。最终 `dist[i]` 仍为极大值则输出 `-1`。
