# A - 数字指纹

**难度**：★☆☆☆☆  
**考点**：循环、取余取整操作、多组输入  
**GESP 级别**：1级

---

## 题目背景

数字王国的每个公民都有一个正整数编号。王国档案馆需要为每个编号生成一份"数字指纹"——即该编号的各位数字之和，用于快速校验身份。

## 题目描述

给定 $T$ 组查询，每组给定一个正整数 $n$，输出 $n$ 的各位数字之和。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 100$）。

接下来 $T$ 行，每行一个正整数 $n$（$1 \le n \le 10^9$）。

## 输出格式

共 $T$ 行，每行输出对应的各位数字之和。

## 样例

**输入**

```
3
12345
100
9
```

**输出**

```
15
1
9
```

**说明**：$12345$ 的各位之和为 $1+2+3+4+5=15$；$100$ 的各位之和为 $1+0+0=1$；$9$ 的各位之和为 $9$。

## 提示

逐位取出数字：每次对 10 取余得到个位，再除以 10 去掉个位，重复直到 $n=0$。

```cpp
int sum = 0;
while (n > 0) {
    sum += n % 10;
    n /= 10;
}
```
