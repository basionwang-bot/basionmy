## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
#include <unordered_map>
using namespace std;
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n; cin >> n;
    unordered_map<int, string> mp;
    for (int i = 0; i < n; i++) {
        int id; string name;
        cin >> id >> name;
        mp[id] = name;
    }
    int m; cin >> m;
    while (m--) {
        int q; cin >> q;
        if (mp.count(q)) cout << mp[q] << "\n";
        else cout << "NOT_FOUND\n";
    }
    return 0;
}
```

**关键点**：
1. 使用 `unordered_map<int, string>` 实现 $O(1)$ 平均查询；
2. 加入 `ios::sync_with_stdio(false)` 和 `cin.tie(nullptr)` 加速 I/O，应对 $10^5$ 级数据；
3. 用 `mp.count(q)` 判断键是否存在，返回 0 或 1；
4. 若用 `mp[q]` 直接访问不存在的键，会插入空字符串，需避免在判断前直接访问。
