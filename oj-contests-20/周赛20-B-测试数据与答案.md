# B - 最大利润 · 测试数据与答案

---

## 测试点 1（样例1）

**输入**
```
6
7 1 5 3 6 4
```
**预期输出**
```
5
```
**说明**：第 2 天买（价格 1），第 5 天卖（价格 6），利润 $6-1=5$。

**输入**

**说明**：买卖价格相同，利润为 0。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    int n; cin >> n;
    int prices[100005];
    for (int i = 0; i < n; i++) cin >> prices[i];
    int minPrice = prices[0], ans = 0;
    for (int i = 1; i < n; i++) {
        ans = max(ans, prices[i] - minPrice);
        minPrice = min(minPrice, prices[i]);
    }
    cout << ans << "\n";
    return 0;
}
```

**关键点**：
1. `minPrice` 维护当前已遍历位置中的最低价，保证买在卖之前；
2. 每步先尝试以当前价格卖出更新答案，再更新最低价；
3. 若不可能盈利（价格不升），`ans` 始终为 0，符合题意。
