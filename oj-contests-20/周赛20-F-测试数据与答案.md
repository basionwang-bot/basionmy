## 测试点 1（样例）

**输入**
```
5 6
1 2 2
1 3 9
2 3 6
2 4 4
3 5 1
4 5 3
```
**预期输出**
```
2: 2
3: 8
4: 6
5: 9
```
**说明**：
- $1\to2$：2
- $1\to2\to3$：$2+6=8$（优于直达 $1\to3=9$）
- $1\to2\to4$：$2+4=6$
- $1\to2\to4\to5$：$2+4+3=9$

## 测试点 4（存在多条路径，取最短）

**输入**
```
4 5
1 2 10
1 3 1
3 2 2
2 4 5
3 4 20
```
**预期输出**
```
2: 3
3: 1
4: 8
```
**说明**：
- 城市 2：直达 $1\to2=10$，经 3 为 $1\to3\to2=1+2=3$，取最短 3。
- 城市 3：$1\to3=1$。
- 城市 4：$1\to3\to2\to4=1+2+5=8$（优于 $1\to3\to4=1+20=21$，也优于 $1\to2\to4=10+5=15$）。

## 测试点 6（有向边，注意方向）

**输入**
```
3 2
1 2 4
3 1 2
```
**预期输出**
```
2: 4
3: -1
```
**说明**：边 $3\to1$ 方向是从 3 到 1，从城市 1 出发无法到达城市 3，输出 -1。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <climits>
using namespace std;

typedef pair<int,int> pii;
vector<pii> graph[505];
int dist[505];

void dijkstra(int src, int n) {
    fill(dist + 1, dist + n + 1, INT_MAX);
    dist[src] = 0;
    priority_queue<pii, vector<pii>, greater<pii>> pq;
    pq.push({0, src});
    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (d > dist[u]) continue;
        for (auto [v, w] : graph[u]) {
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, m; cin >> n >> m;
    for (int i = 0; i < m; i++) {
        int u, v, w; cin >> u >> v >> w;
        graph[u].push_back({v, w});
    }
    dijkstra(1, n);
    for (int i = 2; i <= n; i++) {
        if (dist[i] == INT_MAX) cout << i << ": -1\n";
        else cout << i << ": " << dist[i] << "\n";
    }
    return 0;
}
```

**关键点**：
1. 邻接表 `vector<pair<int,int>> graph[505]` 存有向边；
2. `dist` 初始化为 `INT_MAX` 表示不可达，源点 `dist[1]=0`；
3. 优先队列存 `(距离, 节点)`，使用 `greater<pii>` 实现小根堆；
4. 弹出节点时检查 `d > dist[u]`，跳过已过时的状态（重要优化）；
5. 松弛成功则将新状态压入队列；
6. 最终 `dist[i]` 仍为 `INT_MAX` 说明不可达，输出 `-1`。
