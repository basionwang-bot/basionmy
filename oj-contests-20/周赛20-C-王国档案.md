# C - 王国档案

**难度**：★★★☆☆  
**考点**：结构体 + 条件筛选 + 多关键字排序  
**GESP 级别**：3级

---

## 题目背景

数字王国举办年度优秀公民评选活动。王国档案馆记录了每位公民的姓名、所在城市和综合评分，需要从中筛选出达标公民并按规则公示名单。

## 题目描述

给定 $n$ 名公民的信息（姓名、城市、综合评分），筛选**综合评分 $\ge 60$** 的公民，并按以下规则排序后输出姓名：

1. 综合评分**从大到小**；
2. 评分相同时，姓名**字典序从小到大**。

若无人达标，输出 `none`。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，每行格式为：

```
姓名 城市 评分
```

姓名和城市均为不含空格的字符串（长度 $\le 20$），评分为整数（$0 \le$ 评分 $\le 100$）。

## 输出格式

按排序后顺序，将所有达标公民的姓名输出在一行，以空格隔开。若无人达标，输出 `none`。

## 样例

**输入**

```
5
Alice Beijing 85
Bob Shanghai 55
Carol Beijing 85
Dave Guangzhou 70
Eve Shanghai 45
```

**输出**

```
Alice Carol Dave
```

**说明**：Bob（55 < 60）和 Eve（45 < 60）不达标，淘汰。  
剩余三人：Alice（85）、Carol（85）、Dave（70）。  
按评分降序：Alice 和 Carol 均为 85，Dave 为 70，所以 Dave 排最后。  
Alice 和 Carol 同分，按姓名字典序：Alice < Carol，故 Alice 排前。  
最终顺序：Alice Carol Dave。

## 提示

```cpp
struct Citizen { string name, city; int score; };
bool cmp(const Citizen& a, const Citizen& b) {
    if (a.score != b.score) return a.score > b.score;
    return a.name < b.name;
}
```

筛选时将达标公民加入 `vector`，排序后依次输出。
