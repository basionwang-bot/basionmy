# A - 数字指纹 · 测试数据与答案

---

## 测试点 1（样例）

**输入**
```
3
12345
100
9
```
**预期输出**
```
15
1
9
```
**说明**：$1+2+3+4+5=15$；$1+0+0=1$；$9=9$。

---

## 测试点 2（边界：n=1 与 n=10^9）

**输入**
```
2
1
1000000000
```
**预期输出**
```
1
1
```
**说明**：$n=1$ 各位之和为 1；$1000000000$ 各位之和为 $1+0+\cdots+0=1$。

---

## 测试点 3（各位均为 9）

**输入**
```
2
999
99999999
```
**预期输出**
```
27
72
```
**说明**：$999$ 有 3 个 9，和为 $27$；$99999999$ 有 8 个 9，和为 $72$。

---

## 测试点 4（混合测试）

**输入**
```
5
1
10
99
1001
987654321
```
**预期输出**
```
1
1
18
2
45
```
**说明**：$987654321$：$9+8+7+6+5+4+3+2+1=45$。

---

## 测试点 5（单个多位数）

**输入**
```
1
123456789
```
**预期输出**
```
45
```
**说明**：$1+2+3+4+5+6+7+8+9=45$。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    int T; cin >> T;
    while (T--) {
        long long n; cin >> n;
        int sum = 0;
        while (n > 0) {
            sum += n % 10;
            n /= 10;
        }
        cout << sum << "\n";
    }
    return 0;
}
```

**关键点**：用 `long long` 或 `int` 均可接收 $n$（$10^9$ 在 `int` 范围内）；循环条件为 `n > 0`，不会遗漏任何位。
