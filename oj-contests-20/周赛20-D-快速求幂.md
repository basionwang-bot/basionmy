# D - 快速求幂

**难度**：★★★★☆  
**考点**：递归实现快速幂（分治思想）  
**GESP 级别**：4级

---

## 题目背景

数字王国的密码系统需要频繁计算幂次取模。若用普通循环将 $a$ 连乘 $n$ 次，当 $n$ 高达 $10^9$ 时效率极低。快速幂算法利用分治思想，将时间复杂度从 $O(n)$ 降至 $O(\log n)$。

## 题目描述

给定整数 $a$、$n$、$m$，计算 $a^n \bmod m$。

## 输入格式

一行三个整数 $a$、$n$、$m$（$1 \le a, m \le 10^9$，$0 \le n \le 10^9$）。

## 输出格式

输出一个整数，表示 $a^n \bmod m$ 的结果。

## 样例

**输入 1**

```
2 10 1000
```

**输出 1**

```
24
```

**说明**：$2^{10} = 1024$，$1024 \bmod 1000 = 24$。

---

**输入 2**

```
3 0 7
```

**输出 2**

```
1
```

**说明**：任意整数的 0 次幂为 1，$1 \bmod 7 = 1$。

---

**输入 3**

```
5 3 13
```

**输出 3**

```
8
```

**说明**：$5^3 = 125$，$125 \bmod 13 = 8$（因为 $13 \times 9 = 117$，$125 - 117 = 8$）。

## 提示

快速幂的递归思路：

- $a^0 = 1$（递归边界）
- $n$ 为偶数：$a^n = (a^{n/2})^2$，只需递归计算一半，结果平方取模
- $n$ 为奇数：$a^n = a \times a^{n-1}$，将 $n$ 减 1 变为偶数

```cpp
long long power(long long a, long long n, long long m) {
    if (n == 0) return 1;
    if (n % 2 == 0) {
        long long half = power(a, n / 2, m);
        return half * half % m;
    }
    return a % m * power(a, n - 1, m) % m;
}
```

注意：`half * half` 和 `a * power(...)` 的中间结果可能超过 `int` 范围，务必使用 `long long`。
