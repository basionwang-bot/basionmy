## 测试点 1（样例）

**输入**
```
5
Alice Beijing 85
Bob Shanghai 55
Carol Beijing 85
Dave Guangzhou 70
Eve Shanghai 45
```
**预期输出**
```
Alice Carol Dave
```
**说明**：Bob 和 Eve 不达标；Alice 和 Carol 同分 85，字典序 Alice < Carol；Dave 70 排最后。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>
using namespace std;
struct Citizen { string name, city; int score; };
bool cmp(const Citizen& a, const Citizen& b) {
    if (a.score != b.score) return a.score > b.score;
    return a.name < b.name;
}
int main() {
    int n; cin >> n;
    vector<Citizen> res;
    for (int i = 0; i < n; i++) {
        Citizen c; cin >> c.name >> c.city >> c.score;
        if (c.score >= 60) res.push_back(c);
    }
    if (res.empty()) {
        cout << "none\n";
    } else {
        sort(res.begin(), res.end(), cmp);
        for (int i = 0; i < (int)res.size(); i++) {
            if (i > 0) cout << " ";
            cout << res[i].name;
        }
        cout << "\n";
    }
    return 0;
}
```

**关键点**：
1. 筛选条件为 `score >= 60`（含 60）；
2. 排序主键为评分降序，次键为姓名字典序升序；
3. 输出格式为同一行空格分隔，与样例一致。
