 # basionmy

这里是我在手机上记灵感、整理想法的仓库。

## 快速入口

- [INBOX（灵感收集箱）](./INBOX.md)
- [TODAY（今日焦点）](./TODAY.md)
- [IDEAS（想法库）](./IDEAS.md)
- [PROJECTS（项目总览）](./PROJECTS.md)
- [QUICK-CAPTURE（快速记录）](./QUICK-CAPTURE.md)

## 使用习惯（可以按自己喜好改）

- 随手想到什么就写进 `INBOX.md` 或 `QUICK-CAPTURE.md`
- 有空的时候，把 `INBOX` 里的内容整理到 `IDEAS.md` / `PROJECTS.md`
- 每天早上简单更新一下 `TODAY.md`
