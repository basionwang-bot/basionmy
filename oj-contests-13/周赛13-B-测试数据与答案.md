## 测试点 2（边界：0 和自身异或）

**输入**
```
4
0 0 1
0 0 2
7 7 3
255 0 3
```
**预期输出**
```
0
0
0
255
```

**说明**：
- `0 & 0 = 0`，`0 | 0 = 0`
- `7 ^ 7 = 0`（任何数与自身异或为 0）
- `255 ^ 0 = 255`（与 0 异或不变）

## 测试点 3（左移，结果是 2 的幂次）

**输入**
```
4
1 0 4
1 4 4
1 7 4
128 0 5
```
**预期输出**
```
1
16
128
128
```

**说明**：`1 << k` = $2^k$；`128 >> 0 = 128`。

---

## 测试点 4（按位与、或的掩码应用）

**输入**
```
3
255 15 1
255 15 2
170 85 1
```
**预期输出**
```
15
255
0
```

**说明**：
- `255 & 15 = 11111111 & 00001111 = 00001111 = 15`（低 4 位掩码）
- `255 | 15 = 11111111`（或运算结果仍 255）
- `170 & 85 = 10101010 & 01010101 = 00000000 = 0`（互补）

---

## 测试点 5（右移，整除效果）

**输入**
```
3
100 2 5
200 3 5
128 7 5
```
**预期输出**
```
25
25
1
```

**说明**：`100 >> 2 = 100/4 = 25`；`200 >> 3 = 200/8 = 25`；`128 >> 7 = 1`。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int t;
    cin >> t;
    while (t--) {
        int a, b, op;
        cin >> a >> b >> op;
        if      (op == 1) cout << (a & b)  << "\n";
        else if (op == 2) cout << (a | b)  << "\n";
        else if (op == 3) cout << (a ^ b)  << "\n";
        else if (op == 4) cout << (a << b) << "\n";
        else              cout << (a >> b) << "\n";
    }
    return 0;
}
```

**关键点**：
1. 五种运算符在 C++ 中直接可用，无需额外头文件；
2. `a <= 255`、`b <= 8`，左移最大结果 `255 << 8 = 65280`，不会溢出 `int`；
3. `a ^ a = 0` 和 `a ^ 0 = a` 是两个常用性质，值得记住；
4. 右移对负数有符号扩展问题，但本题 `a >= 0`，不涉及。
