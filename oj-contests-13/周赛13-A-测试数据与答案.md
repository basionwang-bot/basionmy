## 测试点 4（恰好 1000 不算丰收）

**输入**
```
1000 1000 1000
```
**预期输出**
```
3000 0
```

**说明**：题目要求严格大于 1000，恰好等于 1000 不计入丰收季数。

## 测试点 5（只有一季丰收）

**输入**
```
2000 1 1
```
**预期输出**
```
2002 1
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b, c;
    cin >> a >> b >> c;
    int total = a + b + c;
    int cnt = 0;
    if (a > 1000) cnt++;
    if (b > 1000) cnt++;
    if (c > 1000) cnt++;
    cout << total << " " << cnt << "\n";
    return 0;
}
```

**关键点**：
1. 三个独立的 `if` 语句，互不影响；
2. 严格大于：用 `>` 而非 `>=`；
3. 输出总量和计数之间用一个空格分隔。
