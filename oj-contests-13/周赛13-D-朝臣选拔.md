# D - 朝臣选拔

**难度**：★★★☆☆  
**考点**：结构体 + 多关键字筛选排序  
**GESP 级别**：3级

---

## 题目背景

皇帝举办朝臣选拔考试，共设文试和武试两科。只有文武双全者方可入选（两科成绩均须达标）。入选者按综合表现排名，文治优先、武功次之，姓名为最终决胜条件。

## 题目描述

给定 $n$ 位候选人，每人有姓名、文试成绩、武试成绩。

**入选条件**：文试成绩 $\ge 60$ **且** 武试成绩 $\ge 60$。

**排序规则**（对入选者）：
1. 文试成绩**降序**；
2. 文试相同时，武试成绩**降序**；
3. 两科均相同时，姓名**字典序升序**。

输出所有入选者的姓名（每行一个）。若无人入选，输出 `None`。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，每行：姓名（无空格，长度 $1 \sim 20$）、文试成绩、武试成绩（均为 $0 \sim 100$ 的整数）。

## 输出格式

若有入选者，每行输出一个姓名（按排序结果）；否则输出 `None`。

## 样例

**输入**

```
5
ZhaoYun 85 90
GuanYu 75 55
ZhangFei 60 80
LiuBei 45 70
CaoCao 90 60
```

**输出**

```
CaoCao
ZhaoYun
ZhangFei
```

**说明**

- GuanYu 武试 55 < 60，淘汰；LiuBei 文试 45 < 60，淘汰。
- 入选：CaoCao(90,60)、ZhaoYun(85,90)、ZhangFei(60,80)。
- 按文试降序：CaoCao(90) > ZhaoYun(85) > ZhangFei(60)。

## 提示

**结构体 + sort + 自定义比较函数**：

```cpp
struct Official {
    string name;
    int wen, wu;
};

bool cmp(const Official& a, const Official& b) {
    if (a.wen != b.wen) return a.wen > b.wen;   // 文试降序
    if (a.wu  != b.wu)  return a.wu  > b.wu;    // 武试降序
    return a.name < b.name;                      // 姓名字典序升序
}
```

先将入选者收集到 `vector` 中，再调用 `sort`。
