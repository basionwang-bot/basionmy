# A - 粮仓盘点

**难度**：★☆☆☆☆  
**考点**：条件分支、简单计算  
**GESP 级别**：1级

---

## 题目背景

朝廷每年分三季征粮：春、夏、秋。收粮之后，户部需要向皇帝汇报两件事：全年总收粮量，以及哪几季的收粮量超过了 1000 石（超过 1000 石才算"丰收"）。

## 题目描述

给定三季的收粮量 $a$、$b$、$c$（分别对应春、夏、秋），请输出：

1. 全年总收粮量；
2. 收粮量**严格大于** 1000 石的季数。

## 输入格式

一行三个整数 $a$、$b$、$c$（$1 \le a, b, c \le 2000$）。

## 输出格式

一行两个整数，用空格分隔：总收粮量 和 丰收季数。

## 样例

**输入**

```
800 1200 1500
```

**输出**

```
3500 2
```

**说明**

春季 800 石（不超过 1000，不算丰收），夏季 1200 石、秋季 1500 石均超过 1000 石，共 2 季丰收。总量 800+1200+1500=3500。

## 提示

逐一判断三个数是否严格大于 1000，用三个 `if` 语句分别计数即可。
