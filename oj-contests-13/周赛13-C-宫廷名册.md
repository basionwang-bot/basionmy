# C - 宫廷名册

**难度**：★★☆☆☆  
**考点**：字符数组（char 数组、`'\0'`、遍历）  
**GESP 级别**：2级

---

## 题目背景

礼部整理宫廷名册，所有官员的名字以**字符数组**的形式存储在档案中（每个名字是一串英文小写字母，以 `'\0'` 结尾）。礼部书记官需要统计每位官员姓名的长度，以及其中包含的元音字母数量（元音为 `a`、`e`、`i`、`o`、`u`），方便制作名牌。

## 题目描述

给定 $n$ 位官员的名字，每个名字用 **char 数组**读入。请对每个名字输出：名字的**长度**和其中**元音字母**的数量。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 10$）。

接下来 $n$ 行，每行一个名字（仅含英文小写字母，长度 $1 \sim 20$）。

## 输出格式

$n$ 行，每行两个整数，空格分隔：名字长度 和 元音字母数。

## 样例

**输入**

```
3
wang
king
emperor
```

**输出**

```
4 1
4 1
7 3
```

**说明**

- `wang`：长度 4，元音 `a`，共 1 个
- `king`：长度 4，元音 `i`，共 1 个
- `emperor`：长度 7，元音 `e`、`e`、`o`，共 3 个

## 提示

**char 数组**的核心操作：

```cpp
char name[25];
cin >> name;           // 读入（以空格/换行结束，自动添加 '\0'）
int len = strlen(name); // 求长度（需要 #include <cstring>）

// 遍历直到 '\0'
for (int i = 0; name[i] != '\0'; i++) {
    // 处理 name[i]
}
```

判断是否为元音：

```cpp
bool isVowel(char c) {
    return c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u';
}
```

char 数组与 string 的区别：char 数组是 C 风格字符串，必须预留足够空间（末尾有 `'\0'`），`strlen` 不计入 `'\0'`。
