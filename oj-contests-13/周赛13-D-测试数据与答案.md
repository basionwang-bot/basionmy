## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <string>
using namespace std;

struct Official {
    string name;
    int wen, wu;
};

bool cmp(const Official& a, const Official& b) {
    if (a.wen != b.wen) return a.wen > b.wen;
    if (a.wu  != b.wu)  return a.wu  > b.wu;
    return a.name < b.name;
}

int main() {
    int n;
    cin >> n;
    vector<Official> passed;
    for (int i = 0; i < n; i++) {
        Official o;
        cin >> o.name >> o.wen >> o.wu;
        if (o.wen >= 60 && o.wu >= 60) passed.push_back(o);
    }
    if (passed.empty()) {
        cout << "None\n";
    } else {
        sort(passed.begin(), passed.end(), cmp);
        for (auto& o : passed) cout << o.name << "\n";
    }
    return 0;
}
```

**关键点**：
1. 先筛选再排序，不要对所有人排序；
2. `cmp` 函数的三层嵌套对应三个排序关键字，顺序不能颠倒；
3. 字典序升序用 `a.name < b.name`（`string` 支持直接比较）；
4. `passed.empty()` 判断是否有人入选，输出 `None`。
