# E - 圣旨解读

**难度**：★★★☆☆  
**考点**：字符串 find + replace 综合应用（多轮替换）  
**GESP 级别**：3级

---

## 题目背景

皇帝颁布圣旨，为保密起见，圣旨中的关键词以"代号"代替。颁旨前，翰林院书记官需要按照密码本，将圣旨中所有代号依次替换为真实词语，还原圣旨原文。

## 题目描述

给定一段圣旨文本 $s$，以及 $n$ 条替换规则。每条规则包含旧词 $\text{old}$ 和新词 $\text{new}$，将当前文本中**所有**出现的 $\text{old}$ 替换为 $\text{new}$（从左向右不重叠替换）。

规则按顺序依次执行（上一条的结果作为下一条的输入）。输出最终文本。

## 输入格式

第一行：圣旨文本 $s$（长度 $1 \sim 200$，含小写字母和空格）。

第二行：整数 $n$（$1 \le n \le 5$）。

接下来 $2n$ 行，每两行为一条替换规则：第一行是 $\text{old}$，第二行是 $\text{new}$（均只含小写字母，长度 $1 \sim 20$）。

## 输出格式

一行，最终圣旨文本。

## 样例

**输入**

```
the king rewards the brave warrior
2
king
emperor
warrior
hero
```

**输出**

```
the emperor rewards the brave hero
```

**说明**

第一轮：将 `king` 替换为 `emperor` → `the emperor rewards the brave warrior`  
第二轮：将 `warrior` 替换为 `hero` → `the emperor rewards the brave hero`

## 提示

对每条规则，套用上一题（C题）学过的单词替换逻辑：

```cpp
void replaceAll(string& s, const string& old_str, const string& new_str) {
    size_t pos = 0;
    while ((pos = s.find(old_str, pos)) != string::npos) {
        s.replace(pos, old_str.size(), new_str);
        pos += new_str.size();  // 跳过新词，继续向右查找
    }
}
```

在主函数中循环 $n$ 次，每次调用一次 `replaceAll` 即可。

**注意**：用 `getline` 读入含空格的字符串；读完整数 $n$ 后需要用 `cin.ignore()` 消耗掉行尾换行符，再开始 `getline`。
