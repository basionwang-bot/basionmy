# B - 密令解析

**难度**：★★☆☆☆  
**考点**：位运算（&、|、^、<<、>>）  
**GESP 级别**：2级

---

## 题目背景

皇帝传递军机密令，为防止泄露，所有数值均经过位运算加密处理。密码官手中有一份操作手册，记录了五种运算符的编号：

| 编号 | 运算 | 含义 |
|------|------|------|
| 1 | `a & b` | 按位与：对应位均为 1 时结果为 1 |
| 2 | `a \| b` | 按位或：对应位有一个为 1 时结果为 1 |
| 3 | `a ^ b` | 按位异或：对应位不同时结果为 1 |
| 4 | `a << b` | 左移：将 a 的二进制左移 b 位（相当于乘以 $2^b$） |
| 5 | `a >> b` | 右移：将 a 的二进制右移 b 位（相当于除以 $2^b$ 取整） |

请对每条密令执行对应的位运算，输出结果。

## 题目描述

给定 $T$ 组操作，每组包含整数 $a$、$b$ 和操作编号 $\text{op}$（$1 \le \text{op} \le 5$）。输出每组操作的运算结果。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 10$）。

接下来 $T$ 行，每行三个整数 $a$、$b$、$\text{op}$（$0 \le a \le 255$，$0 \le b \le 8$，$1 \le \text{op} \le 5$）。

## 输出格式

$T$ 行，每行一个整数。

## 样例

**输入**

```
5
10 3 1
10 3 2
10 3 3
10 3 4
80 3 5
```

**输出**

```
2
11
9
80
10
```

**说明**

- `10 & 3`：`1010 & 0011 = 0010` = **2**
- `10 | 3`：`1010 | 0011 = 1011` = **11**
- `10 ^ 3`：`1010 ^ 0011 = 1001` = **9**
- `10 << 3`：$10 \times 2^3 = 10 \times 8 =$ **80**
- `80 >> 3`：$80 \div 2^3 = 80 \div 8 =$ **10**

## 提示

五种位运算在 C++ 中直接对应五个运算符：`&`、`|`、`^`、`<<`、`>>`。

```cpp
if (op == 1) cout << (a & b);
else if (op == 2) cout << (a | b);
else if (op == 3) cout << (a ^ b);
else if (op == 4) cout << (a << b);
else              cout << (a >> b);
```

**几个有趣的规律**：
- `a & 1`：判断奇偶（结果为 1 表示奇数，0 表示偶数）
- `a ^ a = 0`：任何数与自身异或为 0
- `a << 1 = a * 2`，`a >> 1 = a / 2`（整除）
