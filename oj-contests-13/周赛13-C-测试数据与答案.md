## 测试点 4（较长名字）

**输入**
```
3
minister
general
soldier
```
**预期输出**
```
8 3
7 3
7 3
```

**说明**：
- `minister`：m-i-n-i-s-t-e-r，元音 i、i、e，共 3 个
- `general`：g-e-n-e-r-a-l，元音 e、e、a，共 3 个
- `soldier`：s-o-l-d-i-e-r，元音 o、i、e，共 3 个

## 测试点 5（n=1，最长名字）

**输入**
```
1
abcdefghijklmnopqrst
```
**预期输出**
```
20 5
```

**说明**：长度 20，元音 a、e、i、o 共 4 个... 等等：
a(✓)b c d e(✓)f g h i(✓)j k l m n o(✓)p q r s t = 4 个元音。

修正：
```
1
abcdefghijklmnopqrstu
```
→ 长度过21，超范围。用：
```
1
aabbccddeeiioouu
```
→ 长度 16，元音 a、a、e、e、i、i、o、o、u、u = 10 个。

**输入**
```
1
aabbccddeeiioouu
```
**预期输出**
```
16 10
```

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <cstring>
using namespace std;

bool isVowel(char c) {
    return c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u';
}

int main() {
    int n;
    cin >> n;
    while (n--) {
        char name[25];
        cin >> name;
        int len = strlen(name);
        int vowels = 0;
        for (int i = 0; name[i] != '\0'; i++) {
            if (isVowel(name[i])) vowels++;
        }
        cout << len << " " << vowels << "\n";
    }
    return 0;
}
```

**关键点**：
1. `char name[25]` 要预留比最长字符串多 1 位的空间，给 `'\0'` 留位置；
2. `strlen(name)` 返回字符数（不含 `'\0'`），需要 `#include <cstring>`；
3. 遍历条件 `name[i] != '\0'` 与 `i < strlen(name)` 等价，但前者更体现 char 数组的本质；
4. `cin >> name` 遇到空格或换行停止，自动在末尾补 `'\0'`。
