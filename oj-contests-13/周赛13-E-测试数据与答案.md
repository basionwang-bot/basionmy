## 测试点 4（链式替换：第一轮的结果作为第二轮的输入）

**输入**
```
i am the ruler
2
ruler
king
king
emperor
```
**预期输出**
```
i am the emperor
```

**说明**：
- 第一轮：`ruler` → `king`，文本变为 `i am the king`
- 第二轮：`king` → `emperor`，文本变为 `i am the emperor`

## 测试点 5（多轮替换，new 比 old 长）

**输入**
```
go east
3
go
march
east
to the east
march
advance
```
**预期输出**
```
advance to the east
```

**说明**：
- 第一轮：`go` → `march` → `march east`
- 第二轮：`east` → `to the east` → `march to the east`
- 第三轮：`march` → `advance` → `advance to the east`

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
using namespace std;

void replaceAll(string& s, const string& old_str, const string& new_str) {
    size_t pos = 0;
    while ((pos = s.find(old_str, pos)) != string::npos) {
        s.replace(pos, old_str.size(), new_str);
        pos += new_str.size();
    }
}

int main() {
    string s;
    getline(cin, s);
    int n;
    cin >> n;
    cin.ignore();
    for (int i = 0; i < n; i++) {
        string old_str, new_str;
        getline(cin, old_str);
        getline(cin, new_str);
        replaceAll(s, old_str, new_str);
    }
    cout << s << "\n";
    return 0;
}
```

**关键点**：
1. `getline(cin, s)` 读含空格的整行；
2. `cin >> n; cin.ignore();` — 读整数后必须用 `ignore()` 跳过残留的 `'\n'`，否则下一次 `getline` 会读到空行；
3. `replaceAll` 函数每次从 `pos=0` 重新扫描当前文本，保证所有出现都被替换；
4. `pos += new_str.size()` 跳过新词，防止对刚替换好的内容再次匹配（避免死循环）；
5. 多条规则顺序很重要，测试点 4 展示了链式效果。
