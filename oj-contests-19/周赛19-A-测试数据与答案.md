## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    int T; cin >> T;
    while (T--) {
        int s; cin >> s;
        if (s >= 90) cout << "S\n";
        else if (s >= 70) cout << "A\n";
        else if (s >= 50) cout << "B\n";
        else cout << "C\n";
    }
    return 0;
}
```

**关键点**：用 `else if` 链从高到低判断，等号不能遗漏（`>=90`、`>=70`、`>=50`），每组独立输出一行。
