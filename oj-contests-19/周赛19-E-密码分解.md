# E - 密码分解

**难度**：★★★★★  
**考点**：唯一分解定理（质因数分解）  
**GESP 级别**：5级

---

## 题目背景

魔法学院的密码库使用数字密钥保护，每把密钥都是一个整数。破解密钥需要将其分解为质因数的乘积形式——这正是数论中的"唯一分解定理"。

## 题目描述

给定一个正整数 $n$（$2 \le n \le 10^9$），输出其质因数分解式。

格式要求：
- 质因子从小到大排列，格式为 `p1^e1 * p2^e2 * ...`
- 若某质因子的指数为 1，则直接写 `p`，不写 `^1`

## 输入格式

一行一个正整数 $n$（$2 \le n \le 10^9$）。

## 输出格式

输出 $n$ 的质因数分解式（一行）。

## 样例

**输入 1**

```
12
```

**输出 1**

```
2^2 * 3
```

**说明**：$12 = 2^2 \times 3$，质因子 2 的指数为 2，质因子 3 的指数为 1（省略 `^1`）。

---

**输入 2**

```
7
```

**输出 2**

```
7
```

**说明**：7 本身是质数，分解结果只有它自己。

---

**输入 3**

```
36
```

**输出 3**

```
2^2 * 3^2
```

---

**输入 4**

```
2
```

**输出 4**

```
2
```

---

**输入 5**

```
1000000007
```

**输出 5**

```
1000000007
```

**说明**：$10^9 + 7$ 是质数，直接输出本身。

## 提示

从 2 开始试除到 $\sqrt{n}$，统计每个质因子的指数：

```cpp
bool first = true;
for (long long p = 2; p * p <= n; p++) {
    if (n % p == 0) {
        int exp = 0;
        while (n % p == 0) { exp++; n /= p; }
        if (!first) cout << " * ";
        first = false;
        if (exp == 1) cout << p;
        else cout << p << "^" << exp;
    }
}
if (n > 1) {  // 剩余的 n 本身是质数
    if (!first) cout << " * ";
    cout << n;
}
```

关键：试除完成后，若 $n > 1$，则剩余的 $n$ 本身是一个质因子（指数为 1）。时间复杂度 $O(\sqrt{n})$，对 $n \le 10^9$ 完全够用。
