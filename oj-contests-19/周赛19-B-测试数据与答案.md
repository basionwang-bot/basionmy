## 测试点 1（样例1）

**输入**
```
7
3 7 5 8 10 2 9
```
**预期输出**
```

```
**说明**：递增段为 [3,7](2)、[5,8,10](3)、[2,9](2)，最长为 3。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    int n; cin >> n;
    int a[1005];
    for (int i = 0; i < n; i++) cin >> a[i];
    int cur = 1, ans = 1;
    for (int i = 1; i < n; i++) {
        if (a[i] > a[i-1]) cur++;
        else cur = 1;
        ans = max(ans, cur);
    }
    cout << ans << "\n";
    return 0;
}
```

**关键点**：
1. 初始化 `cur = 1, ans = 1`（单元素本身就是长度1的段）；
2. 遇到 `a[i] > a[i-1]` 则 `cur++`，否则重置 `cur = 1`；
3. 每步用 `max` 更新答案；
4. 严格递增用 `>`，相等不算递增。
