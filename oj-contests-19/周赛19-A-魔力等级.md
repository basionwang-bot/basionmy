# A - 魔力等级

**难度**：★☆☆☆☆  
**考点**：条件分支、多组输入  
**GESP 级别**：1级

---

## 题目背景

魔法学院每学期末会对学员的魔力值进行考核，并根据分数自动划定等级，决定学员能否晋升。

## 题目描述

给定学员的魔力值 $s$（整数），按以下规则输出等级：

- $s \ge 90$：输出 `S`（大师级）
- $70 \le s < 90$：输出 `A`（优秀）
- $50 \le s < 70$：输出 `B`（良好）
- $s < 50$：输出 `C`（待提升）

给定 $T$ 组查询，每组输出对应等级。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 100$）。

接下来 $T$ 行，每行一个整数 $s$（$0 \le s \le 100$）。

## 输出格式

共 $T$ 行，每行输出对应等级字符（`S`、`A`、`B` 或 `C`）。

## 样例

**输入**

```
4
95
72
50
30
```

**输出**

```
S
A
B
C
```

**说明**：$s=95 \ge 90$ 为 S；$s=72$ 满足 $70 \le 72 < 90$ 为 A；$s=50$ 满足 $50 \le 50 < 70$ 为 B；$s=30 < 50$ 为 C。

## 提示

注意边界：$s=90$ 属于 S，$s=70$ 属于 A，$s=50$ 属于 B，判断时使用 `>=` 并注意等号不要遗漏。

```cpp
if (s >= 90) cout << "S\n";
else if (s >= 70) cout << "A\n";
else if (s >= 50) cout << "B\n";
else cout << "C\n";
```
