## 测试点 1（样例1：合数，混合指数）

**输入**
```
12
```
**预期输出**
```
2^2 * 3
```
**验证**：$12 = 2^2 \times 3$。

## 测试点 3（样例3：两个质因子均有高次幂）

**输入**
```
36
```
**预期输出**
```
2^2 * 3^2
```
**验证**：$36 = 2^2 \times 3^2 = 4 \times 9$。

## 测试点 5（大质数 $10^9+7$）

**输入**
```
1000000007
```
**预期输出**
```
1000000007
```
**说明**：$10^9+7$ 是质数，试除到 $\sqrt{10^9+7} \approx 31623$ 均不整除，最终剩余本身输出。

## 测试点 6（多个质因子，含高次幂）

**输入**
```
2520
```
**预期输出**
```
2^3 * 3^2 * 5 * 7
```
**验证**：$2520 = 8 \times 9 \times 5 \times 7 = 2^3 \times 3^2 \times 5 \times 7$。

## 测试点 7（2 的高次幂）

**输入**
```
1024
```
**预期输出**
```
2^10
```
**验证**：$1024 = 2^{10}$。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    long long n; cin >> n;
    bool first = true;
    for (long long p = 2; p * p <= n; p++) {
        if (n % p == 0) {
            int exp = 0;
            while (n % p == 0) { exp++; n /= p; }
            if (!first) cout << " * ";
            first = false;
            if (exp == 1) cout << p;
            else cout << p << "^" << exp;
        }
    }
    if (n > 1) {
        if (!first) cout << " * ";
        cout << n;
    }
    cout << "\n";
    return 0;
}
```

**关键点**：
1. 用 `long long` 存储 $n$，防止大数试除时溢出；
2. 枚举 $p$ 从 2 到 $\sqrt{n}$，时间复杂度 $O(\sqrt{n})$；
3. 每找到一个质因子，用 `while` 循环连续除尽并统计指数；
4. 试除后若 $n > 1$，则剩余的 $n$ 本身是最大质因子；
5. `first` 标志控制输出时 ` * ` 分隔符，第一个因子前不加。
