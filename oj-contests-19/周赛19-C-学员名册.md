# C - 学员名册

**难度**：★★★☆☆  
**考点**：结构体 + 条件筛选 + 多关键字排序  
**GESP 级别**：3级

---

## 题目背景

魔法学院学期末公示优秀学员名单，要求从全体学员中筛选出魔力值达标的学员，并按规定顺序排列展示。

## 题目描述

给定 $n$ 名学员的信息（姓名、魔力值、年龄），筛选满足以下条件的学员：

- 魔力值 $\ge 70$

对筛选结果按以下规则排序：

1. 魔力值**从大到小**
2. 魔力值相同时，姓名**字典序从小到大**

输出所有符合条件的学员姓名（每行一个）。若没有学员入选，输出 `none`。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，每行格式为：

```
姓名 魔力值 年龄
```

姓名为不含空格的字符串（长度 $\le 20$），魔力值和年龄均为正整数（$0 \le$ 魔力值 $\le 100$，$1 \le$ 年龄 $\le 100$）。

## 输出格式

按排序后顺序，每行输出一个姓名。

## 样例

**输入**

```
4
Alice 85 16
Bob 65 18
Carol 75 15
Dave 75 17
```

**输出**

```
Alice
Carol
Dave
```

**说明**：Bob 魔力值 65 < 70 不入选。  
剩余三人：Alice(85)、Carol(75)、Dave(75)。  
按魔力值降序：Alice 第一（85）；Carol 和 Dave 魔力值均为 75，按姓名字典序 Carol < Dave，故 Carol 排在 Dave 前面。

## 提示

```cpp
struct Student { string name; int power, age; };
bool cmp(const Student& a, const Student& b) {
    if (a.power != b.power) return a.power > b.power;
    return a.name < b.name;
}
```

筛选时判断 `power >= 70`，结果为空时输出 `none`。
