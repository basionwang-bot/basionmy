# D - 螺旋矩阵

**难度**：★★★★☆  
**考点**：二维数组 + 顺时针螺旋填数模拟  
**GESP 级别**：4级

---

## 题目背景

魔法学院的古籍中记载了一种神秘的"螺旋封印阵"，将数字 $1$ 到 $n^2$ 按顺时针螺旋顺序填入 $n \times n$ 的方阵中，据说蕴含着强大的魔力。

## 题目描述

给定正整数 $n$，将 $1, 2, \ldots, n^2$ 按**顺时针螺旋顺序**填入 $n \times n$ 矩阵，并输出该矩阵。

填数方向依次为：向右 → 向下 → 向左 → 向上 → 向右 → $\cdots$，从左上角 $(1,1)$ 开始。

## 输入格式

一行一个正整数 $n$（$1 \le n \le 10$）。

## 输出格式

共 $n$ 行，每行 $n$ 个整数，数字之间用一个空格隔开，行末不加多余空格。

## 样例

**输入 1**

```
3
```

**输出 1**

```
1 2 3
8 9 4
7 6 5
```

---

**输入 2**

```
1
```

**输出 2**

```
1
```

---

**输入 3**

```
4
```

**输出 3**

```
1 2 3 4
12 13 14 5
11 16 15 6
10 9 8 7
```

**说明（n=4）**：从左上角开始，先向右填 1~4，再向下填 5~7，再向左填 8~10，再向上填 11~12，再向右填 13~14，再向下填 15，最后填中心 16。

## 提示

用四个边界变量 `top`、`bottom`、`left`、`right` 控制当前可填区域，每填完一行或一列后收缩对应边界：

```cpp
int mat[11][11] = {};
int top = 1, bottom = n, left = 1, right = n, num = 1;
while (num <= n * n) {
    for (int j = left;  j <= right  && num <= n*n; j++) mat[top][j]    = num++;
    top++;
    for (int i = top;   i <= bottom && num <= n*n; i++) mat[i][right]  = num++;
    right--;
    for (int j = right; j >= left   && num <= n*n; j--) mat[bottom][j] = num++;
    bottom--;
    for (int i = bottom; i >= top   && num <= n*n; i--) mat[i][left]   = num++;
    left++;
}
// 输出 mat[1..n][1..n]
```

注意每次填完一条边后立即收缩边界，并且循环条件要检查 `num <= n*n` 防止越界。
