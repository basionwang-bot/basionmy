## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    int n; cin >> n;
    int mat[11][11] = {};
    int top = 1, bottom = n, left = 1, right = n, num = 1;
    while (num <= n * n) {
        for (int j = left;  j <= right  && num <= n*n; j++) mat[top][j]    = num++;
        top++;
        for (int i = top;   i <= bottom && num <= n*n; i++) mat[i][right]  = num++;
        right--;
        for (int j = right; j >= left   && num <= n*n; j--) mat[bottom][j] = num++;
        bottom--;
        for (int i = bottom; i >= top   && num <= n*n; i--) mat[i][left]   = num++;
        left++;
    }
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            if (j > 1) cout << " ";
            cout << mat[i][j];
        }
        cout << "\n";
    }
    return 0;
}
```

**关键点**：
1. 用 `top/bottom/left/right` 四个边界，每填完一条边立即收缩；
2. 内层循环条件加上 `num <= n*n`，防止奇数阶矩阵最后一格重复填；
3. 数组下标从 1 开始，方便对应行列编号；
4. 输出时相邻数字间加空格，行末不加多余空格。
