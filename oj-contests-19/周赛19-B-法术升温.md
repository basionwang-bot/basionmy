# B - 法术升温

**难度**：★★☆☆☆  
**考点**：最长连续严格递增子序列长度（cur/ans 模式）  
**GESP 级别**：2级

---

## 题目背景

魔法学院的锻炼场每天记录学员施法时的炉温变化。老师希望找到温度**持续上升**时间最长的一段，以分析最佳修炼时机。

## 题目描述

给定 $n$ 个整数表示连续时刻的炉温记录，找出其中**最长的连续严格递增子序列**的长度。

所谓连续严格递增，是指相邻元素满足 $a_i < a_{i+1}$，且这一段连续的元素个数尽可能长。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 1000$）。

第二行 $n$ 个整数（$-10^4 \le a_i \le 10^4$）。

## 输出格式

输出一个整数，表示最长连续严格递增子序列的长度。

## 样例

**输入 1**

```
7
3 7 5 8 10 2 9
```

**输出 1**

```
3
```

**说明**：序列中各段递增子序列为 [3,7]（长度2）、[5,8,10]（长度3）、[2,9]（长度2），最长为 3。

---

**输入 2**

```
5
1 2 3 4 5
```

**输出 2**

```
5
```

**说明**：整个序列严格递增，最长连续递增段长度为 5。

## 提示

用 `cur` 记录当前递增段长度，`ans` 记录历史最大值：

```cpp
int cur = 1, ans = 1;
for (int i = 1; i < n; i++) {
    if (a[i] > a[i-1]) cur++;
    else cur = 1;
    ans = max(ans, cur);
}
```

注意：$n=1$ 时只有一个元素，答案为 1；全部相等时每段长度均为 1，答案也为 1。
