# C - 学员名册 · 测试数据与答案

---

## 测试点 1（样例）

**输入**
```
4
Alice 85 16
Bob 65 18
Carol 75 15
Dave 75 17
```
**预期输出**
```
Alice
Carol
Dave
```
**说明**：Bob 魔力值 65 < 70 被筛掉；Carol 和 Dave 并列 75，按姓名字典序 Carol < Dave。

## 测试点 6（n=1，唯一学员入选）

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>
using namespace std;

struct Student { string name; int power, age; };

bool cmp(const Student& a, const Student& b) {
    if (a.power != b.power) return a.power > b.power;
    return a.name < b.name;
}

int main() {
    int n; cin >> n;
    vector<Student> res;
    for (int i = 0; i < n; i++) {
        Student s; cin >> s.name >> s.power >> s.age;
        if (s.power >= 70) res.push_back(s);
    }
    if (res.empty()) cout << "none\n";
    else {
        sort(res.begin(), res.end(), cmp);
        for (auto& s : res) cout << s.name << "\n";
    }
    return 0;
}
```

**关键点**：
1. 筛选条件 `power >= 70`（含等于 70）；
2. 排序第一关键字魔力值降序，第二关键字姓名字典序升序；
3. 结果为空输出 `none`，否则每行输出一个姓名。
