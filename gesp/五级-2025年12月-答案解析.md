## 一、单选题（每题 2 分，共 30 分）

### 第 1 题 循环单链表遍历

**答案：C**

**题目**：在 `printList` 中横线处应填什么，用于遍历循环单链表并输出？

**选项分析**：

- **A** `while (p != nullptr)`：循环单链表的尾结点 `next` 指向 `head`，没有 `nullptr`，会**死循环**。
- **B** `while (p->next != nullptr)`：同样永远不会为 `nullptr`，**死循环**。
- **C** `do { cout << p->data << " "; p = p->next; } while (p != head);`：先输出当前结点，再移动到下一个，当回到 `head` 时结束，**正确**。
- **D** `for(; p; p=p->next)`：`p` 在循环链表中始终非空，**死循环**。

### 第 2 题 区块链插入新区块

**答案：B**

**题目**：区块链尾部插入新区块，横线处应填什么？

**选项分析**：

- **A** `tail = newBlock->prev`：`newBlock->prev` 是原来的 `tail`，`tail` 不变，**错误**。
- **B** `Block* newBlock = new Block(tail->index + 1, data, tail); tail = newBlock;`：创建新区块指向前一个，并让 `tail` 指向新块，**正确**。
- **C** 使用 `tail->prev` 作为新块的前驱，逻辑错误。
- **D** `tail = newBlock->prev`：`tail` 未更新为新块，**错误**。

### 第 3 题 单链表与双链表删除

**答案：C**

**题目**：关于单链表和双链表删除指定节点的复杂度，哪个正确？

**解析**：

- **双链表**：有 `prev` 和 `next`，已知节点指针时，只需修改前后邻居指针，**O(1)**。
- **单链表**：没有前驱指针，需要从头遍历找到前驱，**O(n)**。

选项中 C 描述双链表 O(1)、单链表 O(n)，**正确**。

### 第 4 题 模同余

**答案：D**

**题目**：若 a ≡ b (mod m)，则 m 是 (a-b) 的因子。以下哪个**不可能**是 m？

**解析**：a ≡ b (mod m) 等价于 m | (a-b)。因此 m 必须是 |a-b| 的约数。题目会给定具体 a、b，使 a-b 的约数中不包含某个选项。常见情况是 a-b 较小（如 6、12），则 9 不是其约数，**9 不可能**为 m。

### 第 5 题 欧几里得算法

**答案：D**

**题目**：关于 gcd1（递归）和 gcd2（迭代），说法**错误**的是？

- A：gcd1 是递归实现，正确。
- B：gcd2 是迭代实现，正确。

### 第 7 题 线性筛

**答案：A**

**题目**：线性筛（欧拉筛）横线处应填什么？

**正确代码**：

```cpp
for (int j = 0; j < primes.size() && i * primes[j] <= n; j++)
```

**解析**：枚举已得到的素数 `primes[j]`，筛去 `i * primes[j]`，且不超过 n。当 `i % primes[j] == 0` 时 `break`，保证每个合数只被其最小素因子筛一次，复杂度 O(n)。

### 第 11 题 lower_bound 实现

**答案：A**

**题目**：实现“第一个 ≥ x 的下标”，代码是否正确？

**解析**：标准 lower_bound 写法：`l=0, r=size`，`while(l<r)`，`mid=l+(r-l)/2`，若 `arr[mid]>=x` 则 `r=mid`，否则 `l=mid+1`，返回 `l`。题目给出的逻辑与此一致，**正确**。

### 第 12 题 木头切割二分

**答案：A**

**题目**：二分找最小的 x，使得在不超过 K 刀内，每段长度 ≤ x。横线处应填？

**解析**：若 `check(L, K, mid)` 为真，说明 mid 可行，要找最小 x，应缩小右界：`r = mid`；否则 `l = mid + 1`。正确答案为：

```cpp
if (check(L, K, mid))
    r = mid;
else
    l = mid + 1;
```

## 三、编程题

### 3.1 编程题 1：数字移动

#### 题目简述

长度为 n 的序列，每个数恰好出现 2 次。每次可把某个数移动到任意位置，花费为该数的值。求最小的 x，使得在每次花费不超过 x 的前提下，能让每对相同数字相邻。

#### 思路

二分答案 x：若允许花费不超过 x，则“花费 > x”的数不能移动，只能保留在原位。把这些数按原顺序抽出，检查是否已经成对相邻。若成对则 x 可行，缩小 x；否则需增大 x。

#### 核心逻辑

- 二分区间 [1, 1e6]。
- 对每个 mid，保留所有 `a[i] > mid` 的数到新数组 b。
- 若 b 中相邻两两相等（b[0]=b[1], b[2]=b[3], …），则可行，`ans=mid, right=mid-1`；否则 `left=mid+1`。

#### 参考代码（与题目所给一致）

```cpp
#include <iostream>
using namespace std;
const int N = 100010;
int a[N], b[N];
int pos;

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    int left = 1, right = 1e6, ans = 1e6;
    while (left <= right) {
        int mid = (left + right) / 2;
        bool possible = true;
        pos = 0;
        for (int i = 0; i < n; i++)
            if (a[i] > mid) b[pos++] = a[i];
        for (int i = 0; i < pos; i += 2) {
            if (b[i] != b[i + 1]) {
                possible = false;
                break;
            }
        }
        if (possible) {
            ans = mid;
            right = mid - 1;
        } else {
            left = mid + 1;
        }
    }
    cout << ans << endl;
    return 0;
}
```

**样例**：`6\n1 2 1 3 2 3` → 不移动时已经成对相邻，最小 x=1。

### 3.2 编程题 2：相等序列

#### 题目简述

给定 n 个正整数，每次可用 1 个金币执行：
1. 把某个数 x 变为 x+p（p 为任意质数）；
2. 把某个数 x 变为 x/p（p 为质数且 p|x）。

求让所有数相同的最少金币数。

#### 思路

- 操作 1：增加某个质因子 p 的指数；
- 操作 2：减少某个质因子 p 的指数。

对每个质因子 p 独立考虑：统计每个数中 p 的指数，形成序列。要让所有数在 p 上的指数相同，等价于把所有指数变为同一值，代价为每个指数到目标值的距离之和。目标值取中位数时，总代价最小。

对每个质数 p：
1. 统计每个数中 p 的指数；
2. 求中位数；
3. 累加每个指数到中位数的距离。

总金币数 = 所有质因子的代价之和。

#### 参考代码（与题目所给一致）

```cpp
#include <iostream>
using namespace std;
const int N = 100010;
int num[N][20];  // num[p][j] = 有多少个数的质因子 p 的指数为 j
int n, a[N];

void calc_prime_factor(int x) {
    for (int i = 2; i * i <= x; i++) {
        if (x % i == 0) {
            int cnt = 0;
            while (x % i == 0) x /= i, cnt++;
            num[i][cnt]++;
        }
    }
    if (x > 1) num[x][1]++;
}

int main() {
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        scanf("%d", &a[i]);
        calc_prime_factor(a[i]);
    }
    long long ans = 0;
    for (int i = 2; i < 100001; i++) {
        int pos = 0;
        for (int j = 0; j < 20; j++) pos += num[i][j];
        num[i][0] = n - pos;  // 指数为 0 的个数（不含质因子 i）
        pos = 0;
        int median_exponent = 0;
        for (int j = 0; j < 20; j++) {
            pos += num[i][j];
            if (pos * 2 >= n) {
                median_exponent = j;
                break;
            }
        }
        for (int j = 0; j < 20; j++)
            ans += num[i][j] * abs(j - median_exponent);
    }
    printf("%lld\n", ans);
    return 0;
}
```

**样例**：`5\n10 6 35 105 42`  
分解后对每个质因子取中位数并累加代价，可得最少金币数为 8。
