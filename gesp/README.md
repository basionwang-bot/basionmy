# GESP 做题

本文件夹用于存放 GESP（软件编程等级考试）相关练习、笔记与题解。

## 目录说明

- **一级**：入门级（变量、顺序/分支/循环、简单数组）
- **二级**：基础算法与结构（函数、字符串、二维数组等）
- **三级及以上**：按 GESP 大纲分模块练习

## 使用方式

- 按级别建子文件夹，或在当前目录下用 `级别-题目标题.md` 记录题目与思路。
- 可把题目描述、输入输出样例、自己的 AC 代码或笔记放在对应文件中。
