第 1 题 循环单链表遍历

答案：C

题目：在 printList 中横线处应填什么，用于遍历循环单链表并输出？

  • A while (p != nullptr)：循环单链表的尾结点 next 指向 head，没有 nullptr，会死循环。
  • B while (p->next != nullptr)：同样永远不会为 nullptr，死循环。
  • C do { cout << p->data << " "; p = p->next; } while (p != head);：先输出当前结点，再移动到下一个，当回到 head 时结束，正确。
  • D for(; p; p=p->next)：p 在循环链表中始终非空，死循环。

第 2 题 区块链插入新区块

答案：B

题目：区块链尾部插入新区块，横线处应填什么？

  • A tail = newBlock->prev：newBlock->prev 是原来的 tail，tail 不变，错误。
  • B Block* newBlock = new Block(tail->index + 1, data, tail); tail = newBlock; 正确。
  • C 使用 tail->prev 作为新块的前驱，逻辑错误。
  • D tail = newBlock->prev：tail 未更新为新块，错误。

三、编程题

3.1 编程题 1：数字移动

题目简述：长度 n 的序列，每数出现 2 次。每次移动花费为该数值。求最小 x，使每次花费≤x 时能让每对相同数字相邻。

思路：二分 x。保留 a[i]>mid 的数，检查是否已两两成对相邻。

#include <iostream>
using namespace std;
const int N = 100010;
int a[N], b[N];
int pos;

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    int left = 1, right = 1e6, ans = 1e6;
    while (left <= right) {
        int mid = (left + right) / 2;
        bool possible = true;
        pos = 0;
        for (int i = 0; i < n; i++)
            if (a[i] > mid) b[pos++] = a[i];
        for (int i = 0; i < pos; i += 2) {
            if (b[i] != b[i + 1]) { possible = false; break; }
        }
        if (possible) { ans = mid; right = mid - 1; }
        else left = mid + 1;
    }
    cout << ans << endl;
    return 0;
}

样例：6 1 2 1 3 2 3 → 1

3.2 编程题 2：相等序列

题目简述：n 个正整数，每次 1 金币可 +质数 或 /质数。求使所有数相同的最少金币数。

思路：对每个质因子独立，统计指数分布，取中位数，累加 |指数-中位数| 之和。

#include <iostream>
using namespace std;
const int N = 100010;
int num[N][20], n, a[N];

void calc_prime_factor(int x) {
    for (int i = 2; i * i <= x; i++) {
        if (x % i == 0) {
            int cnt = 0;
            while (x % i == 0) x /= i, cnt++;
            num[i][cnt]++;
        }
    }
    if (x > 1) num[x][1]++;
}

int main() {
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        scanf("%d", &a[i]);
        calc_prime_factor(a[i]);
    }
    long long ans = 0;
    for (int i = 2; i < 100001; i++) {
        int pos = 0;
        for (int j = 0; j < 20; j++) pos += num[i][j];
        num[i][0] = n - pos;
        pos = 0;
        int median_exponent = 0;
        for (int j = 0; j < 20; j++) {
            pos += num[i][j];
            if (pos * 2 >= n) { median_exponent = j; break; }
        }
        for (int j = 0; j < 20; j++)
            ans += num[i][j] * abs(j - median_exponent);
    }
    printf("%lld\n", ans);
    return 0;
}

样例：5 10 6 35 105 42 → 8
