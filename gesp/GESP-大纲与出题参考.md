> 来源：CCF（中国计算机学会）官方大纲
> 用途：周赛出题时按级别选取知识点，确保难度梯度准确
> 标注说明：✅ 已在周赛中出现 / ⬜ 尚未覆盖

## 考试基本信息

| 项目 | 内容 |
|------|------|
| 主办 | 中国计算机学会（CCF） |
| 分级 | C++ 共 1-8 级 |

## 第 1 级

**定位**：编程入门，掌握最基本的 C++ 语法与输入输出。

| 知识点 | 考查内容 | 周赛覆盖 |
|--------|----------|---------|
| 基本数据类型 | int、float、double、char、bool | ✅ |
| 变量与常量 | 声明、初始化、const | ✅ |
| 算术运算符 | +、-、*、/、% | ✅ |
| 比较与逻辑运算符 | <、>、==、!=、&&、\|\|、! | ✅ |
| 自增自减 | 前置/后置 ++、-- | ✅ |
| 输入输出 | cin、cout，setprecision | ✅ |
| 顺序结构 | 逐行执行 | ✅ |
| 条件分支 | if / else if / else、switch | ✅ |
| 简单循环 | for、while | ✅ |
| 多组输入 | while(T--)、while(cin>>x) | ✅ |
| 浮点输出格式 | fixed、setprecision(n) | ✅ |
| 进制基础 | 二进制、十进制概念 | ✅（周赛12-A） |

**周赛出题建议**：A 题，输入规模小，一次或多组直接计算即可。

## 第 2 级

**定位**：掌握一维数组、字符数组和基础算法思维。

| 知识点 | 考查内容 | 周赛覆盖 |
|--------|----------|---------|
| 一维数组 | 定义、初始化、下标访问 | ✅ |
| 数组遍历 | for 循环枚举 | ✅ |
| 最值统计 | max/min 记录，max_element | ✅ |
| 求和与平均值 | 累加变量，浮点除法 | ✅ |
| 分类计数 | 多个计数器 | ✅ |
| 最长连续段 | cur/ans 模式 | ✅（周赛10-B） |
| 双层循环数字逻辑 | i,j 双层枚举满足等式/整除关系 | ✅（周赛11-B 平方和数对） |
| 进制转换 | 二进制↔十进制↔十六进制 | ✅（周赛12-B） |
| 位运算 | &、\|、^、~、<<、>> | ✅（周赛13-B） |
| 字符数组基础 | char 数组，'\0' 结尾 | ✅（周赛13-C） |
| 前缀和 | 预处理数组，区间求和 O(1) | ✅（周赛04） |
| 滑动窗口 | 固定长度窗口，双指针移动 | ✅（周赛07-B） |

**周赛出题建议**：B 题，以数组操作为核心，加入统计/查找/累加类变体。

## 第 3 级

**定位**：掌握字符串处理、函数定义，理解递推与排序。

| 知识点 | 考查内容 | 周赛覆盖 |
|--------|----------|---------|
| 结构体 struct | 定义、初始化、成员访问 | ✅ |
| 多关键字排序 | sort + 自定义 cmp 函数 | ✅ |
| 条件筛选 | 先过滤再排序 | ✅ |
| 字符串 string | 基本操作，length/size | ✅（周赛11-C） |
| 字符串函数 | find、substr、replace、tolower | ✅（周赛11-C find；周赛12-C substr/replace） |
| 回文判断 | 双指针或 reverse 比较 | ✅（周赛09-D） |
| 函数定义与调用 | 参数传递，返回值 | ✅（周赛09-E GCD函数） |
| 简单模拟 | 按时间顺序/规则模拟过程 | ✅（周赛06-C 叫号系统） |
| 双层循环枚举 | O(n²) 枚举组合/对 | ✅（周赛07-E 宝箱配对） |
| 冒泡/选择排序 | 手写排序过程（理解 O(n²)） | ✅（周赛12-D 冒泡） |

**周赛出题建议**：C 题，结构体 + 多关键字排序 是主线，也可以是模拟或字符串处理。

## 第 4 级

**定位**：掌握函数递归，了解指针与文件，应用二维数组。

| 知识点 | 考查内容 | 周赛覆盖 |
|--------|----------|---------|
| 函数递归基础 | 终止条件 + 递归调用 | ✅（周赛10-D 汉诺塔） |
| 斐波那契/阶乘 | 经典递归/递推示例 | ✅（周赛11-D 族群繁衍） |
| 二维数组 | 定义、行列遍历、初始化 | ✅（周赛06-D 评分矩阵） |
| 二维数组旋转 | 矩阵顺/逆时针旋转 | ✅（周赛07-D） |
| 冒泡排序（完整） | 理解交换过程，O(n²) | ✅（周赛12-D） |
| 选择排序（完整） | 每轮找最小，O(n²) | ✅（周赛13-F） |
| 指针基础 | *ptr，&变量，函数参数传地址 | ✅（周赛15-D） |
| 引用 | int &ref = x | ✅（周赛16-D） |
| 文件读写 | ifstream/ofstream，freopen | ⬜ |
| 数列递推规律 | 线性递推，前几项推后项 | ✅（周赛14-D） |
| 杨辉三角 | 二维数组递推，`c[i][j]=c[i-1][j-1]+c[i-1][j]` | ✅（周赛18-D） |
| 螺旋矩阵 | 四边界收缩，顺时针填数 | ✅（周赛19-D） |
| 快速求幂 | 递归分治 $a^n$，$O(\log n)$ | ✅（周赛20-D） |
| 二维前缀和 | 子矩阵求和 $O(1)$，容斥公式 | ✅（周赛21-D） |
| 路径计数 DP | 二维 DP，`dp[i][j]=dp[i-1][j]+dp[i][j-1]` | ✅（周赛22-D） |

**周赛出题建议**：D 题引入递归（汉诺塔、斐波那契、分治）或二维数组处理。

## 第 5 级

**定位**：初等数论、高精度运算、链表，掌握二分与贪心。

| 知识点 | 考查内容 | 周赛覆盖 |
|--------|----------|---------|
| GCD / LCM | 辗转相除法（欧几里得算法） | ✅（周赛09-E） |
| 素数判断 | 试除法 O(√n) | ✅（周赛11-E 顺带覆盖） |
| 埃氏筛法 | O(n log log n) 批量求素数 | ✅（周赛11-E 质数信号） |
| 唯一分解定理 | 质因数分解 | ✅（周赛19-E） |
| 高精度加法 | 字符串/数组模拟进位 | ✅（周赛10-E） |
| 高精度减法 | 数组模拟借位 | ✅（周赛17-E） |
| 高精度乘法 | 数组逐位相乘累加 | ✅（周赛18-E） |
| 链表（单链表） | 创建、插入、删除、遍历 | ✅（周赛12-E） |
| 二分查找 | 有序数组中查找，O(log n) | ✅（周赛08-D） |
| 二分答案 | 对答案二分，check 函数验证 | ✅（周赛16-E） |
| 归并排序 | 分治思想，O(n log n) | ✅（周赛14-E） |
| 快速排序 | 分治思想，O(n log n) 均摊 | ✅（周赛15-E） |
| 贪心算法 | 局部最优→全局最优 | ✅（周赛09-F 区间调度；周赛21-E 区间合并） |
| 分治算法 | 问题分解，递归求解 | ⬜ |
| 算法复杂度 | O(1)、O(n)、O(n²)、O(log n) 概念 | ⬜ |

**周赛出题建议**：E 题用新数论/高精度/链表知识点，贪心也适合 E 题。

## 第 6 级

**定位**：搜索（DFS/BFS）、树与图基础、栈队列、动态规划入门。

| 知识点 | 考查内容 | 周赛覆盖 |
|--------|----------|---------|
| 栈（stack） | LIFO，括号匹配，表达式计算 | ✅（周赛08-E） |
| 队列（queue） | FIFO，BFS 配套数据结构 | ✅（周赛11-F 配套BFS） |
| 双端队列（deque） | 两端插入/删除，单调队列滑窗最值 | ✅（周赛22-E） |
| DFS 全排列 | 递归枚举所有排列，回溯 | ✅（周赛08-F） |
| DFS 图/树遍历 | 递归访问相邻节点 | ✅（周赛15-F） |
| BFS 最短路径 | 队列层序扩展，网格最短路 | ✅（周赛11-F 星际迷航；周赛17-F） |
| 二叉树 | 基本概念，前/中/后序遍历 | ✅（周赛16-F） |
| 哈夫曼树 | 带权路径长度最短的树 | ✅（周赛19-F） |
| 二叉排序树 | BST 的插入、查找 | ✅（周赛18-F） |
| 图的存储 | 邻接矩阵、邻接表 | ✅（周赛15-F；周赛17-F 邻接表） |
| 动态规划入门 | 最大子段和，爬楼梯，线性DP | ✅（周赛10-F） |
| 0-1 背包 | 经典二维DP，体积/价值选择 | ✅（周赛12-F） |
| 最长上升子序列（LIS） | O(n²) DP | ✅（周赛14-F） |
| 算法复杂度分析 | 搜索、排序、树/图遍历的复杂度 | ⬜ |

**周赛出题建议**：F 题用搜索（BFS/DFS）、DP 新模型（背包、LIS）或树/图相关。
