## 测试点 5（中位数在中间）

**输入**
```
5
1 3 5 7 9
```

**预期输出**
```
12
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <algorithm>
#include <cmath>
using namespace std;

const int N = 1005;
int n, a[N];

int main() {
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    sort(a, a + n);
    int mid = a[n / 2];  // 中位数
    int ans = 0;
    for (int i = 0; i < n; i++)
        ans += abs(a[i] - mid);
    cout << ans << endl;
    return 0;
}
```
