## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int a, b;
        cin >> a >> b;
        cout << a + b << " " << a - b << " " << a * b << endl;
    }
    return 0;
}
```
