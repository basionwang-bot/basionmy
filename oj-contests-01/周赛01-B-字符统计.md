# 周赛 01 - B 题：字符统计

**难度**：★☆☆☆☆  
**考点**：字符串、一维数组、循环与计数

---

## 题目描述

给定一行只包含小写字母的字符串，请统计其中每个字母出现的次数，并按字母表顺序输出「出现次数大于 0」的字母及其次数，格式为 `字母:次数`，每个占一行。

## 输入格式

一行字符串，长度不超过 1000，且只包含小写字母。

## 输出格式

按字母 a 到 z 的顺序，输出出现次数大于 0 的字母及次数，格式为 `字母:次数`，每行一个。

## 样例

**输入**
```
hello
```

**输出**
```
e:1
h:1
l:2
o:1
```

**输入**
```
aabbbcc
```

**输出**
```
a:2
b:3
c:2
```

## 说明

考察字符串遍历与简单计数，可用数组下标映射字母（如 `cnt[s[i]-'a']++`）。
