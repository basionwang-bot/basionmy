# 周赛 01 - C 题：最少操作

**难度**：★★☆☆☆  
**考点**：简单贪心或枚举、数组

---

## 题目描述

给定一个长度为 n 的正整数数组。每次操作你可以将**任意一个**元素加 1 或减 1（减 1 后仍为正整数）。请问至少需要几次操作，才能使得数组中**所有元素都相同**？

## 输入格式

第一行一个整数 n（1 ≤ n ≤ 1000）。  
第二行 n 个正整数（1 ≤ 每个数 ≤ 1000）。

## 输出格式

输出一个整数，表示最少操作次数。

## 样例

**输入**
```
3
1 2 3
```

**输出**
```
2
```

**说明**：可把 1 加 1、3 减 1，都变成 2，共 2 次操作。

**输入**
```
4
1 1 2 2
```

**输出**
```
2
```

**说明**：可把两个 2 都减 1 变成 1，共 2 次操作。

## 提示

可以枚举「最终统一成的那个数」在什么范围内，再对每种情况计算操作次数取最小值。或利用中位数性质（最终值取中位数时总操作最少）。
