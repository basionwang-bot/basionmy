# 周赛 01 - A 题：简单计算

**难度**：★☆☆☆☆  
**考点**：输入输出、循环、基本运算

---

## 题目描述

给定多组测试数据，每组一行两个整数 a 和 b。请计算并输出 a+b、a-b、a×b 的结果，用空格分隔。

## 输入格式

第一行一个整数 T（1 ≤ T ≤ 100），表示测试组数。  
接下来 T 行，每行两个整数 a, b（-1000 ≤ a, b ≤ 1000）。

## 输出格式

对每组数据输出一行三个整数：a+b、a-b、a×b，两两之间用空格分隔。

## 样例

**输入**
```
3
1 2
0 -5
10 10
```

**输出**
```
3 -1 2
-5 5 0
20 0 100
```
