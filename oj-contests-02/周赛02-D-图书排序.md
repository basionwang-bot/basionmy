# 周赛 02 - D 题：图书排序

**难度**：★★★☆☆  
**考点**：结构体、多关键字排序

---

## 题目描述

某书店有 n 本图书，每本书有：书名、作者、价格、库存量。请按以下规则排序后输出：  
1）按价格从小到大；  
2）价格相同则按库存量从大到小；  
3）若仍相同则按书名字典序从小到大。  
输出排序后的每本书信息：书名 作者 价格 库存量，每行一本。

**本题要求使用结构体存储每本书的信息。**

## 输入格式

第一行一个整数 n（1 ≤ n ≤ 100）。  
接下来 n 行，每行格式为：书名 作者 价格 库存量。书名和作者为不含空格的字符串，价格和库存量为非负整数。

## 输出格式

输出 n 行，每行格式为：书名 作者 价格 库存量。按题目要求的顺序输出。

## 样例

**输入**
```
3
Primer Li 45 10
Cookbook Zhang 45 20
Core Wang 30 15
```

**输出**
```
Core Wang 30 15
Cookbook Zhang 45 20
Primer Li 45 10
```

**说明**：价格 30 最小先输出；两本 45 元的书按库存从大到小，故库存 20 的 Cookbook 在前、库存 10 的 Primer 在后；若价格、库存都相同，再按书名字典序。

## 提示

定义结构体包含书名、作者、价格、库存；使用 sort 自定义比较：价格升序、库存降序、书名升序。
