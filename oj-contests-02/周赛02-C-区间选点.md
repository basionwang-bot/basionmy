# 周赛 02 - C 题：区间选点

**难度**：★★☆☆☆  
**考点**：贪心、排序

---

## 题目描述

给定 n 个闭区间 [li, ri]，请选出最少的点，使得每个区间内都至少有一个点被选中。输出最少需要的点的个数。

## 输入格式

第一行一个整数 n（1 ≤ n ≤ 1000）。  
接下来 n 行，每行两个整数 li, ri（0 ≤ li ≤ ri ≤ 10000），表示一个区间。

## 输出格式

输出一个整数，表示最少需要的点的个数。

## 样例

**输入**
```
3
1 3
2 4
3 5
```

**输出**
```
1
```

**说明**：选点 3 即可覆盖三个区间 [1,3]、[2,4]、[3,5]。

**输入**
```
3
1 2
3 4
5 6
```

**输出**
```
3
```

**说明**：三段区间互不重叠，至少需要 3 个点。

## 提示

按区间右端点从小到大排序，依次处理：每次选当前区间右端点作为点，并跳过所有包含该点的区间。
