# 周赛 02 - B 题：最值及下标

**难度**：★☆☆☆☆  
**考点**：数组、遍历

---

## 题目描述

给定 n 个整数，输出其中的最大值、最小值，以及它们第一次出现的下标（下标从 1 开始）。  
输出格式：最大值 最大值下标 最小值 最小值下标，同一行用空格分隔。

## 输入格式

第一行一个整数 n（1 ≤ n ≤ 1000）。  
第二行 n 个整数，用空格分隔（每个数在 int 范围内）。

## 输出格式

一行四个整数：最大值 最大值下标 最小值 最小值下标。

## 样例

**输入**
```
5
3 1 4 1 5
```

**输出**
```
5 5 1 2
```

**说明**：最大值为 5，第一次出现在下标 5；最小值为 1，第一次出现在下标 2。

**输入**
```
3
0 0 0
```

**输出**
```
0 1 0 1
```

## 说明

考察数组遍历与最值记录，注意「第一次出现」的下标。
