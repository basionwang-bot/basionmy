## 测试点 1（样例1：斐波那契数列）

**输入**
```
1 1 1 1 10
```
**预期输出**
```
55
```

**说明**：$A=B=1, p=q=1$ 即标准斐波那契，第10项为55。

---

## 测试点 2（样例2）

**输入**
```
2 1 1 2 5
```
**预期输出**
```
17
```

**说明**：代入 $a_n=A\cdot a_{n-1}+B\cdot a_{n-2}$，$A=1, B=2$：
- $a_1=2, a_2=1$
- $a_3 = 1\cdot a_2 + 2\cdot a_1 = 1\times1+2\times2=5$
- $a_4 = 1\cdot a_3 + 2\cdot a_2 = 1\times5+2\times1=7$
- $a_5 = 1\cdot a_4 + 2\cdot a_3 = 1\times7+2\times5=17$

## 测试点 6（较大 n，验证 long long）

**输入**
```
1 1 2 1 20
```
**预期输出**
```
9369319
```

**说明**（逐项推导）：1, 1, 3, 7, 17, 41, 99, 239, 577, 1393, 3363, 8119, 19601, 47321, 114243, 275807, 665857, 1607521, 3880899, 9369319。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    long long p, q, A, B;
    int n;
    cin >> p >> q >> A >> B >> n;
    long long a[35];
    a[1] = p; a[2] = q;
    for (int i = 3; i <= n; i++)
        a[i] = A * a[i-1] + B * a[i-2];
    cout << a[n] << "\n";
    return 0;
}
```

**关键点**：
1. $n=1$ 时直接输出 $p$，$n=2$ 时直接输出 $q$，数组已初始化，循环不会执行；
2. 使用 `long long` 防止 $n=30$ 时溢出；
3. 下标从 1 开始，与公式保持一致。
