## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int n;
    cin >> n;
    int a[1005], dp[1005];
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) {
        dp[i] = 1;
        for (int j = 0; j < i; j++) {
            if (a[j] < a[i])
                dp[i] = max(dp[i], dp[j] + 1);
        }
    }
    cout << *max_element(dp, dp + n) << "\n";
    return 0;
}
```

**关键点**：
1. `dp[i]` 表示以第 $i$ 个元素结尾的最长严格上升子序列长度，初始为 1；
2. 严格上升：`a[j] < a[i]`（不含等号）；
3. 最终答案是所有 `dp[i]` 的最大值，不一定是 `dp[n-1]`；
4. 时间复杂度 $O(n^2)$，$n\le1000$ 完全够用。
