# D - 潮汐规律

**难度**：★★★★☆  
**考点**：数列递推规律（线性递推）  
**GESP 级别**：4级

---

## 题目背景

海洋研究员发现某海域的潮汐高度遵循一种线性递推规律：每天的潮汐高度由前两天的高度按固定系数计算得出。探险队需要预测第 $n$ 天的潮汐高度。

## 题目描述

潮汐高度数列定义如下：

$$a_1 = p,\quad a_2 = q,\quad a_n = A \cdot a_{n-1} + B \cdot a_{n-2} \quad (n \ge 3)$$

给定 $p$、$q$、$A$、$B$ 和 $n$，求 $a_n$。

## 输入格式

一行五个整数 $p$、$q$、$A$、$B$、$n$（$1 \le p, q \le 100$，$1 \le A, B \le 3$，$1 \le n \le 20$）。

## 输出格式

输出一个整数，表示 $a_n$ 的值。

## 样例

**输入 1**

```
1 1 1 1 10
```

**输出 1**

```
55
```

**说明**：$A=B=1$，$p=q=1$，即斐波那契数列，$a_{10}=55$。

---

**输入 2**

```
2 1 1 2 5
```

**输出 2**

```
17
```

**说明**：代入公式 $a_n = A \cdot a_{n-1} + B \cdot a_{n-2}$，其中 $A=1, B=2$：
- $a_1=2,\ a_2=1$
- $a_3 = 1 \cdot a_2 + 2 \cdot a_1 = 1\times1+2\times2=5$
- $a_4 = 1 \cdot a_3 + 2 \cdot a_2 = 1\times5+2\times1=7$
- $a_5 = 1 \cdot a_4 + 2 \cdot a_3 = 1\times7+2\times5=17$

## 提示

用数组依次计算每一项：

```cpp
long long a[35];
a[1] = p; a[2] = q;
for (int i = 3; i <= n; i++)
    a[i] = A * a[i-1] + B * a[i-2];
cout << a[n];
```

使用 `long long` 防止数值溢出（在最坏约束下 $a_{20}$ 可达 $10^{12}$ 量级）。
