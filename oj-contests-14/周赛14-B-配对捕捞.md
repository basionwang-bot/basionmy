# B - 配对捕捞

**难度**：★★☆☆☆  
**考点**：双层循环枚举  
**GESP 级别**：2级

---

## 题目背景

探险队捕捞到了一批海洋生物，每只生物有一个重量值。队长想找出所有重量之和恰好等于目标值的配对数量，以便分配给两名队员。

## 题目描述

给定 $n$ 个整数组成的数组 $w$，以及目标值 $T$。统计满足 $i < j$ 且 $w[i] + w[j] = T$ 的下标对 $(i, j)$ 的数量。

## 输入格式

第一行两个正整数 $n$、$T$（$2 \le n \le 1000$，$1 \le T \le 2000$）。

第二行 $n$ 个正整数 $w_i$（$1 \le w_i \le 1000$）。

## 输出格式

输出一个整数，表示满足条件的配对数量。

## 样例

**输入**

```
5 7
3 4 1 6 2
```

**输出**

```
2
```

**说明**

满足 $w[i]+w[j]=7$ 的配对为：$(3,4)$ 和 $(1,6)$，共 2 对。

## 提示

用双层循环枚举所有 $i < j$ 的组合，判断两数之和是否等于 $T$：

```cpp
int cnt = 0;
for (int i = 0; i < n; i++)
    for (int j = i + 1; j < n; j++)
        if (w[i] + w[j] == T) cnt++;
```
