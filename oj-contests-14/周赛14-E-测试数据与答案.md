# E - 货舱整理 · 测试数据与答案

---

## 测试点 1（样例1）

**输入**
```
5
3 1 4 1 5
```
**预期输出**
```
1 1 3 4 5
3
```

**说明**：逆序对为 (3,1)、(3,1)、(4,1)，共 3 对。

## 测试点 2（样例2）

**输入**
```
5

```

```
1 2 3 4 5
5
```

**说明**：逆序对为 (5,1)、(5,3)、(5,2)、(5,4)、(3,2)，共 5 对。

## 测试点 4（完全逆序）

**输入**
```

```

## 测试点 6（含重复元素）

**输入**
```
5
2 2 2 1 1
```
**预期输出**
```
1 1 2 2 2
6
```

**说明**：每个 2 与两个 1 构成逆序对，共 $3\times2=6$ 对。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

const int MAXN = 100005;
int a[MAXN], tmp[MAXN];
long long cnt = 0;

void merge(int l, int mid, int r) {
    int i = l, j = mid + 1, k = l;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) {
            tmp[k++] = a[i++];
        } else {
            cnt += mid - i + 1;
            tmp[k++] = a[j++];
        }
    }
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r)   tmp[k++] = a[j++];
    for (int x = l; x <= r; x++) a[x] = tmp[x];
}

void mergeSort(int l, int r) {
    if (l >= r) return;
    int mid = (l + r) / 2;
    mergeSort(l, mid);
    mergeSort(mid + 1, r);
    merge(l, mid, r);
}

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    mergeSort(0, n - 1);
    for (int i = 0; i < n; i++) {
        if (i > 0) cout << " ";
        cout << a[i];
    }
    cout << "\n" << cnt << "\n";
    return 0;
}
```

**关键点**：
1. `cnt += mid - i + 1`：当右半部分元素先被选入时，左半部分 `[i..mid]` 共 `mid-i+1` 个元素都与它构成逆序对；
2. 使用 `long long` 统计，$n=10^5$ 时最多约 $5\times10^9$ 个逆序对；
3. 归并排序自身时间复杂度 $O(n\log n)$，同时完成排序与计数。
