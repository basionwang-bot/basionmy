# E - 货舱整理

**难度**：★★★★★  
**考点**：归并排序 + 逆序数统计  
**GESP 级别**：5级

---

## 题目背景

探险船的货舱里，货物摆放顺序混乱。整理员想知道当前摆放顺序相比有序状态"乱了多少"，这用**逆序数**来衡量：数组中满足 $i < j$ 且 $a[i] > a[j]$ 的下标对 $(i,j)$ 的数量。

整理员同时需要得到整理后的有序结果。

## 题目描述

给定 $n$ 个整数，输出从小到大排序后的结果，以及原数组中逆序对的数量。

要求用**归并排序**实现，在归并过程中统计逆序数。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 10^5$）。

第二行 $n$ 个整数（$1 \le a_i \le 10^9$）。

## 输出格式

第一行：排序后的数组（数字间用一个空格隔开）。

第二行：逆序对数量。

## 样例

**输入 1**

```
5
3 1 4 1 5
```

**输出 1**

```
1 1 3 4 5
3
```

**说明**：逆序对为 $(3,1)$、$(3,1)$、$(4,1)$，共 3 对（第一个 $1$ 出现两次，下标不同）。

---

**输入 2**

```
5
5 1 3 2 4
```

**输出 2**

```
1 2 3 4 5
5
```

**说明**：逆序对为 $(5,1)$、$(5,3)$、$(5,2)$、$(5,4)$、$(3,2)$，共 5 对。

## 提示

归并排序统计逆序数的关键：在 merge 步骤，当右半部分的元素 $a[j]$ 先被选入时，左半部分剩余的 $\text{mid}-i+1$ 个元素都与 $a[j]$ 构成逆序对：

```cpp
long long cnt = 0;

void merge(int a[], int tmp[], int l, int mid, int r) {
    int i = l, j = mid + 1, k = l;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) {
            tmp[k++] = a[i++];
        } else {
            cnt += mid - i + 1;  // 左半部分剩余元素都与 a[j] 构成逆序
            tmp[k++] = a[j++];
        }
    }
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r)   tmp[k++] = a[j++];
    for (int x = l; x <= r; x++) a[x] = tmp[x];
}

void mergeSort(int a[], int tmp[], int l, int r) {
    if (l >= r) return;
    int mid = (l + r) / 2;
    mergeSort(a, tmp, l, mid);
    mergeSort(a, tmp, mid + 1, r);
    merge(a, tmp, l, mid, r);
}
```

逆序数可能很大，使用 `long long` 类型。
