# A - 灯塔计费

**难度**：★☆☆☆☆  
**考点**：条件分支、多组输入  
**GESP 级别**：1级

---

## 题目背景

海洋探险队在夜间航行时需要借助灯塔导航。根据船只与灯塔的距离不同，需要支付不同的导航服务费。

## 题目描述

船只与灯塔的距离为 $d$（单位：海里），收费规则如下：

- $d \le 100$：收费 **50** 金币
- $100 < d \le 300$：收费 **120** 金币
- $d > 300$：收费 **200** 金币

给定 $T$ 组查询，每组输入距离 $d$，输出对应费用。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 100$）。

接下来 $T$ 行，每行一个正整数 $d$（$1 \le d \le 1000$）。

## 输出格式

共 $T$ 行，每行输出对应的费用。

## 样例

**输入**

```
3
80
200
500
```

**输出**

```
50
120
200
```

## 提示

按顺序判断三个区间即可，注意边界值属于哪个区间。
