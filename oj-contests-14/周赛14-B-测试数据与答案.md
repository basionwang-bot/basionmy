# B - 配对捕捞 · 测试数据与答案

---

## 测试点 1（样例）

**输入**
```
5 7
3 4 1 6 2
```
**预期输出**
```
2
```

## 测试点 2（没有满足条件的配对）

**输入**
```
4 10
1 2 3 4
```
**预期输出**
```
0
```

---

## 测试点 3（多对相同值）

**输入**
```
6 6
3 3 3 3 1 5
```
**预期输出**
```
6
```

**说明**：数组中有 4 个 3，任意两个 3 的配对共 $\binom{4}{2}=6$ 对，每对之和为 6。

## 测试点 5（目标值很大，无配对）

**输入**
```
3 2000
500 600 700
```
**预期输出**
```
0
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, T;
    cin >> n >> T;
    int w[1005];
    for (int i = 0; i < n; i++) cin >> w[i];
    int cnt = 0;
    for (int i = 0; i < n; i++)
        for (int j = i + 1; j < n; j++)
            if (w[i] + w[j] == T) cnt++;
    cout << cnt << "\n";
    return 0;
}
```

**关键点**：内层循环从 `j = i + 1` 开始保证每对只统计一次，不重不漏。
