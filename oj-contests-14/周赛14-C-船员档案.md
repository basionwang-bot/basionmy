# C - 船员档案

**难度**：★★★☆☆  
**考点**：结构体 + 条件筛选 + 多关键字排序  
**GESP 级别**：3级

---

## 题目背景

探险队需要从候选船员中筛选出符合出海资格的成员，并按经验年限排序张榜公示。

## 题目描述

给定 $n$ 名候选船员的信息（姓名、年龄、出海年限），筛选出满足以下条件的船员：

- 年龄 $\ge 18$  
- 出海年限 $\ge 2$

对筛选结果按以下规则排序：

1. 出海年限**从大到小**
2. 出海年限相同时，姓名**字典序从小到大**

输出所有符合条件的船员姓名（每行一个）。若没有符合条件的船员，输出 `none`。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，每行格式为：

```
姓名 年龄 出海年限
```

姓名为不含空格的字符串（长度 $\le 20$），年龄和出海年限均为正整数（$1 \le$ 年龄 $\le 100$，$0 \le$ 年限 $\le 50$）。

## 输出格式

按排序后的顺序，每行输出一个姓名。

## 样例

**输入**

```
5
Alice 25 5
Bob 17 3
Carol 30 5
Dave 20 1
Eve 22 3
```

**输出**

```
Alice
Carol
Eve
```

**说明**

Bob 年龄不满 18，Dave 年限不满 2，故不入选。  
剩余三人按年限降序、姓名升序：Alice(5) 和 Carol(5) 年限相同，Alice < Carol，然后 Eve(3)。

## 提示

```cpp
struct Sailor {
    string name;
    int age, years;
};
bool cmp(Sailor a, Sailor b) {
    if (a.years != b.years) return a.years > b.years;
    return a.name < b.name;
}
```
