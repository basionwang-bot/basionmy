# F - 航线规划

**难度**：★★★★★  
**考点**：最长上升子序列（LIS）  
**GESP 级别**：6级

---

## 题目背景

探险队规划了一条穿越群岛的航线，沿途经过 $n$ 个检查点，每个检查点记录了海拔高度。队长希望找到一段"递增子路线"——从中选出若干个检查点，使得它们的高度严格递增，且选出的检查点数量尽可能多。

## 题目描述

给定 $n$ 个整数组成的序列，求**最长严格上升子序列**的长度。

子序列不要求连续，但顺序不能改变（下标必须严格递增，值也严格递增）。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 1000$）。

第二行 $n$ 个整数（$1 \le a_i \le 10^9$）。

## 输出格式

输出一个整数，表示最长严格上升子序列的长度。

## 样例

**输入 1**

```
6
3 1 4 1 5 9
```

**输出 1**

```
4
```

**说明**：LIS 为 $1, 4, 5, 9$ 或 $3, 4, 5, 9$，长度为 4。

---

**输入 2**

```
8
10 9 2 5 3 7 101 18
```

**输出 2**

```
4
```

**说明**：LIS 为 $2, 3, 7, 18$ 或 $2, 5, 7, 18$ 等，长度为 4。

---

**输入 3**

```
4
1 3 2 4
```

**输出 3**

```
3
```

**说明**：LIS 为 $1, 3, 4$ 或 $1, 2, 4$，长度为 3。

## 提示

经典 $O(n^2)$ DP：

```cpp
int dp[1005];
for (int i = 0; i < n; i++) {
    dp[i] = 1;
    for (int j = 0; j < i; j++) {
        if (a[j] < a[i])
            dp[i] = max(dp[i], dp[j] + 1);
    }
}
int ans = *max_element(dp, dp + n);
cout << ans;
```

`dp[i]` 表示以 $a[i]$ 结尾的最长上升子序列长度。
