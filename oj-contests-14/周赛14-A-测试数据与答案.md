## 测试点 2（边界值）

**输入**
```
4
100
101
300
301
```
**预期输出**
```
50
120
120
200
```

**说明**：$d=100$ 属于第一档（$\le100$）；$d=101$ 属于第二档；$d=300$ 属于第二档（$\le300$）；$d=301$ 属于第三档。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int d;
        cin >> d;
        if (d <= 100) cout << 50 << "\n";
        else if (d <= 300) cout << 120 << "\n";
        else cout << 200 << "\n";
    }
    return 0;
}
```

**关键点**：注意边界 $d=100$ 收费 50，$d=300$ 收费 120，不要用 `<` 误判为 `<=`。
