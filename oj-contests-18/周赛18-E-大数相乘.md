# E - 大数相乘

**难度**：★★★★★  
**考点**：高精度乘法（大数 × 普通整数）  
**GESP 级别**：5级

---

## 题目背景

星际航行中，探险队测量到某星系的质量是一个超大整数，需要将其乘以一个倍率系数，得到实际质量值。普通 `long long` 已无法存储如此大的数字，需要高精度计算。

## 题目描述

给定一个大整数 $A$（以字符串形式输入）和一个普通整数 $b$，输出 $A \times b$ 的结果。

结果不应有前导零（若结果为 $0$，输出 `0`）。

$A$ 的位数不超过 100 位，$0 \le b \le 10000$。

## 输入格式

第一行字符串 $A$（不含前导零，$A \ge 0$）。

第二行整数 $b$。

## 输出格式

输出一个整数字符串，表示 $A \times b$。

## 样例

**输入 1**

```
12345
678
```

**输出 1**

```
8369910
```

**说明**：$12345 \times 678 = 8369910$。

---

**输入 2**

```
999999999999999999999
0
```

**输出 2**

```
0
```

---

**输入 3**

```
1
10000
```

**输出 3**

```
10000
```

## 提示

从低位到高位逐位相乘，处理进位：

```cpp
// 将字符串逆序存为数组（低位在前）
// result[i] = a[i] * b + carry
int carry = 0;
for (int i = 0; i < lenA || carry; i++) {
    int cur = (i < lenA ? a[i] * b : 0) + carry;
    result[i] = cur % 10;
    carry = cur / 10;
}
// 去掉前导零后，从高位到低位输出
```

关键：当 $b=0$ 时直接输出 `0`；循环结束后 carry 可能还有值，需继续处理。
