# B - 陨石分析

**难度**：★★☆☆☆  
**考点**：数组统计（计算均值，统计超过均值的元素个数）  
**GESP 级别**：2级

---

## 题目背景

探险队捡到了 $n$ 块陨石，记录了每块的重量。队长想知道：有多少块陨石的重量**严格大于**所有陨石的平均重量？

## 题目描述

给定 $n$ 个正整数，计算它们的平均值，输出其中**严格大于平均值**的元素个数。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 1000$）。

第二行 $n$ 个正整数（$1 \le a_i \le 10^6$）。

## 输出格式

输出一个整数，表示严格大于平均值的元素个数。

## 样例

**输入 1**

```
5
3 7 2 8 5
```

**输出 1**

```
2
```

**说明**：平均值 = $(3+7+2+8+5)/5=5.0$，严格大于 5.0 的有 7 和 8，共 2 个。

---

**输入 2**

```
4
4 4 4 4
```

**输出 2**

```
0
```

**说明**：所有元素等于平均值，没有严格大于的。

## 提示

```cpp
double avg = (double)sum / n;
int cnt = 0;
for (int i = 0; i < n; i++)
    if (a[i] > avg) cnt++;
```

注意用 `double` 计算平均值，避免整数除法丢失精度。
