# E - 大数相乘 · 测试数据与答案

---

## 测试点 1（样例1）

**输入**
```
12345
678
```
**预期输出**
```
8369910
```
**验证**：$12345 \times 678 = 8{,}369{,}910$。

## 测试点 4（进位链）

**输入**
```
9999999999
9999
```
**预期输出**
```
99989999990001
```
**验证**：$9999999999 \times 9999 = 99{,}989{,}999{,}990{,}001$（位数增加）。

## 测试点 5（A=0）

**输入**
```
0
5000
```
**预期输出**
```
0
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;
int main() {
    string sa; int b;
    cin >> sa >> b;
    if (b == 0 || sa == "0") { cout << "0\n"; return 0; }
    // 翻转，低位在前
    reverse(sa.begin(), sa.end());
    int n = sa.size();
    int res[205] = {};
    int carry = 0;
    for (int i = 0; i < n || carry; i++) {
        int cur = (i < n ? (sa[i] - '0') * b : 0) + carry;
        res[i] = cur % 10;
        carry = cur / 10;
        if (i >= n) n = i + 1;  // 扩展长度
    }
    // 去前导零
    int len = n;
    while (len > 1 && res[len-1] == 0) len--;
    for (int i = len-1; i >= 0; i--) cout << res[i];
    cout << "\n";
    return 0;
}
```

**关键点**：
1. $b=0$ 或 $A="0"$ 时直接输出 `0`；
2. 每位乘以 $b$ 后加上进位，取模得当前位，除以 10 得新进位；
3. 进位可能多出新的高位，循环条件包含 `|| carry`。
