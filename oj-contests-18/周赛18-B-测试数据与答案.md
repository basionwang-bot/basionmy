## 测试点 3（只有一个超过均值）

**输入**
```
3
1 2 9
```
**预期输出**
```
1
```
**说明**：avg=(1+2+9)/3=4.0，只有 9>4.0，答案 1。

---

## 测试点 4（n=1）

**输入**
```
1
100
```
**预期输出**
```
0
```
**说明**：avg=100.0，没有严格大于 100 的元素。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    int n; cin >> n;
    int a[1005]; long long sum = 0;
    for (int i = 0; i < n; i++) { cin >> a[i]; sum += a[i]; }
    double avg = (double)sum / n;
    int cnt = 0;
    for (int i = 0; i < n; i++) if (a[i] > avg) cnt++;
    cout << cnt << "\n";
    return 0;
}
```

**关键点**：均值用 `double` 计算，比较用 `>`（严格大于）。
