# A - 星球分级

**难度**：★☆☆☆☆  
**考点**：条件分支、浮点比较、多组输入  
**GESP 级别**：1级

---

## 题目背景

太空探索队发现了一批新天体，需要根据天体的质量（以地球质量为单位）自动将其分级归档。

## 题目描述

给定天体的质量 $M$（单位：地球质量），按以下规则输出类型：

- $M \ge 300$：输出 `GAS`（气态巨行星）
- $1 \le M < 300$：输出 `ROCK`（岩石行星）
- $M < 1$：输出 `DWARF`（矮行星）

给定 $T$ 组查询，每组输出对应类型。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 100$）。

接下来 $T$ 行，每行一个浮点数 $M$（$0.001 \le M \le 10000$）。

## 输出格式

共 $T$ 行，每行输出对应类型字符串。

## 样例

**输入**

```
4
350.0
2.5
0.3
1.0
```

**输出**

```
GAS
ROCK
DWARF
ROCK
```

**说明**：$M=1.0$ 恰好等于 1，属于 ROCK。

## 提示

注意边界：$M=300$ 属于 GAS，$M=1$ 属于 ROCK，使用 `>=` 判断时不要遗漏等号。
