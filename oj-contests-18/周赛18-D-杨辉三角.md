# D - 杨辉三角

**难度**：★★★★☆  
**考点**：二维数组递推  
**GESP 级别**：4级

---

## 题目背景

杨辉三角（Pascal's Triangle）是数学中的经典结构：每个数等于其正上方和左上方两个数之和，两端始终为 1。探险队发现古代星图上刻着杨辉三角的前 $n$ 行，需要程序还原。

## 题目描述

给定正整数 $n$，输出杨辉三角的前 $n$ 行。第 $i$ 行（从 1 开始）有 $i$ 个数，数字之间用一个空格隔开。

## 输入格式

一行一个正整数 $n$（$1 \le n \le 15$）。

## 输出格式

共 $n$ 行，第 $i$ 行输出杨辉三角第 $i$ 行的所有数。

## 样例

**输入**

```
5
```

**输出**

```
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
```

## 提示

用二维数组递推：

```cpp
int c[20][20] = {0};
for (int i = 1; i <= n; i++) {
    c[i][1] = c[i][i] = 1;          // 两端为 1
    for (int j = 2; j < i; j++)
        c[i][j] = c[i-1][j-1] + c[i-1][j];  // 等于上方两数之和
}
// 输出
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= i; j++) {
        if (j > 1) cout << " ";
        cout << c[i][j];
    }
    cout << "\n";
}
```

杨辉三角第 $i$ 行第 $j$ 个数实际上就是组合数 $\binom{i-1}{j-1}$。
