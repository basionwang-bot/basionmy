# F - 星图索引

**难度**：★★★★★  
**考点**：二叉排序树（BST）的插入与查找  
**GESP 级别**：6级

---

## 题目背景

探险队构建了一个星图索引系统，用**二叉排序树（BST）**存储星体编号。BST 的性质保证了快速查找：对任意节点，左子树所有值 < 节点值 < 右子树所有值。

## 题目描述

依次将 $n$ 个整数插入一棵初始为空的 BST（若该值已存在则不重复插入），然后回答 $m$ 次查询：某个值是否在 BST 中？

最后输出 BST 的**中序遍历**结果（即从小到大的有序序列）。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

第二行 $n$ 个整数，依次插入 BST（$-10^4 \le$ 值 $\le 10^4$）。

第三行一个正整数 $m$（$1 \le m \le 50$）。

接下来 $m$ 行，每行一个整数，为查询值。

## 输出格式

前 $m$ 行：每行输出 `YES` 或 `NO`，表示该值是否在 BST 中。

最后一行：BST 的中序遍历序列（数字间用一个空格隔开）。

## 样例

**输入**

```
5
5 3 7 1 4
2
3
6
```

**输出**

```
YES
NO
1 3 4 5 7
```

**说明**：BST 结构：根节点 5，左子树 {3,1,4}，右子树 {7}。查询 3 存在，查询 6 不存在。中序（左-根-右）= 1,3,4,5,7。

## 提示

```cpp
struct Node {
    int val;
    Node *left, *right;
    Node(int v) : val(v), left(nullptr), right(nullptr) {}
};

Node* insert(Node* root, int val) {
    if (!root) return new Node(val);
    if (val < root->val) root->left  = insert(root->left,  val);
    else if (val > root->val) root->right = insert(root->right, val);
    return root;
}

bool search(Node* root, int val) {
    if (!root) return false;
    if (val == root->val) return true;
    if (val < root->val) return search(root->left,  val);
    return search(root->right, val);
}

void inorder(Node* root) {
    if (!root) return;
    inorder(root->left);
    // 输出 root->val
    inorder(root->right);
}
```
