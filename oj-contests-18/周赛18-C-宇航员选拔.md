# C - 宇航员选拔

**难度**：★★★☆☆  
**考点**：结构体 + 条件筛选 + 多关键字排序  
**GESP 级别**：3级

---

## 题目背景

太空探索队需要从候选人中选拔宇航员，要求年龄和综合评分均达标，并按评分高低排列公示名单。

## 题目描述

给定 $n$ 名候选人的信息（姓名、年龄、综合评分），筛选满足以下条件的候选人：

- 年龄在 $[25, 45]$ 之间（含两端）
- 综合评分 $\ge 90$

对筛选结果按以下规则排序：

1. 综合评分**从大到小**
2. 评分相同时，年龄**从小到大**
3. 仍相同时，姓名**字典序从小到大**

输出所有符合条件的候选人姓名（每行一个）。若没有，输出 `none`。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，每行格式为：

```
姓名 年龄 评分
```

姓名为不含空格的字符串（长度 $\le 20$），年龄和评分均为正整数（$18 \le$ 年龄 $\le 60$，$0 \le$ 评分 $\le 100$）。

## 输出格式

按排序后顺序，每行输出一个姓名。

## 样例

**输入**

```
5
Alice 30 95
Bob 50 92
Carol 25 92
Dave 20 98
Eve 35 88
```

**输出**

```
Alice
Carol
Bob
```

**说明**：Dave 年龄 20 不满足 ≥25，Eve 评分 88 不满足 ≥90，淘汰。  
剩余三人：Alice(95)、Carol(92)、Bob(92)。  
按评分降序：Alice 第一；Carol 和 Bob 评分并列，年龄 25 < 50，Carol 排前。

## 提示

```cpp
struct Candidate { string name; int age, score; };
bool cmp(const Candidate& a, const Candidate& b) {
    if (a.score != b.score) return a.score > b.score;
    if (a.age   != b.age)   return a.age   < b.age;
    return a.name < b.name;
}
```
