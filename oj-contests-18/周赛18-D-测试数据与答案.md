## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    int n; cin >> n;
    int c[20][20] = {};
    for (int i = 1; i <= n; i++) {
        c[i][1] = c[i][i] = 1;
        for (int j = 2; j < i; j++)
            c[i][j] = c[i-1][j-1] + c[i-1][j];
    }
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= i; j++) {
            if (j > 1) cout << " ";
            cout << c[i][j];
        }
        cout << "\n";
    }
    return 0;
}
```

**关键点**：每行两端置 1，中间元素 = 上一行左上 + 正上；数组下标从 1 开始更直观。
