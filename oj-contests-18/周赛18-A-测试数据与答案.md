## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;
int main() {
    int T; cin >> T;
    while (T--) {
        double M; cin >> M;
        if (M >= 300) cout << "GAS\n";
        else if (M >= 1) cout << "ROCK\n";
        else cout << "DWARF\n";
    }
    return 0;
}
```
