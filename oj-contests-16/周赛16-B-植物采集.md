# B - 植物采集

**难度**：★★☆☆☆  
**考点**：前缀和 + 区间查询  
**GESP 级别**：2级

---

## 题目背景

探险队沿一条路线依次经过 $n$ 个采集点，每个采集点记录了采集到的植物数量。队长需要快速回答：从第 $l$ 个采集点到第 $r$ 个采集点，共采集了多少植物？

## 题目描述

给定 $n$ 个整数组成的数组，以及 $Q$ 次区间查询，每次查询输出区间 $[l, r]$ 内所有元素之和。

## 输入格式

第一行两个正整数 $n$、$Q$（$1 \le n \le 10^5$，$1 \le Q \le 10^5$）。

第二行 $n$ 个整数（$0 \le a_i \le 1000$）。

接下来 $Q$ 行，每行两个整数 $l$、$r$（$1 \le l \le r \le n$）。

## 输出格式

共 $Q$ 行，每行输出对应区间的元素之和。

## 样例

**输入**

```
6 3
3 1 4 1 5 9
1 3
2 5
3 6
```

**输出**

```
8
11
19
```

**说明**：
- $[1,3]$：$3+1+4=8$
- $[2,5]$：$1+4+1+5=11$
- $[3,6]$：$4+1+5+9=19$

## 提示

预处理前缀和数组，每次查询 $O(1)$：

```cpp
long long pre[100005];
pre[0] = 0;
for (int i = 1; i <= n; i++) pre[i] = pre[i-1] + a[i];
// 查询 [l, r]：
cout << pre[r] - pre[l-1];
```
