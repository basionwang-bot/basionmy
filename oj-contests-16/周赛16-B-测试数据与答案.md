## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, Q;
    cin >> n >> Q;
    long long pre[100005];
    pre[0] = 0;
    for (int i = 1; i <= n; i++) {
        int x; cin >> x;
        pre[i] = pre[i-1] + x;
    }
    while (Q--) {
        int l, r;
        cin >> l >> r;
        cout << pre[r] - pre[l-1] << "\n";
    }
    return 0;
}
```

**关键点**：前缀和 `pre[i]` 表示前 $i$ 个元素之和，区间 $[l,r]$ 的和为 `pre[r] - pre[l-1]`。
