# F - 遗迹密码树

**难度**：★★★★★  
**考点**：二叉树重建（前序 + 中序 → 后序）  
**GESP 级别**：6级

---

## 题目背景

探险队在遗迹中发现了一段古老的密码，经破译后对应一棵二叉树的前序遍历和中序遍历序列。队员需要根据这两个序列还原出后序遍历序列，才能解锁最终密室。

## 题目描述

给定一棵二叉树的**前序遍历**序列和**中序遍历**序列，输出该二叉树的**后序遍历**序列。

节点值互不相同。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 50$）。

第二行 $n$ 个整数，为前序遍历序列（根-左-右）。

第三行 $n$ 个整数，为中序遍历序列（左-根-右）。

## 输出格式

输出 $n$ 个整数，为后序遍历序列（左-右-根），数字间用一个空格隔开。

## 样例

**输入**

```
7
1 2 4 5 3 6 7
4 2 5 1 6 3 7
```

**输出**

```
4 5 2 6 7 3 1
```

**说明**：
```
        1
       / \
      2   3
     / \ / \
    4  5 6  7
```
- 前序（根-左-右）：1 2 4 5 3 6 7 ✓
- 中序（左-根-右）：4 2 5 1 6 3 7 ✓
- 后序（左-右-根）：4 5 2 6 7 3 1 ✓

## 提示

递归重建：前序第一个元素是根，在中序中找到根的位置，左侧为左子树，右侧为右子树。

```cpp
// pre[pl..pr] 是前序，in[il..ir] 是中序
void build(int pre[], int pl, int pr,
           int in[],  int il, int ir) {
    if (pl > pr) return;
    int root = pre[pl];
    // 在中序中找 root 的位置
    int pos = il;
    while (in[pos] != root) pos++;
    int leftLen = pos - il;  // 左子树节点数
    // 递归处理左子树
    build(pre, pl + 1, pl + leftLen, in, il, pos - 1);
    // 递归处理右子树
    build(pre, pl + leftLen + 1, pr, in, pos + 1, ir);
    // 后序：先输出左右子树，最后输出根
    cout << root;
    if (pl < pr) cout << " ";  // 注意空格处理
}
```

后序遍历是"左-右-根"，所以在递归函数返回前（两个子树递归结束后）输出根节点。
