## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int n;
    cin >> n;
    cin.ignore();
    vector<string> lines(n);
    for (int i = 0; i < n; i++) getline(cin, lines[i]);
    string keyword;
    getline(cin, keyword);

    bool found = false;
    for (auto& line : lines) {
        string date = line.substr(1, 10);
        string content = line.substr(12);
        if (content.find(keyword) != string::npos) {
            cout << date << "\n";
            found = true;
        }
    }
    if (!found) cout << "none\n";
    return 0;
}
```

**关键点**：
1. `line.substr(1, 10)` 提取 `[` 后面的 10 个字符，即日期 `YYYY-MM-DD`；
2. `line.substr(12)` 从下标 12 开始（跳过 `[YYYY-MM-DD] `，共12个字符）；
3. `cin >> n; cin.ignore()` 跳过读整数后残留的换行，避免 `getline` 读到空行。
