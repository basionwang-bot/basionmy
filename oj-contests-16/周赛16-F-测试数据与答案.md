# F - 遗迹密码树 · 测试数据与答案

---

## 测试点 1（样例）

**输入**
```
7
1 2 4 5 3 6 7
4 2 5 1 6 3 7
```
**预期输出**
```
4 5 2 6 7 3 1
```

**树结构**：
```
        1
       / \
      2   3
     / \ / \
    4  5 6  7
```

## 测试点 5（较复杂结构）

**输入**
```
5
3 9 20 15 7
9 3 15 20 7
```
**预期输出**
```
9 15 7 20 3
```

**说明**：
```
    3
   / \
  9  20
    /  \
   15   7
```
后序：9, 15, 7, 20, 3。

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <vector>
using namespace std;

int pre[55], in[55];
vector<int> result;

void build(int pl, int pr, int il, int ir) {
    if (pl > pr) return;
    int root = pre[pl];
    int pos = il;
    while (in[pos] != root) pos++;
    int leftLen = pos - il;
    build(pl + 1, pl + leftLen, il, pos - 1);
    build(pl + leftLen + 1, pr, pos + 1, ir);
    result.push_back(root);
}

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) cin >> pre[i];
    for (int i = 0; i < n; i++) cin >> in[i];
    build(0, n - 1, 0, n - 1);
    for (int i = 0; i < n; i++) {
        if (i > 0) cout << " ";
        cout << result[i];
    }
    cout << "\n";
    return 0;
}
```

**关键点**：
1. 前序第一个元素是根（`pre[pl]`）；
2. 在中序中找到根的位置 `pos`，左侧长度 `leftLen = pos - il`；
3. 递归：左子树前序范围 `[pl+1, pl+leftLen]`，右子树前序范围 `[pl+leftLen+1, pr]`；
4. 先递归左右子树，最后 `push_back(root)` → 自然形成后序（左-右-根）。
