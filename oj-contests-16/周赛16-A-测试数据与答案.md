## 测试点 2（边界值）

**输入**
```
4
5 3
4 3
5 2
4 2
```
**预期输出**
```
SAFE
WARNING
WARNING
DANGER
```

**说明**：$(5,3)$ 恰好满足 SAFE；$(4,3)$ $f<5$ 但 $w\ge3$ → WARNING；$(5,2)$ $f\ge5$ 但 $w<3$ → WARNING；$(4,2)$ 两项都不足 → DANGER。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int f, w;
        cin >> f >> w;
        if (f >= 5 && w >= 3) cout << "SAFE\n";
        else if (f < 5 && w < 3) cout << "DANGER\n";
        else cout << "WARNING\n";
    }
    return 0;
}
```

**关键点**：三个分支要覆盖全部情况，注意"其余情况"是 `else`（不需要再写条件）。
