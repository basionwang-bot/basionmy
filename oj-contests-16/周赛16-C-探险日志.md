# C - 探险日志

**难度**：★★★☆☆  
**考点**：字符串 find + substr  
**GESP 级别**：3级

---

## 题目背景

探险队每天记录一篇日志，格式固定为 `[YYYY-MM-DD] 日志内容`。队长想从所有日志中找出包含特定关键词的记录，并提取出对应日期。

## 题目描述

给定 $n$ 条日志，每条日志格式为：

```
[YYYY-MM-DD] 任意内容
```

给定一个关键词 $keyword$，找出所有**内容部分**（方括号之后的部分）包含该关键词的日志，按原始顺序输出对应的日期（格式 `YYYY-MM-DD`，不含方括号）。

若没有匹配的日志，输出 `none`。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，每行一条日志（长度 $\le 200$）。

最后一行为关键词 $keyword$（不含空格，长度 $\le 20$）。

## 输出格式

按顺序输出匹配日志的日期，每行一个。

## 样例

**输入**

```
4
[2024-03-01] found rare orchid near river
[2024-03-02] heavy rain, rested at camp
[2024-03-03] discovered orchid field
[2024-03-04] returned to base
orchid
```

**输出**

```
2024-03-01
2024-03-03
```

## 提示

每条日志格式固定：前 12 个字符是 `[YYYY-MM-DD]`，第 13 个字符（下标 12）是空格，之后是内容。

```cpp
string date = line.substr(1, 10);       // 提取日期
string content = line.substr(12);       // 提取内容
if (content.find(keyword) != string::npos) {
    cout << date << "\n";
}
```
