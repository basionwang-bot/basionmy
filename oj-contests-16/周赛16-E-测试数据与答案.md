# E - 木材切割 · 测试数据与答案

---

## 测试点 1（样例）

**输入**
```
4 11
8 14 9 12
```
**预期输出**
```
3
```

**验证**：$h=3$：$2+4+3+4=13\ge11$；$h=4$：$2+3+2+3=10<11$。

---

## 测试点 2（h=1 才刚好够）

**输入**
```
2 10
3 8
```
**预期输出**
```
1
```

**说明**：$h=1$：$3+8=11\ge10$；$h=2$：$1+4=5<10$。

---

## 测试点 3（完全无法满足，输出 0）

**输入**
```
2 100
3 4
```
**预期输出**
```
0
```

**说明**：即使 $h=1$，也只能切出 $3+4=7$ 段，远少于 100 段。

---

## 测试点 4（所有原木等长）

**输入**
```
3 6
10 10 10
```
**预期输出**
```
5
```

**说明**：$h=5$：$2+2+2=6\ge6$；$h=6$：$1+1+1=3<6$。

---

## 测试点 5（k=1，最大 h 等于最长原木）

**输入**
```
3 1
5 12 8
```
**预期输出**
```
12
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int n;
long long k;
int a[100005];

bool check(long long h) {
    long long total = 0;
    for (int i = 0; i < n; i++) total += a[i] / h;
    return total >= k;
}

int main() {
    cin >> n >> k;
    for (int i = 0; i < n; i++) cin >> a[i];
    long long l = 1, r = *max_element(a, a + n), ans = 0;
    while (l <= r) {
        long long mid = (l + r) / 2;
        if (check(mid)) {
            ans = mid;
            l = mid + 1;
        } else {
            r = mid - 1;
        }
    }
    cout << ans << "\n";
    return 0;
}
```

**关键点**：
1. `check(h)` 验证 $h$ 是否可行（可切出 $\ge k$ 段）；
2. 二分时，可行则记录答案并继续往右找更大的 $h$；
3. 若 `check(1)` 为 false，`ans` 保持 0，输出 0；
4. `total` 和 `k` 用 `long long` 防溢出（$n=10^5$，每根最多 $10^8$ 段）。
