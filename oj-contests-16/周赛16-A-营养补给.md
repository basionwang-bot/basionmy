# A - 营养补给

**难度**：★☆☆☆☆  
**考点**：多条件组合的条件分支  
**GESP 级别**：1级

---

## 题目背景

探险队在丛林中采集食物，每份食物包含两种营养素：纤维素（$f$ 克）和水分（$w$ 毫升）。根据含量判断食物的安全等级。

## 题目描述

给定食物的纤维素含量 $f$ 和水分含量 $w$，按以下规则输出安全等级：

- $f \ge 5$ **且** $w \ge 3$：输出 `SAFE`
- $f < 5$ **且** $w < 3$：输出 `DANGER`
- 其余情况：输出 `WARNING`

给定 $T$ 组查询，每组输入 $f$ 和 $w$，输出对应等级。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 100$）。

接下来 $T$ 行，每行两个整数 $f$、$w$（$0 \le f, w \le 20$）。

## 输出格式

共 $T$ 行，每行输出对应等级字符串。

## 样例

**输入**

```
4
6 4
2 1
5 2
3 5
```

**输出**

```
SAFE
DANGER
WARNING
WARNING
```

**说明**：
- $(6,4)$：$f\ge5$ 且 $w\ge3$ → SAFE
- $(2,1)$：$f<5$ 且 $w<3$ → DANGER
- $(5,2)$：$f\ge5$ 但 $w<3$ → WARNING
- $(3,5)$：$f<5$ 但 $w\ge3$ → WARNING
