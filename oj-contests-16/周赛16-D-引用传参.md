# D - 引用传参

**难度**：★★★★☆  
**考点**：引用（int& 参数）  
**GESP 级别**：4级

---

## 题目背景

探险队用一个数组记录各营地的补给数量。每次到达一个营地时，队长会通过"引用传参"方式更新该营地的补给数量。

## 题目描述

给定 $n$ 个营地的初始补给数量，执行 $m$ 次更新操作。每次操作给出位置 $x$ 和新值 $v$，将 $a[x]$ 更新为 $v$。

要求用**引用传参**的 `update` 函数实现（函数签名：`void update(int& x, int v)`）。

输出所有操作完成后的数组。

## 输入格式

第一行两个正整数 $n$、$m$（$1 \le n \le 100$，$1 \le m \le 100$）。

第二行 $n$ 个整数，为初始数组（$1 \le a_i \le 1000$）。

接下来 $m$ 行，每行两个整数 $x$、$v$（$1 \le x \le n$，$1 \le v \le 1000$），表示将位置 $x$ 的值更新为 $v$。

## 输出格式

输出一行，为操作后的数组（数字间用一个空格隔开）。

## 样例

**输入**

```
4 3
10 20 30 40
2 99
4 5
1 50
```

**输出**

```
50 99 30 5
```

**说明**：
- 将 $a[2]$ 更新为 99：`[10,99,30,40]`
- 将 $a[4]$ 更新为 5：`[10,99,30,5]`
- 将 $a[1]$ 更新为 50：`[50,99,30,5]`

## 提示

```cpp
void update(int& x, int v) {
    x = v;
}

// 调用：
update(a[x], v);
```

- `int& x` 是引用参数，`x` 是实参的别名，修改 `x` 直接修改原变量；
- 与指针的区别：引用在声明时绑定，语法更简洁（不需要 `*` 和 `&`）。
