# E - 木材切割

**难度**：★★★★★  
**考点**：二分答案  
**GESP 级别**：5级

---

## 题目背景

探险队发现了一片倒木区，需要从中切割出 $k$ 段相同长度的木材用于搭建营地。队员想尽可能让每段木材长一些，以节省材料。

## 题目描述

有 $n$ 根原木，长度分别为 $l_1, l_2, \ldots, l_n$。现在用一台切割机，设定切割高度 $h$：对每根原木，从其底部截取，超过 $h$ 的部分被切掉，即每根原木可以得到 $\lfloor l_i / h \rfloor$ 段长度为 $h$ 的木材（若 $l_i < h$ 则该根原木不贡献木材）。

求可以切出**至少 $k$ 段**木材的最大切割高度 $h$（$h$ 为正整数）。若无法切出 $k$ 段（即连 $h=1$ 都不够），输出 $0$。

## 输入格式

第一行两个正整数 $n$、$k$（$1 \le n \le 10^5$，$1 \le k \le 10^8$）。

第二行 $n$ 个正整数 $l_i$（$1 \le l_i \le 10^8$）。

## 输出格式

输出一个整数，表示最大切割高度 $h$。

## 样例

**输入**

```
4 11
8 14 9 12
```

**输出**

```
3
```

**说明**：
- $h=3$：$\lfloor8/3\rfloor+\lfloor14/3\rfloor+\lfloor9/3\rfloor+\lfloor12/3\rfloor=2+4+3+4=13\ge11$，可行。
- $h=4$：$2+3+2+3=10<11$，不可行。
- 故最大 $h=3$。

## 提示

对 $h$ 进行**二分**：

```cpp
bool check(int h, int a[], int n, long long k) {
    long long total = 0;
    for (int i = 0; i < n; i++) total += a[i] / h;
    return total >= k;
}

int l = 1, r = *max_element(a, a + n), ans = 0;
while (l <= r) {
    int mid = (l + r) / 2;
    if (check(mid, a, n, k)) {
        ans = mid;
        l = mid + 1;  // 尝试更大的 h
    } else {
        r = mid - 1;
    }
}
cout << ans;
```

- 单调性：$h$ 越大，能切出的段数越少；
- 二分右边界取原木最大长度；
- `total` 用 `long long` 防溢出。
