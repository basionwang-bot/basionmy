# A - 辣度分级·测试数据与答案

**输入**
```
4
85
50
10
30
```

**输出**
```
SPICY
MEDIUM

MEDIUM
```

---

## 测试点 2（边界值）

**输入**
```
5
70
69
30
29
0
```

**输出**
```
SPICY
MEDIUM
MEDIUM
MILD
MILD
```

**说明**：70→SPICY，69→MEDIUM（特辣下界）；30→MEDIUM，29→MILD（中辣下界）；0→MILD。

## 参考代码（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int T; cin >> T;
    while (T--) {
        int s; cin >> s;
        if (s >= 70) cout << "SPICY\n";
        else if (s >= 30) cout << "MEDIUM\n";
        else cout << "MILD\n";
    }
}
```
