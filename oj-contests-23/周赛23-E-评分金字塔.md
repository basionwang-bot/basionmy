# E - 评分金字塔

**难度**：★★★★☆  
**考点**：二维数组递推（数字三角形最大路径和）  
**GESP 级别**：4级（综合）

---

## 题目背景

美食大赛的评委把每道菜的得分摆成一座"评分金字塔"：第 1 行 1 个数，第 2 行 2 个数……第 $n$ 行 $n$ 个数。

主厨从塔顶出发往下走，每一步只能走到**正下方**或**右下方**相邻的那个数，一直走到塔底。他想让走过的所有得分之和**最大**。

## 题目描述

给定一个 $n$ 行的数字三角形，从顶部出发，每步可以移动到下一行**相邻**的左下或右下位置，求从顶到底一条路径上数字之和的最大值。

具体地，若当前在第 $i$ 行第 $j$ 个数，下一步可以走到第 $i+1$ 行的第 $j$ 个或第 $j+1$ 个数。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

接下来 $n$ 行，第 $i$ 行有 $i$ 个整数（$0 \le a_{ij} \le 1000$）。

## 输出格式

输出一个整数，表示从顶到底的最大路径和。

## 样例

**输入 1**

```
3
1
2 3
4 5 6
```

**输出 1**

```
10
```

**说明**：最优路径 $1 \to 3 \to 6$，和为 $1+3+6=10$。

---

**输入 2**

```
1
5
```

**输出 2**

```
5
```

**说明**：只有塔顶一个数。

## 提示

**从底往上递推**最简单。设 $dp[i][j]$ 为从第 $i$ 行第 $j$ 个数走到塔底的最大和：

- 最底行：$dp[n][j] = a[n][j]$
- 其余：$dp[i][j] = a[i][j] + \max(dp[i+1][j],\ dp[i+1][j+1])$

最终答案为 $dp[1][1]$。

```cpp
int dp[105][105];
// 先把最后一行抄进 dp
for (int j = 1; j <= n; j++) dp[n][j] = a[n][j];
// 自底向上
for (int i = n - 1; i >= 1; i--)
    for (int j = 1; j <= i; j++)
        dp[i][j] = a[i][j] + max(dp[i+1][j], dp[i+1][j+1]);
cout << dp[1][1];
```

时间复杂度 $O(n^2)$，$n=100$ 时只有约 $5000$ 次运算。
