# A - 辣度分级

**难度**：★☆☆☆☆  
**考点**：条件分支 / 分段判断  
**GESP 级别**：1级

---

## 题目背景

美食厨房新上了一批菜品，每道菜都有一个辣度值。后厨需要根据辣度值给菜品贴上对应的标签，方便顾客挑选。

## 题目描述

给定 $T$ 道菜，每道菜有一个辣度值 $s$，按下面规则输出辣度等级：

- $s \ge 70$：输出 `SPICY`（特辣）
- $30 \le s < 70$：输出 `MEDIUM`（中辣）
- $s < 30$：输出 `MILD`（微辣）

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 100$），表示菜品数量。

接下来 $T$ 行，每行一个整数 $s$（$0 \le s \le 100$），表示一道菜的辣度值。

## 输出格式

对每道菜，输出一行对应的辣度等级。

## 样例

**输入**

```
4
85
50
10
30
```

**输出**

```
SPICY
MEDIUM
MILD
MEDIUM
```

**说明**：
- 85 ≥ 70 → SPICY
- 50 在 [30,70) → MEDIUM
- 10 < 30 → MILD
- 30 恰好达到中辣下界 → MEDIUM

## 提示

```cpp
if (s >= 70) cout << "SPICY\n";
else if (s >= 30) cout << "MEDIUM\n";
else cout << "MILD\n";
```

注意边界值：70 算 SPICY，30 算 MEDIUM。
