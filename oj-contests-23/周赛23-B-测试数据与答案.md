# B - 人气菜品·测试数据与答案

## 测试点 1（样例 1）

**输入**
```
5
10 20 30 40 50
```

**输出**
```
2
```

**说明**：平均值 30，严格大于的有 40、50。

---

## 测试点 2（样例 2，全相等）

**输入**
```
4
5 5 5 5
```

**输出**
```
0
```

---

## 测试点 3（n=1）

**输入**
```
1
100
```

**输出**
```
0
```

**说明**：只有一道菜，它本身等于平均值，不严格大于。

---

## 测试点 4（平均值为小数）

**输入**
```
6
1 2 3 4 5 6
```

**输出**
```
3
```

**说明**：平均值 $21/6 = 3.5$，严格大于 3.5 的有 4、5、6，共 3 道。**若用整数除法会算成 3，导致结果出错。**

---

## 测试点 5（少数高人气）

**输入**
```
7
8 1 1 1 1 1 1
```

**输出**
```
1
```

**说明**：平均值 $14/7 = 2$，只有 8 严格大于 2。

---

## 测试点 6（大数值）

**输入**
```
3
1000000 1 1
```

**输出**
```
1
```

**说明**：平均值约 333334，只有 1000000 大于它。累加和需用 `long long`。

---

## 参考代码（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n; cin >> n;
    vector<int> a(n);
    long long sum = 0;
    for (int i = 0; i < n; i++) { cin >> a[i]; sum += a[i]; }
    double avg = (double)sum / n;
    int cnt = 0;
    for (int i = 0; i < n; i++)
        if (a[i] > avg) cnt++;
    cout << cnt << "\n";
}
```
