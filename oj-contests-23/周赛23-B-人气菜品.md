# B - 人气菜品

**难度**：★★☆☆☆  
**考点**：数组遍历 / 平均值 / 计数  
**GESP 级别**：2级

---

## 题目背景

美食厨房统计了今天 $n$ 道菜的点单次数。店长想知道有多少道菜是"人气菜品"——也就是点单次数**严格高于平均值**的菜。

## 题目描述

给定 $n$ 道菜的点单次数，先求出平均值，再统计**严格大于平均值**的菜品数量。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 1000$）。

第二行 $n$ 个整数，表示每道菜的点单次数（$0 \le a_i \le 10^6$）。

## 输出格式

输出一个整数，表示严格大于平均值的菜品数量。

## 样例

**输入 1**

```
5
10 20 30 40 50
```

**输出 1**

```
2
```

**说明**：平均值为 $(10+20+30+40+50)/5 = 30$，严格大于 30 的有 40、50，共 2 道。

---

**输入 2**

```
4
5 5 5 5
```

**输出 2**

```
0
```

**说明**：平均值为 5，没有任何一道严格大于 5。

## 提示

注意用浮点数计算平均值，避免整数除法丢失小数：

```cpp
long long sum = 0;
for (int i = 0; i < n; i++) sum += a[i];
double avg = (double)sum / n;
int cnt = 0;
for (int i = 0; i < n; i++)
    if (a[i] > avg) cnt++;
```
