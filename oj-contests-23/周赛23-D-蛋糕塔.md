# D - 蛋糕塔

**难度**：★★★★☆  
**考点**：递推（爬楼梯型，一次可走 1/2/3 级）  
**GESP 级别**：4级

---

## 题目背景

美食厨房做了一座 $n$ 层的巨型蛋糕塔。小厨师要从地面爬到塔顶，每一步可以向上跨 **1 层、2 层或 3 层**。他想知道一共有多少种不同的爬法。

## 题目描述

给定蛋糕塔的层数 $n$，每次可以向上走 1 层、2 层或 3 层，求从地面（第 0 层）恰好到达第 $n$ 层的不同走法总数。

## 输入格式

一行一个正整数 $n$（$1 \le n \le 30$）。

## 输出格式

输出一个整数，表示不同的爬法总数。

## 样例

**输入 1**

```
3
```

**输出 1**

```
4
```

**说明**：到第 3 层的走法有 `1+1+1`、`1+2`、`2+1`、`3`，共 4 种。

---

**输入 2**

```
1
```

**输出 2**

```
1
```

**说明**：只有一层，只能走 `1`，共 1 种。

---

**输入 3**

```
5
```

**输出 3**

```
13
```

## 提示

设 $f(i)$ 为到达第 $i$ 层的走法数。最后一步可能是从 $i-1$、$i-2$ 或 $i-3$ 层跨上来：

$$f(i) = f(i-1) + f(i-2) + f(i-3)$$

初始值：$f(0)=1$（站在地面算一种），$f(1)=1$，$f(2)=2$。

```cpp
long long f[35];
f[0] = 1; f[1] = 1; f[2] = 2;
for (int i = 3; i <= n; i++)
    f[i] = f[i-1] + f[i-2] + f[i-3];
cout << f[n];
```

$n=30$ 时结果约 $5.4 \times 10^7$，用 `long long` 稳妥。
