# D - 蛋糕塔·测试数据与答案

## 测试点 1（样例 1）

**输入**
```
3
```

**输出**
```
4
```

**说明**：`1+1+1`、`1+2`、`2+1`、`3`，共 4 种。

```
1
```

**输出**
```
1
```

---

## 测试点 3（样例 3）

**输入**
```
5
```

**输出**
```
13
```

**说明**：$f(4)=f(3)+f(2)+f(1)=4+2+1=7$，$f(5)=f(4)+f(3)+f(2)=7+4+2=13$。

---

## 测试点 5（n=10）

**输入**
```
10
```

**输出**
```
274
```

**说明**：逐项递推 $f$：1,1,2,4,7,13,24,44,81,149,**274**。

---

## 测试点 6（最大规模 n=30）

**输入**
```
30
```

**输出**
```
53798080
```

**说明**：$f(30)=53798080$。数值较大，建议用 `long long`。

---

## 参考代码（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n; cin >> n;
    long long f[35] = {};
    f[0] = 1; f[1] = 1; f[2] = 2;
    for (int i = 3; i <= n; i++)
        f[i] = f[i-1] + f[i-2] + f[i-3];
    cout << f[n] << "\n";
}
```
