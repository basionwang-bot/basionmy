## 测试点 3（销量相同，比菜名字典序）

**输入**
```
4
Zebra 100 90
Apple 100 85
Mango 100 99
Banana 100 80
```

**输出**
```
Apple
Banana
Mango
Zebra
```

**说明**：四道菜销量都是 100，评分都 ≥80，按菜名字典序升序排列。

## 测试点 4（边界评分 80 上榜，79 淘汰）

**输入**
```
3
Soup 50 80
Salad 60 79
Steak 40 100
```

**输出**
```
Soup
Steak
```

**说明**：Soup 评分恰好 80，上榜；Salad 79 淘汰。Soup 销量 50 > Steak 销量 40。

---

## 测试点 5（单条记录上榜）

**输入**
```
1
Solo 0 88
```

**输出**
```
Solo
```

**说明**：销量为 0 也可以上榜，只要评分达标。

## 参考代码（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;
struct Dish { string name; int sales, rating; };
int main() {
    int n; cin >> n;
    vector<Dish> v(n);
    for (int i = 0; i < n; i++)
        cin >> v[i].name >> v[i].sales >> v[i].rating;
    vector<Dish> res;
    for (auto& d : v)
        if (d.rating >= 80) res.push_back(d);
    if (res.empty()) { cout << "none\n"; return 0; }
    sort(res.begin(), res.end(), [](const Dish& a, const Dish& b){
        if (a.sales != b.sales) return a.sales > b.sales;
        return a.name < b.name;
    });
    for (auto& d : res) cout << d.name << "\n";
}
```
