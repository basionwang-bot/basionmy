# C - 暗语替换

**难度**：★★★☆☆  
**考点**：字符串 `find` + `substr` / `replace`  
**GESP 级别**：3级

---

## 题目背景

密室第三关：墙上贴着一段密文，旁边有一张"替换规则卡"，写着把某个词换成另一个词，文意才能显现出来。你需要把密文中所有出现的旧词替换成新词，读出真正的信息。

## 题目描述

给定一个字符串 $s$，以及两个字符串 $\text{old}$ 和 $\text{new}$。请把 $s$ 中**所有不重叠**的 $\text{old}$ 替换成 $\text{new}$（从左向右扫描，找到一个替换一个，替换后从新词结尾处继续向右查找），输出替换后的字符串。

若 $\text{old}$ 在 $s$ 中不存在，则原样输出 $s$。

## 输入格式

共三行：
- 第一行：字符串 $s$（长度 $1 \sim 200$，只含小写字母和空格）
- 第二行：字符串 $\text{old}$（长度 $1 \sim 20$，只含小写字母）
- 第三行：字符串 $\text{new}$（长度 $1 \sim 20$，只含小写字母）

## 输出格式

一行，替换后的字符串。

## 样例

**输入**

```
hello world
world
room
```

**输出**

```
hello room
```

## 提示

**核心方法**：循环调用 `string::find`，配合 `string::replace` 原地修改。

```cpp
string s, old_str, new_str;
getline(cin, s);
getline(cin, old_str);
getline(cin, new_str);

size_t pos = 0;
while ((pos = s.find(old_str, pos)) != string::npos) {
    s.replace(pos, old_str.size(), new_str);
    pos += new_str.size();   // 跳过新词，避免重复匹配
}
cout << s;
```

**注意**：替换后 `pos` 要跳过 `new_str` 的长度，而不是回到 `pos+1`，否则当 `new_str` 本身包含 `old_str` 时会死循环。

**也可以用 `substr` 手工拼接**：

```cpp
size_t pos = 0;
string result = "";
while (true) {
    size_t found = s.find(old_str, pos);
    if (found == string::npos) {
        result += s.substr(pos);
        break;
    }
    result += s.substr(pos, found - pos);  // old 之前的部分
    result += new_str;                      // 新词
    pos = found + old_str.size();           // 跳过 old
}
cout << result;
```
