# F - 宝箱装载 · 测试数据与答案

## 测试点 1（样例）

**输入**
```
4 10
2 3
3 4
4 5
5 6
```
**预期输出**
```
13
```

**说明**：选 (2,3)+(3,4)+(5,6)，重量=10，价值=13。

## 测试点 2（所有物品都装不下，价值为 0）

**输入**
```
2 1
2 10
3 20
```
**预期输出**
```
0
```

**说明**：每件物品重量都超过背包容量 1，无法取任何物品。

## 测试点 3（全部可以装下）

**输入**
```
3 10
1 1
2 2
3 3
```
**预期输出**
```
6
```

**说明**：三件物品总重量 1+2+3=6 ≤ 10，全部取走，价值 1+2+3=6。

## 测试点 4（恰好装满，经典两件组合）

```

2 3
3 4
```
**预期输出**
```
7
```

**说明**：两件都取，重量 2+3=5=W，价值 3+4=7。

## 测试点 5（较大数据，需权衡选择）

**输入**
```
5 15
3 4
4 5
5 7
6 9
7 10
```
**预期输出**
```
21
```

**说明**：选第 2、3、4 件物品：重量 4+5+6=15=W，价值 5+7+9=21。

- 处理完所有物品后，dp[15]=21。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int dp[1005];

int main() {
    int n, W;
    cin >> n >> W;

    for (int i = 0; i < n; i++) {
        int w, v;
        cin >> w >> v;
        for (int j = W; j >= w; j--) {
            dp[j] = max(dp[j], dp[j - w] + v);
        }
    }

    cout << dp[W] << "\n";
    return 0;
}
```

**关键点**：
1. `dp[]` 全局数组自动初始化为 0，无需手动 `memset`；
2. 内层循环**从 W 到 w 倒序**，确保每件物品只选一次（0-1 背包的核心）；
3. 若改成正序（从 `w` 到 `W`），则变为完全背包（每件可选无限次）；
4. 最终 `dp[W]` 即为恰好使用容量 ≤ W 时的最大价值（`dp[j]` 的语义是"容量上限为 j 时的最优解"）；
5. 空间 $O(W)$，时间 $O(nW)$，本题均在限制范围内。
