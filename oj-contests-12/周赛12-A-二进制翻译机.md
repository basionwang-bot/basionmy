# A - 二进制翻译机

**难度**：★☆☆☆☆  
**考点**：进制基础（二进制转十进制）  
**GESP 级别**：1级

---

## 题目背景

密室第一关：门口的显示板上闪烁着若干串 0 和 1 组成的神秘信号。墙上的破解提示写道：

> "二进制是计算机的语言。每一位的权重是 2 的幂，从右往左依次是 $2^0, 2^1, 2^2\ldots$ 把所有位的值加起来，就是对应的十进制数字。"

输入正确的十进制密码，门才会开启。

## 题目描述

给定 $T$ 个二进制字符串，每个字符串只含字符 `0` 和 `1`，长度不超过 8 位，无前导零（最高位为 `1`）。请输出每个二进制串对应的十进制整数。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 10$）。

接下来 $T$ 行，每行一个二进制字符串（长度 $1 \sim 8$，只含 `0`/`1`，无前导零）。

## 输出格式

$T$ 行，每行一个十进制整数。

## 样例

**输入**

```
3
1010
1111
1
```

**输出**

```
10
15
1
```

**说明**

- `1010`：$1\times2^3 + 0\times2^2 + 1\times2^1 + 0\times2^0 = 8+0+2+0 = 10$
- `1111`：$8+4+2+1 = 15$
- `1`：$1$

## 提示

**从左往右扫描字符串**，利用"逐位累加"法（Horner 法）：

```cpp
int val = 0;
for (char c : s) {
    val = val * 2 + (c - '0');
}
```

每读一位，把当前值乘以 2（相当于左移一位），再加上这一位的值（0 或 1）。

长度 ≤ 8 位，最大值为 $2^8 - 1 = 255$，不会溢出 `int`。
