# F - 宝箱装载

**难度**：★★★★☆  
**考点**：0-1 背包动态规划  
**GESP 级别**：6级

---

## 题目背景

密室最终关：面前摆着 $n$ 个宝箱，每个宝箱有各自的重量和价值。你只有一个承重上限为 $W$ 的背包，每个宝箱只能选择**拿或不拿**。请计算能带走的最大总价值。

## 题目描述

给定背包最大承重 $W$ 和 $n$ 件物品，第 $i$ 件物品的重量为 $w_i$，价值为 $v_i$。每件物品最多选取一次，求在总重量不超过 $W$ 的前提下，能获得的最大总价值。

## 输入格式

第一行两个整数 $n$ 和 $W$（$1 \le n \le 100$，$1 \le W \le 1000$）。

接下来 $n$ 行，每行两个整数 $w_i$ 和 $v_i$（$1 \le w_i, v_i \le 1000$）。

## 输出格式

一个整数，最大总价值。

## 样例

**输入**

```
4 10
2 3
3 4
4 5
5 6
```

**输出**

```
13
```

**说明**

选取第 1、2、4 件物品：重量 $2+3+5=10$，价值 $3+4+6=13$，恰好装满背包。

其他组合（如选 1+2+3，重量 9，价值 12）价值更低。

## 提示

**0-1 背包 DP**：

定义 `dp[j]` 为背包容量恰好为 $j$ 时的最大价值（滚动数组，一维 DP）。

初始：`dp[0..W] = 0`

对每件物品 $(w_i, v_i)$，**从大到小**枚举容量 $j$：

```cpp
for (int j = W; j >= w[i]; j--) {
    dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
}
```

**为什么从大到小？**  
从大到小枚举保证每件物品只被考虑一次（不会被重复选取）。若从小到大，`dp[j-w]` 可能已经在本轮被更新，导致物品被选多次（变成完全背包）。

最终答案：`dp[W]`

复杂度：$O(n \times W)$
