# B - 十进制密码锁

**难度**：★★☆☆☆  
**考点**：进制转换（十进制→二进制）  
**GESP 级别**：2级

---

## 题目背景

密室第二关：密码锁面板上显示的不是普通数字，而是一串串二进制编码。你手里只有十进制的线索卡，需要把每个十进制数转换成对应的二进制编码，才能输入解锁。

## 题目描述

给定 $T$ 个十进制正整数 $n$（$1 \le n \le 1000$），请输出每个数对应的**二进制字符串**（不含前导零）。

## 输入格式

第一行一个正整数 $T$（$1 \le T \le 10$）。

接下来 $T$ 行，每行一个正整数 $n$（$1 \le n \le 1000$）。

## 输出格式

$T$ 行，每行一个二进制字符串（无前导零）。

## 样例

**输入**

```
3
10
6
1
```

**输出**

```
1010
110
1
```

**说明**

- $10 = 8+2 = 1\times2^3+0\times2^2+1\times2^1+0\times2^0$，二进制为 `1010`
- $6 = 4+2$，二进制为 `110`
- $1 = 1$，二进制为 `1`

## 提示

**短除法**：不断对 2 取余，把余数从低位到高位收集，最后反转得到二进制字符串。

```cpp
string toBinary(int n) {
    string res = "";
    while (n > 0) {
        res += (char)('0' + n % 2);  // 取最低位
        n /= 2;                       // 右移一位
    }
    reverse(res.begin(), res.end());  // 余数是从低到高收集的，反转即得高→低
    return res;
}
```

$n \le 1000$，二进制最多 10 位（$2^{10} = 1024 > 1000$）。
