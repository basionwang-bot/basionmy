## 测试点 3（完全逆序，交换次数最多）

**输入**
```
5
5 4 3 2 1
```
**预期输出**
```
1 2 3 4 5
10
```

**说明**：逆序数组 n=5 时交换次数为 $n(n-1)/2 = 10$。

验证：
- 第 1 轮：4 次（每步都换）→ `4 3 2 1 5`
- 第 2 轮：3 次 → `3 2 1 4 5`
- 第 3 轮：2 次 → `2 1 3 4 5`
- 第 4 轮：1 次 → `1 2 3 4 5`
- 共 4+3+2+1 = 10 次

## 测试点 4（含重复元素）

**输入**
```
4
3 1 3 2
```
**预期输出**
```
1 2 3 3
3
```

**说明**：
- 第 1 轮：3>1 换(1)→`1 3 3 2`；3=3 不换；3>2 换(2)→`1 3 2 3`
- 第 2 轮：1<3 不换；3>2 换(3)→`1 2 3 3`
- 第 3 轮：1<2 不换
- 共 3 次

## 测试点 5（n=1，单个元素）

**输入**
```
1
42
```
**预期输出**
```
42
0
```

---

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int a[105];
    for (int i = 0; i < n; i++) cin >> a[i];

    int cnt = 0;
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (a[j] > a[j + 1]) {
                swap(a[j], a[j + 1]);
                cnt++;
            }
        }
    }

    for (int i = 0; i < n; i++) {
        if (i > 0) cout << " ";
        cout << a[i];
    }
    cout << "\n" << cnt << "\n";
    return 0;
}
```

**关键点**：
1. 外层 `i` 控制轮数（共 n-1 轮），内层 `j` 范围随 `i` 缩短（每轮最大值沉底，无需再比较）；
2. `a[j] > a[j+1]` 用严格大于，相等不交换，保证排序稳定；
3. 不能用 `sort()`，否则体现不出冒泡排序的交换计数过程。
