## 测试点 3（连续出现，new 比 old 短）

**输入**
```
aaaa
aa
b
```
**预期输出**
```
bb
```

**说明**：
- pos=0：找到 `aa`，替换为 `b`，s 变为 `"baa"`，pos=0+1=1
- pos=1：找到 `aa`（s[1]='a', s[2]='a'），替换为 `b`，s 变为 `"bb"`，pos=1+1=2
- pos=2：未找到，结束

**输入**

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s, old_str, new_str;
    getline(cin, s);
    getline(cin, old_str);
    getline(cin, new_str);

    size_t pos = 0;
    while ((pos = s.find(old_str, pos)) != string::npos) {
        s.replace(pos, old_str.size(), new_str);
        pos += new_str.size();
    }
    cout << s << "\n";
    return 0;
}
```

**关键点**：
1. 用 `getline` 读入，保留字符串中的空格；
2. `s.find(old_str, pos)` 从 `pos` 开始查找，返回第一个匹配位置；
3. `s.replace(pos, old_str.size(), new_str)` 替换从 `pos` 起长度为 `old_str.size()` 的子串；
4. `pos += new_str.size()` 跳过新词，防止对新词本身再次匹配（重要！）；
5. 返回 `string::npos` 表示未找到，循环终止。
