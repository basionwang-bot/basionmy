# E - 约瑟夫密码

**难度**：★★★★☆  
**考点**：单链表模拟（循环链表）  
**GESP 级别**：5级

---

## 题目背景

密室第五关：$n$ 名破解者围成一圈，每人手持一块密码碎片。游戏规则如下：从 1 号开始数数，数到第 $k$ 个人，该人离开并交出碎片，然后从下一个人重新开始数。如此循环，直到所有人都交出碎片。

按照离开的顺序拼合碎片，就能得到最终密码。

## 题目描述

$n$ 名破解者编号 $1$ 到 $n$ 围成一圈。从 1 号开始，每数 $k$ 个人淘汰一人；被淘汰者出圈后，从下一个人继续从 1 开始数。请输出所有人被淘汰的顺序。

## 输入格式

一行两个整数 $n$ 和 $k$（$1 \le n \le 100$，$1 \le k \le 100$）。

## 输出格式

一行 $n$ 个整数，表示淘汰顺序，整数之间用空格分隔。

## 样例

**输入**

```
5 3
```

**输出**

```
3 1 5 2 4
```

**说明**

初始圈：1→2→3→4→5→（循环）

| 轮次 | 从…数起 | 数 3 个 | 淘汰 | 剩余 |
|------|---------|---------|------|------|
| 1 | 1 | 1,2,**3** | 3 | 1 2 4 5 |
| 2 | 4 | 4,5,**1** | 1 | 2 4 5 |
| 3 | 2 | 2,4,**5** | 5 | 2 4 |
| 4 | 2 | 2,4,**2** | 2 | 4 |
| 5 | 4 | — | 4 | — |

## 提示

**用单链表（循环链表）模拟**：

建立含 $n$ 个节点的循环链表，每次从当前节点出发走 $k-1$ 步，到达的节点即被淘汰，删除后从下一个节点继续。

```cpp
struct Node {
    int val;
    Node* next;
    Node(int v) : val(v), next(nullptr) {}
};

// 建环形链表
Node* head = new Node(1);
Node* tail = head;
for (int i = 2; i <= n; i++) {
    tail->next = new Node(i);
    tail = tail->next;
}
tail->next = head;  // 尾接头，形成环

Node* cur = head;
Node* prev = tail;  // prev 始终指向 cur 的前驱节点

for (int i = 0; i < n; i++) {
    for (int j = 0; j < k - 1; j++) {   // 走 k-1 步
        prev = cur;
        cur = cur->next;
    }
    // cur 是被淘汰的节点
    prev->next = cur->next;              // 从链表中摘除
    Node* del = cur;
    cur = cur->next;                     // 下一轮从 cur->next 开始
    delete del;
}
```

- 链表节点只需 `val` 和 `next` 两个字段；
- 删除时更新 `prev->next`，不需要改动其他节点；
- $n \le 100$ 时，双重循环 $O(nk)$ 不超时。
