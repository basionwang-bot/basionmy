## 测试点 2（k=1，顺序淘汰）

**输入**
```
4 1
```
**预期输出**
```
1 2 3 4
```

**说明**：每数 1 个就淘汰，即按顺序依次淘汰。

---

## 测试点 3（k=n）

**输入**
```
5 5
```
**预期输出**
```
5 1 3 4 2
```

**说明**：每轮数到当前最后一个人。

验证（手动追踪）：
- 圈 1 2 3 4 5，从 1 数 5 个 → 淘汰 5，下一轮从 1 开始
- 圈 1 2 3 4，从 1 数 5 个：1,2,3,4,1 → 淘汰 1，下一轮从 2 开始
- 圈 2 3 4，从 2 数 5 个：2,3,4,2,3 → 淘汰 3，下一轮从 4 开始
- 圈 2 4，从 4 数 5 个：4,2,4,2,4 → 淘汰 4
- 淘汰 2
- 输出：5 1 3 4 2

## 测试点 5（经典 n=7 k=2）

**输入**
```
7 2
```
**预期输出**
```
2 4 6 1 5 3 7
```

**说明**：约瑟夫问题 k=2 的经典结果，每隔一人淘汰一人。

## 答案（C++ 参考代码）

```cpp
#include <iostream>
using namespace std;

struct Node {
    int val;
    Node* next;
    Node(int v) : val(v), next(nullptr) {}
};

int main() {
    int n, k;
    cin >> n >> k;

    Node* head = new Node(1);
    Node* tail = head;
    for (int i = 2; i <= n; i++) {
        tail->next = new Node(i);
        tail = tail->next;
    }
    tail->next = head;

    Node* cur = head;
    Node* prev = tail;

    for (int i = 0; i < n; i++) {
        for (int j = 0; j < k - 1; j++) {
            prev = cur;
            cur = cur->next;
        }
        if (i > 0) cout << " ";
        cout << cur->val;
        prev->next = cur->next;
        Node* del = cur;
        cur = cur->next;
        delete del;
    }
    cout << "\n";
    return 0;
}
```

**关键点**：
1. 构建**循环链表**：尾节点的 `next` 指向 `head`；
2. `prev` 始终是 `cur` 的前驱，删除 `cur` 时只需 `prev->next = cur->next`；
3. 删除后 `cur = cur->next`（即被删节点的后继），作为下一轮起点；
4. 走 `k-1` 步（不是 `k` 步），因为"数到第 k 个"时当前节点就是起点本身算第 1 个；
5. `delete del` 释放内存，避免内存泄漏（竞赛中可省略，但养成好习惯）。
