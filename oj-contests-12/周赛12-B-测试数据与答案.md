## 测试点 2（边界：最小值和最大值）

**输入**
```
2
1
1000
```
**预期输出**
```
1
1111101000
```

**说明**：$1000 = 512+256+128+64+32+8 = 1111101000_2$（10 位）。

验证：
```
1000 / 2 = 500  余 0
500  / 2 = 250  余 0
250  / 2 = 125  余 0
125  / 2 = 62   余 1
62   / 2 = 31   余 0
31   / 2 = 15   余 1
15   / 2 = 7    余 1
7    / 2 = 3    余 1
3    / 2 = 1    余 1
1    / 2 = 0    余 1
```
余数从下往上读：`1111101000` ✓

## 测试点 5（混合多组）

**输入**
```
5
7
15
100
200
999
```
**预期输出**
```
111
1111
1100100
11001000
1111100111
```

**说明**：
- 7 = 4+2+1 → `111`
- 15 = 8+4+2+1 → `1111`
- 100 = 64+32+4 → `1100100`
- 200 = 128+64+8 → `11001000`
- 999 = 512+256+128+64+32+4+2+1 → `1111100111`

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

string toBinary(int n) {
    string res = "";
    while (n > 0) {
        res += (char)('0' + n % 2);
        n /= 2;
    }
    reverse(res.begin(), res.end());
    return res;
}

int main() {
    int t;
    cin >> t;
    while (t--) {
        int n;
        cin >> n;
        cout << toBinary(n) << "\n";
    }
    return 0;
}
```

**关键点**：
1. `n % 2` 取最低位，`n /= 2` 相当于右移一位；
2. 余数是从低位到高位收集的，最后 `reverse` 得到正确的高→低顺序；
3. $n \ge 1$ 保证循环至少执行一次，不会输出空字符串。
