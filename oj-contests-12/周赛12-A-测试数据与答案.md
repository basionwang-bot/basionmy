## 测试点 4（混合值）

**输入**
```
3
1100100
1111010
10110
```
**预期输出**
```
100
122
22
```

**说明**：
- `1100100` = 64+32+4 = 100
- `1111010` = 64+32+16+8+2 = 122
- `10110` = 16+4+2 = 22

## 测试点 5（多组，含高低位混合）

**输入**
```
2
10000001
11001011
```
**预期输出**
```
129
203
```

**说明**：
- `10000001` = 128+1 = 129
- `11001011` = 128+64+8+2+1 = 203

## 答案（C++ 参考代码）

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int t;
    cin >> t;
    while (t--) {
        string s;
        cin >> s;
        int val = 0;
        for (char c : s) {
            val = val * 2 + (c - '0');
        }
        cout << val << "\n";
    }
    return 0;
}
```

**关键点**：
1. `val = val * 2 + (c - '0')` 是 Horner 法：从高位到低位逐步累加，等价于按权展开；
2. 长度 ≤ 8 位，最大值 255，`int` 完全够用；
3. `c - '0'` 把字符 `'0'`/`'1'` 转成整数 0/1。
