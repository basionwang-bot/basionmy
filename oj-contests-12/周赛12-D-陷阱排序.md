# D - 陷阱排序

**难度**：★★★☆☆  
**考点**：冒泡排序完整实现  
**GESP 级别**：4级

---

## 题目背景

密室第四关：地板上散落着 $n$ 块刻有数字的石板，必须按从小到大的顺序排列，才能踩踏通过——踩错顺序会触发陷阱。

更麻烦的是，石板只能与**相邻**的石板交换位置，所以你必须一步步地把较大的数"冒泡"到正确位置。为了评估危险程度，你还需要记录总共发生了多少次交换。

## 题目描述

给定 $n$ 个整数，用**冒泡排序**将其从小到大排序。输出排序后的数组和排序过程中发生的**总交换次数**。

冒泡排序规则：外层循环 $i$ 从 $0$ 到 $n-2$，内层循环 $j$ 从 $0$ 到 $n-2-i$，若 $a[j] > a[j+1]$ 则交换。

## 输入格式

第一行一个正整数 $n$（$1 \le n \le 100$）。

第二行 $n$ 个整数（$1 \le a_i \le 1000$）。

## 输出格式

第一行：排序后的数组，数字间用一个空格隔开。

第二行：总交换次数。

## 样例

**输入**

```
5
5 3 1 4 2
```

**输出**

```
1 2 3 4 5
7
```

**说明**

冒泡排序过程（每轮结束后的状态）：

| 轮次 | 数组状态 | 本轮交换 |
|------|---------|---------|
| 第 1 轮 | 3 1 4 2 **5** | 4 次 |
| 第 2 轮 | 1 3 2 **4** 5 | 2 次 |
| 第 3 轮 | 1 2 **3** 4 5 | 1 次 |
| 第 4 轮 | 1 2 3 4 5 | 0 次 |

共交换 $4+2+1+0 = 7$ 次。

## 提示

**冒泡排序模板**：

```cpp
int cnt = 0;
for (int i = 0; i < n - 1; i++) {
    for (int j = 0; j < n - 1 - i; j++) {
        if (a[j] > a[j + 1]) {
            swap(a[j], a[j + 1]);
            cnt++;
        }
    }
}
```

- 每轮结束后，当前最大值"冒"到末尾，所以内层范围每轮缩短一位（`n-1-i`）；
- 时间复杂度 $O(n^2)$，对本题 $n \le 100$ 完全够用。
